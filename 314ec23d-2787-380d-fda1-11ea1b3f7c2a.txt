        烟花殆尽后，傅夜沉从山上回到了屋内，见自己的房间里没有了童筱颖的身影，于是去了隔壁房间，但是他压了几下门上的手柄，未见门开，才意识到童筱颖将房门给反锁了。
<br/>
<br/>        所以……
<br/>
<br/>        她这是打算跟他分房睡？
<br/>
<br/>        傅夜沉剑眉微蹙，握在手柄上的拳头越来越紧，最终，他还是松了手。
<br/>
<br/>        好……
<br/>
<br/>        他给她时间……
<br/>
<br/>        给她慢慢地去忘记阿景的时间……
<br/>
<br/>        无论多久，他都愿意等。
<br/>
<br/>        第二天，他起床了，她还未起床，他去上班了，她才从房间里出来，他下班回来了，她早已提前吃过饭又回到了房间，并将房门反锁。
<br/>
<br/>        第三天，亦是如此。
<br/>
<br/>        第四天……
<br/>
<br/>        童筱颖拿起手机，无意间点开了许久未开的电子邮件。
<br/>
<br/>        里面除了澳洲那家宠物幼儿园发来让她去接“小爱”的邮件以外，剩下的未读邮件，全部来自于“小卫”。
<br/>
<br/>        她本想一次性全部删除，但是其中一封电子邮件的标题，让她突然感到心悸。
<br/>
<br/>        童筱颖心慌地点了进去。
<br/>
<br/>        筱颍，我把药停了，在外婆家等你来替我收尸。我过奈何桥的时候，不会喝孟婆汤，我想下辈子也记住你，然后找到你，再续前缘。——爱你的小卫。
<br/>
<br/>        药！
<br/>
<br/>        童筱颖再清楚不过是什么药了。
<br/>
<br/>        傅昼景做了肾移植手术，抗排斥的药物，医生叮嘱过她，要监督他终生服用。
<br/>
<br/>        如果停用，后果不堪设想！
<br/>
<br/>        童筱颖看了一下傅昼景的发件时间，立即收好手机，换了衣服，提起包包，拿了车钥匙离开了私宅。
<br/>
<br/>        她先去药店里买了药，然后将车子停在了高铁站外的停车场，买了票去了云城。
<br/>
<br/>        当她赶到外婆家时，刚好日落黄昏。
<br/>
<br/>        夕阳，轻轻照耀着种满各色明艳蔷薇的篱笆墙。
<br/>
<br/>        傅昼景微蹲在蔷薇花圃前，正卷着衣袖细心地给蔷薇花浇水。
<br/>
<br/>        他身上穿着白衬衫，被夕阳染上了一层淡淡的橘色，俊脸上显得有些病态的苍白。
<br/>
<br/>        听到这边的动静声，傅昼景站起身来，带着希冀的目光，看向了童筱颖。
<br/>
<br/>        几天不见，或许是她的穿着，将她衬得比之前更漂亮更有气质了。
<br/>
<br/>        她的脸庞依旧清瘦，但脸上不再对他露出甜美的笑容，那双明亮的眼眸中流露出的，也不再是对他的爱慕，而是一种淡淡的疏离。
<br/>
<br/>        童筱颖的周身散发的生人勿近的气息，这些变化让傅昼景惊诧不已，隐隐地感到有些不安。
<br/>
<br/>        “把药吃了！”
<br/>
<br/>        她朝他走了过去，第一件事情就是从她的手提包里掏出那几瓶药，让他吃下去。
<br/>
<br/>        傅昼景垂眸看了一眼童筱颖手中的那几个药瓶，不知道是要高兴好，还是要伤感好。
<br/>
<br/>        瞧！
<br/>
<br/>        她还记得他要吃的是什么药。
<br/>
<br/>        他不屑地笑了笑，抬眸对上她的眼睛，深沉而阴郁地说道：“筱颍，我把肾还给你，你回到我身边来，好吗？”
<br/>
<br/>        “我已经不需要了！”童筱颖淡淡地说，将药塞到了傅昼景的手中，“昼景，我已经放下你了。你对我，也放手，好吗？不要再拿自己身体来当赌注了。这是最后一次。”
<br/>
<br/>        “筱颍，我知道你还爱着我，你为什么就是不愿意承认？其实，你也相信我的话，对不对？我们真真切切地拥有过彼此三年的幸福时光。”他反转着抓住了她的手腕，按在了自己的心口上，而手中那几个药瓶“啪嗒啪嗒”全掉到了地上。
<br/>
<br/>        童筱颖想将手抽回来，傅昼景却紧紧地抓着她的手不放。
<br/>
<br/>        如果没有季思妍，她现在一定已经怀上他的孩子了。
<br/>
<br/>        如果没有季思妍，他们现在一定还居住在澳洲，两人彼此相拥在沙发上，看着她喜欢的综艺节目。
<br/>
<br/>        傅昼景幻想过很多“如果”，可是现实不许他拥有那些“如果”。
<br/>
<br/>        “筱颍，我们重新开始，好不好？”他哽咽的声音里，透着一丝情难自控的哀求。
<br/>
<br/>        比起上一次见他，她还有想哭的冲动，这一次，她显得要平静了许多。
<br/>
<br/>        “我已经结婚了，所以，我们之间也已经不可能了。”童筱颖冷淡地开口。
<br/>
<br/>        傅昼景怔愣地凝视着童筱颖的眼睛，眼神里充满了难以置信。
<br/>
<br/>        “我说的是真的。”童筱颖强颜欢笑，微微扬起嘴角，接着说道，“你看，我手上都戴着求婚戒指了。已经领了证，婚礼将至。我只是不打算请你。现在，你既然知道了，到时候，不如带着尚轩一起来吃喜糖。”
<br/>
<br/>        傅昼景垂下眼帘，拉起童筱颖的这只手，这才看到她中指上的钻戒。
