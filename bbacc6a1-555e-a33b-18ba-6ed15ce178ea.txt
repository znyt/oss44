　　骗子什么的最可恶了，李承乾前一世就对这样的人深恶痛绝，不过那个时候一直没有能力去修理他们，所以只能看着这帮人在那里耀武扬威的炫耀。
<br/>　　现在终于有机会，也有能力，这位太子殿下如何能放过这些为了一己自私而坑人的骗子。
<br/>　　不过你别说，这杨常林身体条件还真不错，五十多岁的人了，被净身之后竟然屁事儿没有，五天之后就能下地行走，十天之后就可以凭着一张利嘴忽悠人了。
<br/>　　这不，李承乾刚刚躺下，外面就响起了‘啪啪’的敲门声，一个尖细的声音说道：“殿下，时间差不多了，应该……”
<br/>　　“应该你大爷，给老子滚……”前奏刚刚进行完就特么来催，敬事房的这帮王八蛋看来是活够了，李承乾一边骂一边钻进了被子里，软香温玉在怀，好不消魂。
<br/>　　然而，还没等他继续下去，拍门的声音又响了：“殿下，已经半天多时辰了，按规定杨诏训该出来了！”
<br/>　　这特么不是搅和么，看着怀中玉人，李承乾怒从心头起，恶向胆边生。
<br/>　　娘、、的，虽然敬事房可以管宫闱之事，可以管那事儿的时间，毕竟是怕皇上和皇子过于贪欢而误了国事。
<br/>　　可是自有唐以来这个部门其实也就是徒有其名，除了给人净个身之外，其他事务已经并不怎么用他们管了，毕竟有李二这个狂拽酷霸的皇帝在，谁也不敢去撩拨他。
<br/>　　再说长孙皇后也不是吃素的，如果敢打扰了这两位的好事儿，估计没有十个八个脑袋是不够掉的。
<br/>　　……
<br/>　　可这事情总有那么一两次例外，老头子这几天不知怎么就想起这一出儿来了，搞了个一宫两治，弄的李承乾经常是不上不下的，好不容易有点兴致了，这特么敬事房的人这来拍门。
<br/>　　这一次两次的他也就忍了，多少给敬事房，不，给老头子一点面子。
<br/>　　而且这事儿讲个礼上往来，我敬你一尺，你怎么也得还我一丈吧？特么我敬你一尺，你就往前挪一尺算怎么回事？找病？还是特么觉得老子好欺负？丫特么不是新来的吧？
<br/>　　想到这里，李承乾从炕上爬了起来，看了一眼身边略有些幽怨的小雨姑娘，在她肩膀上按了一下又把被子给她掖了掖，然后对外面说道：“进来吧！”
<br/>　　“喏！”刚刚在外面拍门的那个声音传了进来，接着房门被人推开，四个内侍在一个小头目的带领下进了房间。
<br/>　　不过就在其刚刚靠近床边，还没等伸手抓向床上的被子时，李承乾已经一记大脚踹了过去，同时口中一声大喝：“来人，抓刺客！”
<br/>　　卧槽，刺客？几个内侍愣了一下，还没等反应过来，已经被首先冲进来的小白和夜魅扯着领子丢了出去，发出一连串的惨叫声。
<br/>　　不过事情似乎并没有就此结束，李承乾‘睚眦必报’的名声又不是一天两天了，怎么可能被几个内侍欺到头上而不报复。
<br/>　　这家伙平时没事儿都能搞出三分事儿来，更不要说被人惹到头上了。
<br/>　　……
<br/>　　“说吧，你们几个到底为何行刺本宫？”换过衣服，几个内侍已经被东宫禁卫押在阶下。
<br/>　　“没，没有，太子殿下，臣，臣冤枉啊！”那领头的家伙苦着脸叫道。
<br/>　　“冤枉？你特么是新来的吧？”李承乾走近两步，居高临下的看着那内侍头目：“本宫会随便冤枉人？”
<br/>　　“不，不是，太子殿下，臣真的没有想要行刺啊。”内侍头目脑袋上汗都下来了，他很清楚行刺太子是什么样的罪名。
<br/>　　“没有？那你刚刚鬼鬼祟祟的干什么？”李承乾眯着眼睛问道。
<br/>　　“我，我……”
<br/>　　“难道你是想偷东西？”李承乾睁开眼睛，盯着那家伙问道。
<br/>　　“我，是，是管事看中了殿下宫中的东西，指，指使小人来，来偷。”被李承乾想要杀人的目光盯着，那内侍头目终于反应过来这位太子殿下是想要干什么了，连忙点头应是，拿自己的小命做了一次豪赌。
<br/>　　毕竟李承乾以前从来都没有为难宫人的先例，这次突然指着他说其行刺，这本身就不合常理。
<br/>　　联系刚刚自己打断了这位太子爷的‘雅兴’，内侍头目如果还不知道这其中的门道，那可真是白活了，就算真的被李承乾派人拉出去砍了脑袋都不算冤。
<br/>　　果然，他这一点头李承乾立刻不再找他麻烦，命人将他押到一边，便传那敬事房管事过来问讯。
<br/>　　……
<br/>　　敬事房管事此时早已经入睡，被人从床上拖下来时连外衣都来不及穿，等被带到李承乾面前的时候，已经吓的尿了好几回的裤子。
<br/>　　看着这位太子爷阴郁的表情，再看看被押在一旁的几个手下，管事几乎立刻就意识到发生了什么，不由为自己的鬼迷心窍而暗暗后悔。
<br/>　　可是现在这个时候说什么都晚了，他的一切行为明显是引起那位小爷的不满，如果不能平息他的怒火，估计能不能见到明天的太阳真不好说。
<br/>　　“说吧，把本宫想知道都说出来，若有半句敷衍，老子让人生撕了你。”看着哆哆嗦嗦被押在面前的敬事房管事，李承乾面无表情的问道。
<br/>　　“臣……”
<br/>　　“说！”无缘无故被针对，若说李承乾没有火气那是不可能的，现在把敬事房最大的头目押来了，他自己要讨个说法，什么借口，什么理由都不顾了。
<br/>　　“是，是是，这，这一切都和臣无关啊，都是，都是杨常林那家伙搞的鬼，是他向陛下建议整，整顿宫中秩序的。”
<br/>　　管事的话音刚话，李承乾便是一声怒喝：“放屁！”
<br/>　　“真的，臣说的都是真的，千真万确！”
<br/>　　“那杨常林在什么地方？你可敢与他当面对质。”
<br/>　　敬事房管事已经快要被吓哭了，语速极快的说道：“被调到庭掖局去了，具体在哪里臣也不知道，不过臣敢与他对质，臣以脑袋担保，臣说的一切都是真的，若有半句假话，天打雷劈！”
