　　“小子！我不管你到底是什么来头，今日你非死不可！”
<br/>　　南日山大吼了一声，双手飞快结印，
<br/>　　轰隆隆！
<br/>　　伴随着烈日宗数十名强者一起出手，周围空间一阵晃动，很快在空中形成一个光罩，将叶尘围困在里面。
<br/>　　而叶尘，却一脸平淡，似乎根本没有放在心上，依旧静静地站在那里，无比从容。
<br/>　　“本宗主还就不信了，我们这么多人联手，难道还打不过你一个？”
<br/>　　“看招！”
<br/>　　南日山大喝一声，调动阵法之力，猛然一拳朝叶尘狠狠砸落！
<br/>　　轰隆！
<br/>　　集合数十名强者，加上南日山又是元婴境的高手，这一拳之下，当真有惊天动地之威。
<br/>　　就连叶尘也不由得眉头微微一皱，
<br/>　　倘若在他全盛之时，对方这一拳，他自然不会放在眼里，可惜他如今功力尚未完全恢复，只能发挥出四五成的实力，而且还不能牵动体内的伤势。
<br/>　　一番思量之下，叶尘身体微微一晃，瞬间倒退了数十米，从容躲开了对方这一拳。
<br/>　　哗！
<br/>　　众人看到这一幕，顿时又纷纷惊呼起来，
<br/>　　“这难道就是烈日宗最有名的真武大阵吗？果然厉害啊！”
<br/>　　“传说这真武大阵，能将数十名强者的功力聚集在一处，号称化神之下无敌，今日一见，果然名不虚传啊！”
<br/>　　“南日山竟然连这个大阵都使了出来，那小子恐怕插翅也难逃了！”
<br/>　　……
<br/>　　听到众人的议论声，原本才刚刚松了一口气的云灵儿，顿时再次脸色大变，满脸担忧之色。
<br/>　　而楚星河等人，则暗暗松了一口长气，
<br/>　　之前叶尘的表现，着实把他们吓得不轻！
<br/>　　倘若连烈日宗都铩羽而归，叶尘又岂会放过他们？
<br/>　　好在看目前的情景，烈日宗使出真武大阵之后，终究还是稳占上风了。
<br/>　　“最好他们双方打个两败俱伤，这才是对我流光派最好的结果！”
<br/>　　楚星河在心中暗暗嘀咕道。
<br/>　　轰轰轰！
<br/>　　南日山一连数次攻击，都被叶尘已经精妙的身法躲避开来，顿时让他又气又怒，
<br/>　　“小子！你就只会躲吗？”
<br/>　　叶尘却不闻不问，依旧从容不迫的躲闪着。
<br/>　　他不想牵动体内的伤势，所以唯一的办法，就是先消耗对方的实力，然后再找准机会，一举击败对方。
<br/>　　只不过，这落在周围旁观者的眼中，显然都以为他实力不济，毫无还手之力。
<br/>　　唯有南日山心里最清楚，他使出了全身解数，却愣是连对方的衣角都没有摸到，若非这小子已经被真武大阵困住，恐怕早就从容而退了。
<br/>　　“啊！！”
<br/>　　南日山愤怒之下，不由得大吼了一声，
<br/>　　“变阵！由‘攻’字诀，改为‘缚’字诀！”
<br/>　　“是！”
<br/>　　随着南日山一声令下，烈日宗其余众强者，顿时纷纷变换招式。
<br/>　　原来，这真武大阵，一共有六字真诀，可攻可守，变化莫测。
<br/>　　而这‘缚’字诀，顾名思义，就是以束缚对手为第一要务，而攻击力则会相对减弱。
<br/>　　轰隆隆！
<br/>　　伴随着真武大阵发生改变，整个大阵猛然收缩，一道道由真元所化的锁链，纷纷朝叶尘缠绕而来，令他的行动大大受限，速度也随之锐减。
<br/>　　“桀桀桀！”
<br/>　　看到‘缚’字诀奏效，南日山顿时兴奋的狂笑起来，
<br/>　　“小子！本宗主看你这次还往哪逃？给我受死吧！”
<br/>　　南日山再次一拳砸出！
<br/>　　轰隆隆！
<br/>　　叶尘被那些真元锁链缠绕，此时已经退无可退，避无可避！
<br/>　　然而，面对南日山这一拳，叶尘却嘴角微微一扬，露出一抹冷笑，
<br/>　　“唉！原本还想让你再多活片刻，既然你自己找死，那可就怪不得我了！”
<br/>　　“给我，破！”
<br/>　　随着那‘破’字落下，叶尘身上陡然金芒四射，而那些缠绕在他身上的真元锁链，瞬间全都爆裂开来！
<br/>　　“什么！”
<br/>　　南日山看到这一幕，顿时脸色狂变，可惜他一击发出，如今箭在弦上，他已经退无可退，只好调动全部的力量，将所有的赌注，都压在这一招上面。
<br/>　　“我这一拳，集中了我烈日宗数十名强者的力量，我就不信你能撑得住！！”
<br/>　　南日山在心头狂吼，整个人状若癫狂，身上竟升起一股浓浓的血色！
<br/>　　而反观叶尘，则抬起拳头，轻飘飘的一拳，缓缓迎了上去。
<br/>　　轰隆！
<br/>　　两人的拳劲，在空中狠狠相撞在一处，迸发出一道璀璨的光芒，连带着整个空间，都一阵剧烈晃动，似乎要裂开了一般。
<br/>　　下方周围的众人，顿时全都死死盯着璀璨的光团，想要第一时间看清楚结果，
<br/>　　“谁赢了？”
<br/>　　“那还用说吗？肯定是烈日宗啊！”
<br/>　　“南日山宗主本就是元婴境强者，又有真武大阵的加持，在化神天君之下堪称无敌，除非那少年，真的是化神天君，否则焉能保住性命？”
<br/>　　“不错！如此年轻的化神天君，便是在那最高等的九级星域，只怕也是凤毛麟角吧？又岂会出现在他们这个小小的二级星域？”
<br/>　　“不过，能将烈日宗逼迫到如此地步，此人也算是少有的奇才了！”
<br/>　　……
<br/>　　众人纷纷议论着，竟没有一人认为，叶尘能够取胜。
<br/>　　没办法，烈日宗在天狼星域称霸近千年，在众人心中的霸主地位，早已经根深蒂固，否则之前那楚星河，也不会如此迫不得已的出卖叶尘，来讨好南日山了。
<br/>　　“星河，这次多亏你反应够快，这才让我流光派逃过一劫，等此人一死，我们便将所有责任，全都推到他的身上，想来烈日宗，应该不会再找我流光派的麻烦了吧？”
<br/>　　流光派掌门莫长林，冲着身旁的楚星河，一脸欣慰道。
<br/>　　楚星河阴阴一笑，
<br/>　　“师尊放心！弟子已经想好了说辞，定让烈日宗找不到任何借口，而且这次烈日宗同样损失惨重，肯定不敢再轻举妄动了！”
<br/>　　楚星河说完这话，正暗自得意，却忽然两眼一瞪，死死盯着上空，眼眸中透着浓浓的难以置信之色，失声道：
<br/>　　“这——这怎么可能！”
<br/>　　原来，只见此时，九天之上，待那光芒消失之后，露出一道瘦削的少年身影，昂然立在空中，周身金芒万丈，恍然神灵！
<br/>　　不是叶尘，却又是何人？
