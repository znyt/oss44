　　　　楚风眠不想招惹事端，但是也不会人人欺辱。
<br/>　　“既然如此，那楚某就教训教训你！给我破！”
<br/>　　楚风眠大手一挥，一股灵力爆发而出，这一股灵力打出，便是铺天盖地，满是龙威。
<br/>　　那数道血气长鞭，顿时在这龙威之下，顿时全然破碎。
<br/>　　战龙龙威，克制一切阴邪之物，更何况这小小的血气，这蓝衣女子打出的血气长鞭，还远远不如楚风眠的杀戮剑术。
<br/>　　轻而易举，便是就被全数化解。
<br/>　　楚风眠站在原地，单单是大手一挥，一股灵力，便是将这蓝衣女子的无数血气长鞭全部化解。
<br/>　　这让无数围观众人看来，简直是不可思议到了极点。
<br/>　　那郑公子，都挨不过这一道血气长鞭，而楚风眠却是站在原地，便是如此轻易的化解了如此之多的血气长鞭。
<br/>　　这可远远不是御风境武者可以做得到的。
<br/>　　“此人是隐藏了实力。”
<br/>　　“此人的实力，居然还要压过这蓝衣女子一头，不是是哪个国度哪个家族中的天才。”
<br/>　　“不管如何，杀一杀这蓝衣女子的傲气也好。”
<br/>　　看热闹不嫌事大，无数围观无数纷纷议论道。
<br/>　　隐藏实力？
<br/>　　在那蓝衣女子的眼神中，她看的出来，楚风眠的境界的确就是御风境巅峰不错。
<br/>　　只是凭借这一身龙族武技，实在是太过克制她，令她的无数手段都失效。
<br/>　　龙族武技，的确是克制这等阴邪武技。
<br/>　　不过她可是神力境的人物，对付一个御风境的蝼蚁，居然都无法对付，反而被人轻易化解，这让她如何忍受。
<br/>　　“小子，你真以为凭借这龙族武技就可以跟我抗衡？今日我要让你明白，敢招惹我的下场！”
<br/>　　蓝衣女子啸叫一声，声音刺耳无比，如同嘶吼一般。
<br/>　　在她的身上，一股股的灵力猛然凝聚而出，刚刚的实力，居然还不是她的全力。
<br/>　　她的大手一挥，顿时周围无数武者都是惨叫一声，居然都被抽取了无数精血，全部凝聚在了她的手中，化为了一道血红魔爪。
<br/>　　在这蓝衣女子身边的数尊武者，都是纷纷倒下，他们的精血都没抽空，一个个变的虚弱无比。
<br/>　　这蓝衣女子的手段，极为狠辣，居然是直接抽取他人的精血，来填补她的力量。
<br/>　　这简直就是一尊魔头的作风。
<br/>　　“你！”
<br/>　　周围的几尊圣者，看到这蓝衣女子的举动，都是不由的大怒了，纷纷站出来，怒视着那蓝衣女子。
<br/>　　两人交手，还波及到其他无关人，这已经是引起了公愤。
<br/>　　“怎么？你们也想要来被我吸干精血吗？”
<br/>　　蓝衣女子不屑的看了一眼那几尊圣者，大手一挥，突然一条丹药长河出现，足足是有着数亿九鼎丹之多。
<br/>　　“拿着这些丹药，就算是给你们的补偿了，快滚！”
<br/>　　“你……”
<br/>　　那几尊圣者，接过丹药，犹豫了一下，还是带着那几名被吸干精血的武者离开了。
<br/>　　这蓝衣女子的实力，的确强大，不好对付，她既然出了丹药，这些圣者也不愿意跟她为敌。
<br/>　　“小子，这次我看你的龙族武技，还能不能跟我的血手抗衡！”
<br/>　　蓝衣女子看着楚风眠，怒喝一声，
<br/>　　一双血手，直接对着楚风眠，轰击过来，要将楚风眠的性命都给取了一样。
<br/>　　“住手！”
<br/>　　一声惊喝，突然凭空响起。
<br/>　　只见一尊身影，顿时从城主府内飞了出来，此人是一名中年人，器宇轩昂，气质非凡，显然是一方强者。
<br/>　　这中年人一出现，顿时大手一挥，便是将那蓝衣女子凝聚出的一双血手，直接打散。
<br/>　　“是南宫城主！”
<br/>　　“这泉城国度的主人，他居然都被惊动了！”
<br/>　　“不过这蓝衣女子的实力，已经是比得上许多巅峰圣者了，若是南宫城主不出手，只怕也没有人可以震的住她。”
<br/>　　一道道议论声响起。
<br/>　　“南宫城主？”
<br/>　　楚风眠一双眼睛，也是看向了那南宫城主，此人就是这泉城国度的主人。
<br/>　　“生死境。”
<br/>　　这南宫城主的境界，便是不折不扣的生死境强者，实力远超赵无敌。
<br/>　　按照力量指数来算，至少是有着二十点之多。
<br/>　　单纯论起力量，比起楚风眠还要强大一些。
<br/>　　不过楚风眠身上有着无数奇遇，更是有着远古战龙血脉跟造化之力的加持，真正战斗起来，楚风眠不说可以击败这南宫城主，也足以跟他抗衡。
<br/>　　“生死境强者。”
<br/>　　那蓝衣女子，看到南宫城主出现，觉察到南宫城主的力量，嚣张的神情也是收敛了许多。
<br/>　　生死境强者，她还是需要极为忌惮的，她虽然实力要远远超越寻常的巅峰圣者，但是生死境强者，再弱小，也可以轻易击败她。
<br/>　　在九域中，实力便是地位，面对强者，只能敬畏。
<br/>　　就算是如此嚣张的她，如今也要收敛起来。
<br/>　　“诸位都是青年才俊，都是准备去参加北邙学院考核的，也许还是同门，如今没必要在这里生死战斗吧。”
<br/>　　南宫城主脸上轻笑着，开口道。
<br/>　　“泉城国度中有着规定，可以切磋，但是必须要点到为止，诸位若是想要生死之战，离开这泉城国度随意。”
<br/>　　南宫城主开口，话语虽然还算是平静，但是其中的意思却是明白。
<br/>　　那边是在这泉城国度中，不允许生死之战，他出面，也是要让这战斗平静。
<br/>　　“哼。”
<br/>　　蓝衣女子听到南宫城主的话，不由的冷哼一声，撇了楚风眠一眼，冷笑道。
<br/>　　“既然如此，这一次就放过你一命！”
<br/>　　说完，蓝衣女子便是直接大步走向传送法阵，离开了。
<br/>　　她也明白，没有必要如今白白得罪一尊生死境强者，直接便是选择离开了。
<br/>　　“我们也走吧。”
<br/>　　楚风眠看了一眼旁边的寒月漓跟拓跋鼎，开口道。
<br/>　　他本来便是懒得跟这蓝衣女子计较，直接便是呼唤了他们二人，准备离开。
<br/>　　来到传送法阵中，楚风眠便是缴纳了一人一千九鼎丹，便是直接传送。
<br/>　　瞬息之间，楚风眠的面前便是天旋地转，一瞬间，便是前往了万里之外的国度。
