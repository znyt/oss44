　　千年之后，元始便不再多讲，而是停了下来。
<br/>　　不是元始小气不多讲，而是，就算将太多，他们也消化不了。
<br/>　　老子通天这两人刚生智慧不久，只能简单交流，听太多，反而不好。讲再多，他们也听不进去，讲了又有何用，不如不讲。
<br/>　　况且，元始自己还有事情要做，不可能一直在此为他们讲道。
<br/>　　“好了，此次讲道到此为止，你俩在此认真修炼，争取早日化形。我就先去洪荒逛逛。”元始对着两团元神说道。
<br/>　　“你……小心……”这时，元神之中在此传来一道断断续续的声音，显然，同为三清，这是在为他担心。
<br/>　　元始听罢，大笑道：“哈哈哈，放心吧，这洪荒虽然强者众多，但，我也不是吃素的。吾元始虽然不是最强的，但是，想杀我的人，恐怕还没出现呢！”
<br/>　　这点自信元始还是有的。语罢，便不再多说，简单布置一番，元始便走出山谷，向着洪荒大地走去。
<br/>　　洪荒初开，此时，凶兽与众大神之间的大战刚刚结束，最终结果是以五帝大神为首的荒古众神赢得此场战争的胜利，作为失败一方，凶兽皇神逆消失在太空之中，四大凶兽被封印在深渊之中。
<br/>　　同时，在这场战争之中，也有许多势力纷纷崛起，比如，以龙祖为首的龙祖，以风祖凰祖为首的凤凰族，以麒麟王为首的麒麟族，就是其中最大的三个势力。
<br/>　　之后还有一些，如霸虎族，青丘狐一族，百花族，灵族等百族也纷纷崛起，在这洪荒之中上演百族争霸的场面。
<br/>　　战后新生，洪荒大地生机勃勃，劫煞之气也消散殆尽，也许不是消散殆尽，而是在酝酿着下一场大劫。
<br/>　　洪荒之中，除了生灵演化，百族大战，但，真正的大能们却还没有出世，依然避世修行，等待时机的到来。
<br/>　　元始的邻居，昆仑玉京山，鸿钧道人的道场。鸿钧道人，头顶造化玉牒残片，造化之光玄之又玄，各种奥妙显现在面前，鸿钧道人徐徐领悟着其中的大道法则，整个玉京山都在大道希音的笼罩之下。
<br/>　　洪荒西部的灵山，罗睺盘坐在十二品灭世黑莲之上，周身四柄宝剑，缓缓旋转，是为诛仙四剑，一阵阵恐怖的黑气弥漫整个山峰，连带着周边亿万里都受到波及。这还没完，似乎还要想着向更远处延伸扩展，却感觉似乎还不到时候，微微有些收敛。
<br/>　　洪荒大海之中，无穷巨浪缓缓升起，再缓缓而落，隐隐之间有生灵在嬉戏。
<br/>　　……
<br/>　　整个洪荒都还沉浸在大战之后的整顿之中，根本不知道将来会发生什么事，只觉得现在非常美好，并不关心未来之事。
<br/>　　不过，这一切都与元始无关，因为他此刻正在往不周山的方向赶去。
<br/>　　洪荒第一神山，不周山！
<br/>　　来一遍洪荒，没有见识到洪荒第一神山不周山，就不算是一个很好是洪荒生灵。
<br/>　　身为洪荒之灵，怎能不见识一下天地玄黄，宇宙洪荒，太古第一，神山不周的风采？那岂不是太遗憾了。
<br/>　　元始并没有施展什么飞行之术，而是在一步一步的走向洪荒。不过，虽然没有用飞行之术，但一步数万丈，比起大多数飞行之术快多了，就连后世的筋斗云，一个跟头十万八千里，也比不上他快。
<br/>　　一步又一步，感受着这新生的洪荒天地，领悟着天地的奥妙，宇宙的神秘，感叹生灵造化的美妙。
<br/>　　就像悟道，也需一步一步地，不能太急。急了，就意味着不稳，不稳，就会跌倒。
<br/>　　所以，路要一步一步的走，道也要一步一步的修，修道路上，没有投机取巧。
<br/>　　感悟天地玄妙，领悟苍生造化，不知不觉之间，竟让他来到了一片庞大的山域之前，还未踏足进去，便有一股磅礴的气势，恢宏逼压而来，令他忍不住的为之瞬间惊醒过来，抬眼望去，诧然一幕，令人震撼！
<br/>　　天有多高？地有多厚？
<br/>　　元始曾经也不知道，但现在，他却是知道了，因为，眼前这座大山，会给出他最明确的答案。这一刻，他仿佛之间，看到了一个顶天立地的巍峨巨人！
<br/>　　那是一种难以言喻的磅礴大势，给他一最强烈的震撼，很难想象，天地之间，怎么会有如此一座山岳存在。
<br/>　　山高亿万丈，不知顶位何处？
<br/>　　尽管早有猜测，但，当元始看到这一幕时，还是忍不住的为之震撼，震撼这世间又怎会有如此造化！
<br/>　　元始仰望高山，顿生一股渺小如蝼蚁般的感觉，进入山岳范围之后，饶是以他堪比大罗金仙圆满的修为，却连越空飞行都做不到。
<br/>　　一股莫名的威压笼罩在这片山岳之间，好似禁锢了一切，让修行者不能使用任何神通，霸道绝伦至极。
<br/>　　“不周山！”
<br/>　　行到山脚之处，元始身形一滞，神色震撼之间，感应到一种冥冥流转的意志，告诉他这座山的名字。
<br/>　　“不周山！洪荒第一神山，果然名不虚传！也只有亲眼看到，才知道自己是多么的渺小！”元始震撼道。
<br/>　　“这还仅仅只是盘古以自身脊梁显化而成的天地支柱！”元始感叹道，
<br/>　　“仅仅一座山，就有如此之威，盘古，真不知你到底有多强？当初你究竟到了何种地步？对比盘古，什么天道圣人，又算得了什么！”
<br/>　　感叹之余，元始又想到了当初的盘古，当时盘古是何等自信，何等霸道至极，何等的骄傲，何等的强大，但如今，却只能身化洪荒天地。
<br/>　　这是命中注定，还是一切早有天意？
<br/>　　这是一个值得深思的问题。
<br/>　　思索之间，又不禁感慨，就连盘古这样强大的人都不能不朽，那，到底什么才算是真正的不朽？
<br/>　　圣人？天道？亦或是大道？
<br/>　　不，圣人虽然元神寄天道，天地不灭，圣人不死。
<br/>　　但，天地同样有劫，无量量劫，天地五衰，天力有时尽。天地同样有寿命，当天地寿命耗尽，试问，又有几人能够逃脱？
<br/>　　天地有轮回，那么，大道呢？大道是否也有轮回？
<br/>　　元始不禁想到这个问题，要是大道也有轮回，那么，我们所追求的大道不朽，还有什么意义？
<br/>　　思索许久，元始并没有得到自己想要的答案，他知道，自己现在还很弱小，这些问题也不是他能解决的。
<br/>　　况且，就算知道又如何？不知又如何？以他如今实力，什么都做不到。他只知道，想要不朽，想太多问题并没有什么意义。
<br/>　　他只知道，只有自己的实力更高，更强，才能知道能多，做到更多。
<br/>　　天地奥妙，值得自己穷极一生去探寻！
