　　“是了。不过，这件事暂且放下，等眼前大事处理之后，再去调查这少年的下落吧……徒儿，法术公会要为那对徒儿举办婚宴的事，你以为如何？”
<br/>　　伏明子调整了一下情绪，把之前有关叶少阳的困惑暂且放下，思考了一下，道：“大家都收到喜帖了。我以为，法术公会这么做，也是想要展现一下实力，这么多门派，一同赴宴，声势浩大，一来，可以让那些小门派感受到法术公会的强大，更加忠心耿耿，二来，也是做给我们这些人看，展现实力，对我们示威。三来，也是想试探我们的态度，如果我们赴宴，至少说明一定程度上不会跟他们作对……师伯以为呢？”
<br/>　　云春生缓缓点头，道：“我已与那几大门派商谈过。宴席，是要赴的。但是不能光赴宴，否则会让人以为我们也被法术公会收伏……”
<br/>　　“那当如何？”
<br/>　　“得在喜宴上制造点麻烦，杀杀法术公会的锐气，也显示我们的立场。”
<br/>　　伏明子沉默了一瞬，道：“这很危险。”
<br/>　　“是了，一不小心就会撕破脸皮……但我等实力不够啊。”
<br/>　　两人商量了一下，也没什么好办法，最后也只能约定明天再去找龙虎山等几个门派去讨论。
<br/>　　“徒儿，劝说桃花山弟子下山的事，怎么样了？”
<br/>　　伏明子摇摇头，“法术公会经我等交涉，也是同意让那些弟子离开，但是……不管我们怎么劝说，他们就是不走，铁了心要跟法术公会斗争到底……”
<br/>　　“凭什么？”
<br/>　　“是啊，凭什么？”
<br/>　　师徒二人先是看着窗外，随后目光落在对方脸上，都是一脸的困惑。
<br/>　　同一时间，山河道人，带着自己几个师弟、还有嫡传的徒弟，走上了桃花山的顶峰、曾经是他自己的地盘，现在却是李浩然的修炼之所。
<br/>　　李浩然手持一把笛子，站在山之巅，轻轻吹奏着。他吹的不是道风用来装比的那种长笛，而是最普通的那种，乡间牧童用芦苇刻成的短笛，吹奏的也不是什么动听的乐声，只是简单的音符，透着一种古朴和自然之意。
<br/>　　在李浩然的身边，亭子的一角，坐着一个身穿云衫的少女，听着笛声，目光深邃地望着远处。
<br/>　　山河道人一众人等，站在亭子下面，躬身低头，静静地听着李浩然吹奏完一曲，仍然不敢开口。
<br/>　　“当年，就是这把笛子……”李浩然抚摸着手里的短笛，“我主西出函谷关，一路之上，用它奏乐取乐，无丝竹乱耳，无笙箫忘形，只有最简单的音符，如春风拂山岗，如冬月临大江，清静无为，道法自然……你等今日有幸重听笛声，当有所悟。”
<br/>　　众人一听这笛子竟然是道祖太上老君用过的，齐齐跪在地上，心中既是惶恐，又是欣喜万分，想到之前李浩然吹奏短笛的过程，还有刚说的这番话，结合起来，各人都悟到了什么，但是千头万绪，却是无法抓住。
<br/>　　一个少年突然盘膝坐在地上，闭着双眼，一道清气，自周身而出，盘旋而上，汇成三花。
<br/>　　身边众人啧啧称奇，但是不敢言语。
<br/>　　李浩然走到这少年身前，伸出右手中指，在他眉心处点了一下，一道清光溢出，渗透进少年身内。
<br/>　　少年睁开眼睛，见是李浩然，笑了笑，道：“祖师，我悟了。”
<br/>　　李浩然点点头，道：“你叫什么名字？”
<br/>　　“上官雨晨。”
<br/>　　“好，你既悟道，便可修功法，从今往后，可随我左右。”
<br/>　　上官雨晨站起来，对着李浩然躬身行礼，随后转身望着山河道人，面色有些为难。
<br/>　　他是山河道人的嫡传弟子。
<br/>　　“造化，造化啊！”山河道人比上官雨晨更加激动，抓着他的双手，激动地说道：“既然祖师点化于你，这是你天大的造化，也是我桃花山的造化，你无需为难，一心跟随祖师修行道法，便是对我桃花山最大的报答！”
<br/>　　“谨遵师命。”上官雨晨拜倒在山河道人面前，起身之后，走到了李浩然身后。
<br/>　　李浩然面对众人，说道：“你等找我有事？”
<br/>　　“禀告祖师，法术公会联合诸多门派，已围困我桃花山，近日不断交涉，劝我等下山。”
<br/>　　“造衅一开，我无法护得你等周全，你等不走？”
<br/>　　山河道人等人一听，立刻现出激动的神色。山河道人拜倒在李浩然脚下：“祖师，我等近日来蒙祖师教诲，乃是天大的造化，纵然粉身碎骨，为祖师，为法术界，我桃花山上下数百弟子，一个不走，只求护得祖师金身周全！”
<br/>　　李浩然表情没有什么变化，转头问上官雨晨：“你以为如何？”
<br/>　　“生死事小，道法为大。朝闻道，夕死可矣。他们有幸霖听霖听祖师教诲，若有幸殉道，反而是他们的造化。”
<br/>　　山河等人齐声道：“正是如此！”
<br/>　　李浩然道：“虽如此说，你等可告知众位弟子，若想下山，绝不挽留，若不下山，或是一起殉道而已。你们去吧。”
<br/>　　山河道人和一众弟子领命离去。
<br/>　　李浩然转身对上官雨晨道：“你虽然开悟，但根基太差，我传你一套修行法门，你这就下山，寻个清静的地方闭关修行，此间事了，我去寻你。”
<br/>　　说完将自己的修炼法门面授给他，上官雨晨牢记之后，也不多说什么，躬身行礼，然后下山。
<br/>　　碧清走下亭子，来到李浩然身边，望着上官雨晨的背影，说道：“师兄觉得这孩子能成大器？”
<br/>　　“或许吧。看他的悟性和造化了。”
<br/>　　“这么说，师兄要带他去百年之后了？”
<br/>　　“自然。待我寻得叶少阳和徐福，我们便走。”
<br/>　　碧清转身望着山下，道：“不日之后，这里或许生灵涂炭，一片血海。这些我都不怕，若星月奴亲自来，该当如何？”
<br/>　　李浩然道：“我无惧于她，只是，这毕竟不是属于我的世界，我为法术界挡此一劫，但愿能拖住法术公会进军人间的势头……”
<br/>(本章完)
