　　三千年前……
<br/>　　听到这里，楚风眠的心中倒是没有在太多的惊讶，他所知道的一些远古遗迹，都是万年之前发现的。
<br/>　　这归墟天，既然是三千年前才出现的话，那楚风眠不知道，倒是也并不奇怪。
<br/>　　“北邙圣牌，你应该没有用吧。”
<br/>　　突然玄北圣者看向楚风眠，开口道。
<br/>　　楚风眠一愣，随后在他的手心之中，便是多出来了一个木制牌子，正是北邙圣牌。
<br/>　　这北邙圣牌，足以让任何人直接成为北邙学院的内门弟子，都不需要进行考核。
<br/>　　不过对楚风眠来说，这北邙学院的考核，简直是轻轻松松，所以他根本不需要动用。
<br/>　　更何况楚风眠也是从拓跋鼎哪里知晓，这北邙圣牌，还有着大用，所以他就一直放在空戒之中，只是如今楚风眠都快要忘记了。
<br/>　　提到玄北圣者的突然提起，楚风眠这才回想起来，这北邙圣牌，似乎正是可以兑换一个什么资格。
<br/>　　“果然，以你的资质，应该是用不到北邙圣牌，就可以来到北邙学院的。”
<br/>　　玄北圣者看着楚风眠手中的，木质牌子笑了笑道。
<br/>　　“本来我给你留下这圣牌，只是为了让你在走投无路的时候，来到北邙学院，没想到你倒是没有动用，不过正好，有了这北邙圣牌，就算是你去参加这次北域武斗大会，也不会有什么老古董说些什么了。”
<br/>　　“武斗大会？”
<br/>　　楚风眠的眼神中，露出了几分疑惑之意。
<br/>　　“不错，这归墟天，每年可以进入其中的名额，整个九域的名额，只有一百个，所以在我们北域，会举行武斗大会，拥有资格的武者，可以参加，其中的前十名，就可以进入到归墟天中。”
<br/>　　玄北圣者平静的解释道。
<br/>　　“这归墟天的资格，一般在我们北邙学院，也只有那些对学院有着贡献的弟子，才有资格去参加的，你刚刚晋升成为种子弟子，按理来说是不行的。”
<br/>　　“不过这北邙圣牌，却是可以直接换取一次圣地修行的资格，你将圣牌交出来，也算是对学院有着大贡献。”
<br/>　　玄北圣者笑着开口道。
<br/>　　“圣地修行，对你现在，已经没有多大用作了，不如就将北邙圣牌交出来，也正好可以去参加这次的归墟天的比斗大会，陆家的人，想要阻止你，也不可能。”
<br/>　　“全凭师尊安排。”
<br/>　　楚风眠直接将北邙圣牌，交给了玄北圣者。
<br/>　　对玄北圣者的话，楚风眠还是极为安心的，既然是如此安排，定然这是对楚风眠最为有利的结局。
<br/>　　“嗯，那些老古董，我去帮你说，过几日，应该就是前往武王城，参加武斗大会的日子了，你可以跟随其他弟子，一同前去，寒月漓也会跟你一起去。”
<br/>　　玄北圣者说道这里，眉头中突然露出一丝凝重道。
<br/>　　“不过你要小心，陆家的人，很有可能会从其中捣鬼，陆家的势力，远远比起你想象之中的可怕，而且你崛起的消息，肯定已经被地魔门，甚至是北域中的无数势力所知道，他们都会出手对付你，你必须要小心一点。”
<br/>　　“嗯。”
<br/>　　楚风眠点了点头。
<br/>　　他在北邙圣域杨威的消息，肯定也会很快的传到外面去，像是地魔门，跟北邙学院有着大仇，肯定会不惜一切代价，抹杀楚风眠的。
<br/>　　不过楚风眠心中倒也不担心，以他现在的实力，也没多少人，可以对付的了他。
<br/>　　更何况，要成为一代强者，哪里不是经历过无数次生死战斗的，一次次的战斗，只能是让楚风眠磨练。
<br/>　　“这次武斗大会，以你的实力，进入前十，应该不难，不过你也要小心，这武斗大会，只要是各大宗门中的弟子，都可以参加，其中不乏一些度过了生死大劫的天才。”
<br/>　　玄北圣者看着楚风眠，凝重的开口道。
<br/>　　“龙未央，陆画他们，虽然强大，但是毕竟没有度过生死大劫，在北域百绝榜中，只能算是中游，在其中的一些佼佼者，已经是度过生死大劫，实力已经足以媲美生死境巅峰，你可要小心。”
<br/>　　“嗯？”
<br/>　　这个消息，倒是让楚风眠微微有些诧异。
<br/>　　本以为这武斗大会，要对付的，只有神力境武者，没想到就连生死境武者，都有参悟。
<br/>　　一旦度过生死大劫，武者的力量都会数倍，十倍的提升。
<br/>　　像是楚风眠，虽然现在足以轻易击败龙未央，陆画，但是他们二人一旦是度过了生死大劫，只怕也足以对楚风眠造成威胁。
<br/>　　“我知道了。”
<br/>　　楚风眠点了点头道。
<br/>　　这个消息，如今被楚风眠知道，也没有让楚风眠任何惊讶，反倒是让楚风眠心中激起了战意。
<br/>　　他的战龙血脉，已经是开始影响到了楚风眠的性格，遇到更多强者，楚风眠才会更加兴奋，他的实力，才可以更进一步。
<br/>　　看着楚风眠充满战意的目光，玄北圣者不由感叹一声。
<br/>　　这种心智，才能有着真正成就强者的资格。
<br/>　　玄北圣者开口道。
<br/>　　“走吧，准备准备，过几日就准备前去武王城吧。”
<br/>　　“是。”
<br/>　　楚风眠点了点头，便是离开了宫殿之中。
<br/>　　在外，白萱已经离开，只剩下寒月漓一人，如今坐在地上，似乎是在参悟这灵气之中的恒古法则。
<br/>　　“楚风眠，我先离开去，去圣域中的种种地方，参悟剑术。”
<br/>　　白萱只是给楚风眠，留下了这一句话。
<br/>　　之前的战斗，也给白萱刺激不小，白萱本身也是一方天才，如今却是被楚风眠直接超越。
<br/>　　甚至是现在楚风眠的实力，都不是她可以想象得了的了。
<br/>　　来到了这北邙圣域之中，白萱便是离开，开始去参悟自己的剑术。
<br/>　　“这倒是也好，以白萱的资质，将来在北邙圣域中，也不是没有立足之地。”
<br/>　　楚风眠心中暗自想到。
<br/>　　他在北邙学院的势力，还太小了，白萱的实力可以再进一步，对楚风眠也是有着极大的帮助。
<br/>　　不过现在，实力最需要晋升的，还是楚风眠自己。
<br/>　　本来楚风眠现在的力量，身躯，都已经是到达了极限，但是楚风眠隐约之中还是觉得，还差一步，现在贸然引来生死大劫，还是有着极大的危险。
<br/>　　如今正好是足以前往归墟天，寻求奇遇，来令他的实力，再进一步。
<br/>　　倒时候只要楚风眠真正度过了生死大劫，踏上生死境，就算是面对那些生死台阶的老古董，楚风眠也不是毫无还手之力了。
