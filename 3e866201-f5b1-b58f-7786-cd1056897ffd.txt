　　夜晚的街道有些空旷，因为这次行动，古董咖啡厅附近的几条街道都被封锁了，一些居民，也早就被转移了出去。
<br/>　　本来在闹市中是最不适合进行这种行动的，因为很容易就能伤及到普通民众的性命。
<br/>　　但是这场行动，却是出乎意料的顺利，在转移民众的过程中，古董咖啡厅的那些喰种并没有采取任何措施。
<br/>　　就像是对他们的动作没有丝毫察觉一般，这让负责这次行动的总指挥有些疑惑。
<br/>　　不过这样也好，他最担心的事情，就是这群喰种会狗急跳墙了。
<br/>　　这些怪物，可没有任何顾虑，为了活着它们什么事情都做得出来。
<br/>　　大部队悄无声息的缓慢推进着，最前方负责探查的先遣队员已经到达了古董附近，正在观察着咖啡厅内部的状况。
<br/>　　“这就是喰种开的咖啡厅吗，和其他店面好像也没有什么区别。”什造跟在刘哔身边，悄声嘀咕道。
<br/>　　此时咖啡厅内黑沉沉的，只有路灯照射进来的些许光亮，让人看不清里面的状况。
<br/>　　各区的搜查官们，在各自负责人的带领下，缓缓将店面围住，配合上四周全副武装的士兵们，宛如一张大网一样，将附近的所有路线全部封锁。
<br/>　　几名先遣队的军人站在店门前，犹豫了片刻，正欲进入，动作却停顿了下来。
<br/>　　寂静的夜里，突兀响起一串清晰无比的脚步声，咖啡厅的台阶上，缓缓走下来两个带着面具的喰种。
<br/>　　一个像是红色的恶猿面具，狰狞无比，另一个的面具却像是黑犬一般，有些妩媚，更多的是让人感觉诡异。
<br/>　　看着这两张熟悉的面具，指挥官的脸色沉了下来，身为CCG的老人，他经历过无数战役，自然知道这两张面具代表着什么。
<br/>　　‘魔猿’和‘黑狗’，这两个消失已久的家伙怎么会在这里，请报上可是没有表明这一点。
<br/>　　更令他担心的是，这两名喰种的手下，会不会也隐藏在咖啡厅里面，如果真是这样的话，这次讨伐的结果就有些难说了。
<br/>　　“欢迎光临。”不等指挥官下达命令，魔猿和黑色杜宾立即出手，袭向了最前方的先遣队员。
<br/>　　顿时，激烈的枪声响起，向两名喰种战立的地方扫射而去。
<br/>　　但他们只是普通的军人，又如何能抵抗两名SS级喰种的联手攻击，只是一眨眼的功夫，先遣队的二十余名军人，就倒在了血泊中。
<br/>　　“第一梯队的搜查官进入战场，第二梯队和第三梯队警戒，特等队待命！”总指挥脸色有些阴沉，用对讲机中传达了命令。
<br/>　　随后，看着前方惨遭屠戮的特种军人们，指挥官眼神有些愤怒，双目充血的喊道：“敌人只有两只，宰了它们！”
<br/>　　命令刚刚传达下去，原本在一旁待命的搜查官们就立即开始了行动，看到同伴在自己眼前被屠杀，他们早就按捺不住了。
<br/>　　但作为军人，他们要服从命令，如果没有指挥官下令，即使前方先遣队的战友们全部死光，他们也不能有丝毫动作。
<br/>　　场中正在屠杀的两名喰种也稍稍停顿了下来，两双猩红色的赫眼，冷漠的注视着正向他们冲来的搜查官们，不知为何，所有人都能从那两双眼睛中看见嘲讽。
<br/>　　“他们说只有两只呢。”黑狗扭动了一下脖子，声音听起来倒像是个妩媚的女子。
<br/>　　“还真是迟钝啊。”魔猿抛了抛手上的黑色匕首，语气嘲讽的回答道。
<br/>　　随后，两名喰种冲天而起，战立到了一旁的房屋顶上，魔猿伸出沾染了些许红色的右手，打了个响指。
<br/>　　四周的建筑群顶端，密密麻麻的出现了两拨身影，这两拨喰种，都带着和魔猿以及黑狗一样的面具，就连穿着也都大致相同。
<br/>　　“魔猿，黑色杜宾，我就知道。”指挥官的脸色阴沉了下来，稍稍犹豫了一下，拿起对讲机，部署了起来。
<br/>　　看着周围很久没有一起并肩战斗过的手下们，魔猿心中微微有些触动，这些家伙，一直在等他回来啊。
<br/>　　可是，自己再也不想制造杀戮了啊。
<br/>　　叹了口气，魔猿扭动了一下脖子，将目光投向建筑群下方的人群中。
<br/>　　“我说你们，本领应该没有生疏吧？”看了一眼自己的手下，魔猿语气有些兴奋。
<br/>　　回答他的异口同声，他的手下和他一样兴奋。
<br/>　　而黑色杜宾一方，相比之下却安静了许多，黑狗静静的站在建筑上方，平静的开口：“你们，就为我而死吧。”
<br/>　　她身后那些和她穿着打扮一样的黑衣人们，纷纷应了一声，语气也是十分平静，似乎她们所说的事情，和生死无关。
<br/>　　死亡，永远战胜不了信仰，而首领，就是她们的信仰。
<br/>　　“大闹一场吧。”魔猿双手撑地，率先从高处跃了下来，随后，所有的喰种都开始行动了起来。
<br/>　　血花四溅，战场中枪声密集且杂乱，五颜六色的库克因在战场中纷飞着，每一秒，都有喰种或人类死去。
<br/>　　在这样的战斗中，性命似乎变成了一种十分廉价的东西，往日里CCG视若珍宝的精英搜查官们，此时都浑身浴血，像是一头头野兽一般。
<br/>　　二十区并没有被列入第一梯队，筱原幸纪早就不知道去了什么地方，现在负责指挥的是真户一等。
<br/>　　虽然二十区还有准特等搜查官参与了这次行动，但没有人敢去和真户吴绪争抢指挥权，毕竟，这家伙要是性格正常一些，或许现在都已经应该是特等了。
<br/>　　看着前方战场上腾挪跳跃的喰种们，真户吴绪握着密码箱的手微微颤抖，对于不能上场战斗，亲手屠杀那些肮脏的喰种们，他心中很是失望。
<br/>　　但指挥官没有下达任何命令，他也只能站在这里，和根木头一样傻等着。
<br/>　　刘哔将身体隐藏在众人身后，看着前方的战场，也是有些可惜。
<br/>　　这可是一个赚取积分的大好机会。
<br/>　　但是他现在还不能行动，芳村功善可是记得他身上的气味，现在时间太早，他还不能出场。
