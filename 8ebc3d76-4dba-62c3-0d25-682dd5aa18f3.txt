　　瓦斯汀！
<br/>　　这是毫无疑问的答案。
<br/>　　只要巴坦伦大选成功，顺利当选下一届总统，那么他第一个要除掉的人必然就是自己的竞争对手瓦斯汀，他绝对不会让自己的竞争对手从事中药植物，这无疑就是给自己埋下一颗定时炸弹！
<br/>　　如果瓦斯汀想要活命，就只能放手一搏。
<br/>　　“我觉得，我们不能把希望全都压在瓦斯汀的身上。”
<br/>　　齐凡在心中仔细地做过分析后，说道：“对于瓦斯汀来说，他其实还有另外一个选择，那就是放弃大选，在大选开始前就从这里逃离，这样就不存在跟巴坦伦竞争的说法，巴坦伦就能顺利的当选新一届的总统，而巴坦伦也不会花大力气再去追杀瓦斯汀。如果瓦斯汀有放弃大选，从这里离开的打算，那么我们就不能找他谈判。因为如果他已经打算明哲保身，我们再找到他，很有可能把谈判变成自投罗网，把我们这些人出卖给巴坦伦，换取自己一条命，这么合算的买卖我想巴坦伦是不会拒绝的，通过我们巴坦伦还能再从布克尔特那里拿到一大笔前，完成自己的大选。”
<br/>　　齐凡的分析十分在理，细思之下，这种情况也是有大概率发生的，大选在即，瓦斯汀在基本没有胜算的情况下还没有采取什么措施，他很有可能会放弃这次大选，直接外逃。
<br/>　　“这就有点难办了。”
<br/>　　布谷挠了挠脑袋，说道：“费了这么大的劲弄来的东西，居然没有任何作用，这也太让人郁闷了。”
<br/>　　“没有任何作用也未必。”
<br/>　　罗昊手指轻敲着桌面，说道：“既然瓦斯汀靠不住，那我们可以再找一个靠得住的人。比洛多，同样也是联合政府军中的实权者，这家伙是一个坚定的禁毒主义者，他的这种理念就跟巴坦伦相违背。如果巴坦伦大选成功，那么这个国家的cocaine就会被‘合法化’，再假设瓦斯汀会在大选前外逃，巴坦伦一旦当选为新一届的总统，他必然会把这个国家的军权牢牢的抓在自己手中，那么到时候，比洛多肯定会被巴坦伦清理掉。”
<br/>　　通过对比洛多的情况分析，包括莱昂在内的所有人都认为，寻求跟比洛多合作要比找瓦斯汀更合适。
<br/>　　“比洛多的女儿也在一所教堂里照顾那里的孤儿，我们可以通过比洛多的女儿来找到比洛多。”罗昊用铅笔在地图上圈出教堂的所在地，说道：“这应该是能最快找到比洛多的办法。”
<br/>　　罗昊没有带太多的人去教堂，第一，人太多目标太大；第二，没有那个必要。
<br/>　　罗昊驾驶着一辆皮卡车，皮卡车上带了莱昂、布谷、齐凡三个人；除了布谷携带了一把XM2010增强型狙击步枪外，其他三个人都携带了两把手枪，一把手枪中装得是麻醉弹，另外一把手枪中装得是空尖弹，配备三个备用弹匣。
<br/>　　皮卡车停在教堂门口，罗昊对着坐在后排的布谷使了个眼色，布谷推开车门走下汽车，背着装有狙击步枪的枪袋朝着教堂钟楼走去，钟楼的视野最开阔，可以看到教堂周围的情况。
<br/>　　教堂钟楼只有在打扫的时候，才会有人上去，平时的时候都处于关闭状态，也很少有人朝这边来，布谷没有费很多功夫就打开了教堂钟楼被锁住的门，沿着楼梯爬上了钟楼。
<br/>　　布谷摘下背在身后的枪袋，从里面取出狙击步枪，在枪口位置拧上消音器，装入一个压满被甲铅芯弹的弹匣，把枪架在钟楼窗口处，在窗台上还摆放着一个压着穿甲燃烧弹的弹匣，如果发生交战，这是用来破坏对方载具的。
<br/>　　即便这次是来跟比洛多谈判合作的，可还是要做好发生交火的预案。
<br/>　　“布谷鸟就位，完毕！”
<br/>　　布谷眼睛盯着狙击镜，左手绕过下巴按着插在右耳中的耳麦，通知皮卡车中的三个人可以进入教堂。
<br/>　　皮卡车中，三个人各自检查了自己身上的武器后，推开车门走进教堂。
<br/>　　教堂中的修女看了眼这三个陌生人，并没有引起什么怀疑，还跟三个人点头示意了一下。
<br/>　　“比洛多的女儿在教堂后面的院子里正在跟孩子玩耍。”
<br/>　　位于教堂钟楼上的布谷，依靠位置优势，很快就找到了比洛多女儿的位置。
<br/>　　罗昊、齐凡、莱昂三个人很快就在院子里照看着正在玩耍的塔尼特?比洛多。三人朝着塔尼特走去，挡在了她跟那些孩子之间。
<br/>　　塔尼特打量着站在自己面前的三个人，皱眉问道：“你们是谁？要找谁？”
<br/>　　“我们是专程来找你的，塔尼特小姐。”莱昂回答道。
<br/>　　塔尼特闻言，脸色一变，转身就要朝着自己身后的屋子跑去；但是却被罗昊拦住，“塔尼特小姐，别紧张，我们来找你需要你帮忙。”
<br/>　　“你们这些混蛋，别以为抓住了我，我的父亲就会向巴坦伦妥协，做梦吧！”塔尼特眼看自己逃跑无望，所幸也就不跑了，看着三个人说道：“我的父亲不会向你们妥协，我也同样不会向你们妥协！”
<br/>　　罗昊、齐凡、莱昂三个人对视了一眼，眼中闪过了然之色，塔尼特之前应该是受到过巴坦伦的威胁，而且还不止一次，所以在看到自己这些人的时候，才会有这么大的反应。
<br/>　　“塔尼特小姐，我希望你不要误会。”
<br/>　　罗昊抬起双手，向塔尼特做着解释道：“我们来找你并没有恶意，我们只是想要跟你的父亲谈些事情，而且我认为我们应该很容易就能达成共识。”
<br/>　　“我凭什么相信你？”塔尼特还是对自己面前的三个人，保持着怀疑。
<br/>　　“如果我们是巴坦伦派来的，就不会跟你解释这么多，而是直接动手了不是吗？”罗昊看着塔尼特说道。
<br/>　　教堂钟楼上，布谷通过狙击镜发现有两辆吉普车正朝着教堂的方向驶来，布谷抬起头朝着吉普车的方向看了眼，然后放大了狙击镜的倍率，看清楚两辆吉普车里的情况后，通过耳麦对罗昊说道：“狂龙，有两辆吉普车正朝着教堂的方向过来。”
