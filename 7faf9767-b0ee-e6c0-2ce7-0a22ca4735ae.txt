　　夜色苍茫，清风萧瑟。
<br/>　　一道人影隐藏其中，快速奔离。
<br/>　　人影正是萧逸。
<br/>　　稚嫩的脸庞上，不乏帅气，但以往那丝慵懒，此刻却荡然无存，只留下坚毅与成熟。
<br/>　　紫云城的月光，很亮；紫云城的夜色，苍茫中又带着如墨如画。
<br/>　　但他没心情观看半分，漫漫远方，正有一个人儿等着他。
<br/>　　他必须去追寻，即使不知路在何方。
<br/>　　....
<br/>　　炎武王国三十六郡，辽阔得让人难以想象。
<br/>　　弱小的武者，可能一生都无法完全踏遍这个王国所有地方。
<br/>　　北山郡，一百五十城，人口数以亿计，地域同样广阔。
<br/>　　萧逸的第一个目的地，裂天剑派，距离紫云城极为遥远，差不多要横穿数十个城池。
<br/>　　幸好，裂天剑派招收弟子之日，在数月以后，萧逸倒是有足够的时间赶路。
<br/>　　裂天剑派是一个门派，不同于家族等势力，每年都会定期敞开山门，有资质的武者只要通过考验，便能进入其中修习。
<br/>　　.....
<br/>　　“破玄城。”萧逸赶了好些天的路，最后在一座大城下停了下来。
<br/>　　看着那巨大城门上的三个大字，‘破玄城’，萧逸微微点了点头。
<br/>　　破玄城，是附近十数城中最有名也是最强大的城池，距离紫云城极远，彼此隔了五六座城池。
<br/>　　“离开紫云城这么远，北山慕容家的人就算要追杀我，怕也难寻到我的踪迹。”萧逸笑笑，走进了破玄城。
<br/>　　自己不过是个后天境，弱小的气息，顶多在空气中残留半天就消失了，根本不用担心。
<br/>　　走进破玄城，里面的景象让萧逸不禁大开眼界。
<br/>　　熙熙攘攘的路人，热闹繁华的景象，远非紫云城能比的。
<br/>　　行人中，不乏武者，凡境九重乃至后天境，萧逸遇到了不少。
<br/>　　甚至连先天境武者萧逸与看到了几个。
<br/>　　要知道，在紫云城中，凡境九重已经是三大家族的重要力量了，后天境更是贵为执事，先天则是长老，平日里养尊处优，极少外出。
<br/>　　哪里会在普通的街道中看到这些强大的武者。
<br/>　　无论是武者数量，抑或武者水平，破玄城都远超紫云城。
<br/>　　与这里相比，紫云城倒像个边缘小城。
<br/>　　萧逸闲逛了一会儿，最终在一座高大的黑色楼宇中停了下来。
<br/>　　高大黑色楼宇大门前，有一牌匾，龙飞凤舞地写着三个大字‘猎妖殿’。
<br/>　　牌匾上传来一阵骇人的血腥味，让人一阵心神紧张。
<br/>　　萧逸在破玄城停留的目的，就是因为猎妖殿。
<br/>　　猎妖殿，顾名思义，这是一个由猎妖师组成的势力。
<br/>　　传闻，这是一个非常古老的势力。
<br/>　　在无数年前的远古时代，妖兽横行，妖族肆虐整个大陆。
<br/>　　人类为了压制妖族，还天下安定，便诞生了猎妖师这个职业。
<br/>　　猎妖殿也是因此而出现，由最早的一批猎妖师联手创立。
<br/>　　传说，最早的一批猎妖师，尽是人族大能，个个都有滔天修为。
<br/>　　无数年后的今日，妖兽被驱赶到荒山野外、深山野谷，不敢随意冒头，人类得以在城池内安稳生存，猎妖殿功不可没。
<br/>　　猎妖师，是一份光荣的职业，代表着战斗、鲜血以及荣耀。
<br/>　　猎妖殿，有无数分殿，分布于炎龙大陆的各个角落，给予各个猎妖师很大的帮助。
<br/>　　北山郡内，有十座分殿。
<br/>　　大概每十数城范围便有一殿，给予这十数城池附近的猎妖师帮助。
<br/>　　在北山郡郡都，有一座大型分殿。
<br/>　　而在炎武王国的王都，则有一座主殿。
<br/>　　猎妖殿算是一个势力，但很松散，任何一个猎妖师都可以加入，而且没有太特别的限制。
<br/>　　无论你是一国的皇子，还是一个家族的子弟，门派的弟子，只要你是猎妖师，便能加入。
<br/>　　言归正传。
<br/>　　破玄城作为附近最强大的大城，自然有一座分殿。
<br/>　　萧逸走入猎妖殿，立刻被一阵阵喧嚣的声音给吵得耳朵生疼。
<br/>　　与外头看起来的古朴、庄严、严肃不同，里头吵闹得跟菜市场一样，倒让萧逸有些意外。
<br/>　　猎妖殿内很大，有专门的猎妖殿人员，也有不同的办事地方。
<br/>　　其中两个地方最为显眼，一个是‘悬赏处’，一个是‘任务处’。
<br/>　　悬赏处，顾名思义是给人发布悬赏任务的。
<br/>　　只要能支付酬劳，谁都可以到猎妖殿任何一个分殿发布悬赏任务。
<br/>　　发布的任务，会由工作人员审核，并收取发布者的定金，最终给任务评级，最后发布出去。
<br/>　　任务处，自然是给猎妖师们接领任务的。
<br/>　　各个任务都有评级，猎妖师可根据自身实力，选择适合自己的任务去完成。
<br/>　　完成任务后，悬赏金中的一成属于猎妖殿，剩下九成属于猎妖师。
<br/>　　殿内那些吵闹的声音，大多来自于聚集在任务处的猎妖师们。
<br/>　　萧逸看了眼殿内某个冷冷清清的角落，而后走了过去。
<br/>　　“这位武者，有什么可以帮到你的吗？”角落的一个办事处里，一道礼貌的声音响起。
<br/>　　这里是申请成为猎妖师的地方，也是为新手猎妖师解答疑惑的地方，几乎没人来这。
<br/>　　说话的，是一个二十出头的年轻女子，面容清秀，颇为礼貌。
<br/>　　“咦，是个少年？”女子看到来人是个少年后，微微惊疑。
<br/>　　“我想成为猎妖师，有什么要求吗？”萧逸淡淡地问道。
<br/>　　女子反应过来，微笑道，“少年，你家中长辈没有告诉你猎妖师是一门很危险的职业吗？哦，你是第一次出来历练的吧...”
<br/>　　萧逸皱眉打断道，“怎么，成为猎妖师需要查根究底不成？”
<br/>　　女子连忙道，“我不是这个意思。”
<br/>　　一般来猎妖殿的，都是些五大三粗，高大魁梧的汉子。难得来了个面容俊朗，气质不凡的少年郎，女子才打趣一番罢了，并没有恶意。
<br/>　　女子见萧逸表情冷酷，想来是个不愿多开玩笑的人，便认真地说道，“倒没有特别的要求，只需报上姓名就可以了。”
<br/>　　萧逸思索了一下，道，“易霄。”
<br/>　　萧逸并不打算用真名，毕竟现在被北山慕容家的人追杀，还是低调点好。
<br/>　　“容易的易，云霄的霄吗？”女子礼貌地问道。
<br/>　　“嗯。”萧逸点点头。
<br/>　　“呵呵。”女子微笑道，“轻易冲上云霄，你家中长辈想必是对你寄予厚望的...”
<br/>　　“额。”女子想起来萧逸不喜欢开玩笑，收起了话语。
<br/>　　“你等一下，我去给你准备令牌。”女子礼貌地点点头，缓步离去。
<br/>　　半晌，待女子回来后，交给萧逸一个黑黝黝的令牌。
<br/>　　女子道，“你只要注入体内真气，令牌便会自动记录你的气息。以后，这块令牌代表着你在猎妖殿的身份以及信息，你接任务和交任务都要使用它。”
<br/>　　女子继续道，“令牌还有其它功能，一旁有关于猎妖殿各种需要注意事项和信息的书籍，你可以随意观看，有不懂的，可以随时来问我。”
<br/>　　萧逸接过令牌，点点头，往十数米外的一排书架走去。
<br/>　　背后，年轻女子微笑低语道，“好冷酷的少年，不过也很帅呢。”
<br/>　　.....
<br/>　　书架前，萧逸翻了基本书籍，无非就是记录猎妖殿的历史事件，还有介绍等等。
<br/>　　萧逸权当是故事书，随便翻了翻遍放下了。
<br/>　　还有些书籍记录猎妖师的等级，还有各种事项等等。
<br/>　　猎妖师，分为九级，从低到高，分别是一级猎妖师、二级猎妖师....九级猎妖师。
<br/>　　再翻看了几本，萧逸在一本名为‘兑换图册’的书籍中停了下来。
<br/>　　猎妖师，可通过完成悬赏任务来得到猎妖殿的任务点。
<br/>　　任务点，可用于在猎妖殿中兑换各种物品。功法武技、丹药、材料等等，乃至珍贵的灵器、地阶武技等等。
<br/>　　猎妖殿，作为庞大而古老的势力，底蕴非常恐怖。
<br/>　　几乎只要是武者想要的东西，猎妖殿中都能兑换到。
<br/>　　前提是你拥有足够的任务点。
<br/>　　萧逸翻了几页，而后面露喜色，“果然，这里能找到培源朱果。”
<br/>　　这才是萧逸特地来猎妖殿的目的。
