        以前这跑腿的活儿，向来是交由赵斌去做。
<br/>
<br/>        现在赵斌在帮他做项目资料没空，既然她季思妍是过来当学徒的，那就让她先从跑腿学起吧！
<br/>
<br/>        季思妍一点都不介意这跑腿的活儿，拿起资料后，对傅昼景露出了一脸灿烂的笑容。
<br/>
<br/>        “嗯，我很快就会回来的！只要是你安排的，我做什么都乐意！”季思妍将资料抱在怀里，微笑着看着傅昼景说道。
<br/>
<br/>        傅昼景这才将目光从她的脸上收回，继续埋头忙着自己手里的工作。
<br/>
<br/>        刚刚，有那么一刹那晃神的时间，他恍若觉得自己又回到了学生时代，看到了她脸上那久违了的稚气未脱、天真无邪的笑容。
<br/>
<br/>        季思妍离开后，赵斌忍不住打趣地说：“傅总，这位傅大Boss介绍过来的季小姐，怕是对你有意思，你艳福不浅啊！”
<br/>
<br/>        “我已经有筱颖了，以后不要乱说话。”傅昼景头也不抬地说道。
<br/>
<br/>        赵斌立即识趣地闭上嘴。
<br/>
<br/>        但他总觉得，这季思妍和傅昼景之间，有一种说不清道不明的情愫在羁绊着他俩。

<br/>        季思妍到了傅氏集团总部大厦后，先去了一趟洗手间，站在洗手台上的镜子前，她解开了自己衣襟上的两粒纽扣，让自己的酥胸半露，并秀出了一条美丽的“事业线”。
<br/>
<br/>        在傅昼景的面前，她尽量让自己保持着五年前那清纯甜美的样子，而在傅夜沉的面前，她又不得不展现出自己最性感的一面。
<br/>
<br/>        傅夜沉是个什么样的男人？
<br/>
<br/>        听说，他年少的时候就曾打入过黑帮替警察当过卧底。
<br/>
<br/>        所以，傅夜沉一定什么样的女人都见过，要想吸引傅夜沉的注意，清纯甜美又保守内敛的风格可不适合他那么一个血气方刚的男人。
<br/>
<br/>        整理好自己的衣着后，季思妍又给自己补了妆，才拿起资料离开了卫生间。
<br/>
<br/>        她也只是听说，傅夜沉现在还只是代理总裁一职，并未正式继任，但傅氏集团的实质权利却差不多已经完全被傅夜沉掌握了。
<br/>
<br/>        无论她哥哥怎么暗中调查，他们也查不到傅夜沉的任何底细。
<br/>
<br/>        这样一个神秘莫测的男人，着实令他们有点儿束手无策，才会想到用“美人计”。
<br/>
<br/>        俗话说，英雄难过美人关……
<br/>
<br/>        季思妍一边想着，一边走进电梯，去了顶楼的总裁办。
<br/>
<br/>        出来接待她的男人，是傅夜沉的助理郑忠威，在这之前，他一直忠于傅老爷子傅炎廷，是个商场老将。
<br/>
<br/>        郑忠威接过季思妍手里的资料后，便让季思妍在外面的休息区里候着，压根就不给她接近傅夜沉的机会。
<br/>
<br/>        季思妍想进去，又不敢贸然打扰，只好耐着性子，坐在沙发上等着。

<br/>        她在沙发上坐了不到十分钟，见郑忠威还未将文件送出来，于是侧身卧在沙发上小憩。
<br/>
<br/>        季思妍故意让自己的睡姿既撩人又不失优雅，“V”字衣襟口，酥胸半露，呼吸间，呵气如兰。
<br/>
<br/>        当傅夜沉从办公室门口走出来时，压根就注意到那边休息区的沙发上还躺着一个人，而是自顾自地径直离开了。
<br/>
<br/>        他身后的郑忠威倒是将手里的文件，给季思妍拿了过去。

<br/>        季思妍耳闻身边有了动静，下意识地睁开眼睛，坐了起来。

<br/>        “季小姐，昨晚没睡好？”郑忠威一边随口问道，一边将手里的资料递了过去。
<br/>
<br/>        季思妍不好意思地接过资料，应了声：“谢谢。”
<br/>
<br/>        明明她是想引诱傅夜沉的，结果，自己的春光，全被郑忠威这只老狐狸给欣赏了去。

<br/>        郑忠威刚刚朝季思妍走过来的时候，确实无意间看到了她胸前那大片白花花的春光。

<br/>        其实，要怪，就怪她季思妍自己“骚”，明明衣襟口的几粒纽扣是可以扣上的，她偏偏不扣，故意酥胸半露地侧卧着，明摆着是在勾引人。
