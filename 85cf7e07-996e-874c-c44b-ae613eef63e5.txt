　　第69章
<br/>　　这是落叶城的上午，天气很好。
<br/>　　李大头总是起得很早，饭店是勤行，很早就要准备一天的食材。身为四海楼的掌柜，他自然要以身作则。
<br/>　　事实上，李大头出身并不好，纵然从小伙计熬到掌柜这个位置，也总是不那么让人看得起。后来，他又将自己妹妹送给苏四海做小妾，大家对他表面上的恭敬是有的，但背地里说的话就有些不中听了。
<br/>　　直到苏四海的原配死了，自己妹妹做了正房，自己的身份才算是水涨船高。
<br/>　　如今苏四海又病重，身子是一日不如一日，一天不如一天，只有能将苏樱赶出落叶城，那苏家的万贯家财以后就要姓李。
<br/>　　日后，自己也能配个员外的身份，和城里的贵人们坐在一起，吃吃花酒，吹吹牛。
<br/>　　本来事情进行得很顺利，苏樱得了一家濒临倒闭的酒楼，眼看落叶城就无她的容身之地。
<br/>　　可她不知从哪里凑到一笔钱，竟然将酒楼重新装潢起来，今天要重新开业。
<br/>　　开业！哼，一个小妮子懂怎么往酒里兑水么，知道怎么以次充好么，知道怎么克扣工人工钱么！
<br/>　　“小二，去对面借几副凳子，就说我们今天生意太好啦，凳子不太够用，先借他们几张。”
<br/>　　“那估计他们不会借给我们。”
<br/>　　“借不借的，都不重要，你只要记得说一句话就行。”李大头顿了顿：“告诉他们，反正你们也没生意。”
<br/>　　渐渐日头偏南，到了中午饭点，四海楼这几日的生意是极不错的。主要原因还是那辣椒的缘故，全天下独一份，城里贵人要想吃，只能到自己的店来。借着这个机会，李大头也很是结交了一批有身份的人，让他整日飘飘然，欣欣然。
<br/>　　“张员外，里面请，我今天可一早就等着您呢。”
<br/>　　一个颌下留着胡子的中年男人携朋友步入四海楼，他挥挥手：“要个雅间，上几道你们的拿手菜，我这几位朋友外阜的朋友，可就是奔着你们四海楼来的。”
<br/>　　说着，张员外回头冲他的朋友道：“这四海楼的辣椒可是一绝，你们虽然走南闯北，见多识广，我却是保准你们没尝过。”
<br/>　　“哈，早就听人吹过，是不是真的，今天可要耳听为虚，眼见为实了。”
<br/>　　“几位您二楼请，我立刻让厨房准备着。”
<br/>　　身为一家酒楼的掌柜，听得客人如此说，他总是有几分骄傲的。这时一旁的小二冲他使个眼色，将他拉到后厨。
<br/>　　“怎么会事，没看我正忙着么！”
<br/>　　“大掌柜，有件事……”
<br/>　　“什么事，说，别吞吞吐吐的！”
<br/>　　“辣椒没了。”
<br/>　　“什么没了！”李大头听清楚了，却还是问了一句。
<br/>　　四海楼的辣椒本都是苏樱带回来的，一共也没有多少，如同使金子那般使，却也撑不了多长日子。
<br/>　　“掌柜的，您快想想办法，许多客人可都是奔着辣椒来的。”
<br/>　　冷汗早已顺着李大头的脖子流下来，他嘴里恶狠狠的骂了一声：“小娘皮，好狠的心！”
<br/>　　然后，便急匆匆来到张员外的包厢。
<br/>　　“怎么，大头，饭菜准备好了，还劳你这掌柜的亲自送。”
<br/>　　“这个……那个，张员外，店里的辣椒用完了，您看要不要换个别的菜？”
<br/>　　“用完了……哈，我明白了。我都听说了，这辣椒是你们四海楼独一份，所以只卖给有身份的人，可你看我的样子，是拿不起钱的么，快快上来，钱不是问题。”
<br/>　　“真不瞒员外爷，是真没了！”李大头。
<br/>　　张员外眉头皱紧，脸上有些难堪，今日宴请的人，都是从外地远道而来的老饕，还是听了他的吹嘘，才特意过来。可如今他们来了，却吃不到菜，这让自己面子往那里搁。
<br/>　　“呵，你四海楼做得好生意！”
<br/>　　张员外不阴不阳道了一声，却是令李大头汗如雨下。他现在心中别个不恨，只恨苏樱，她肯定早已算到这里，先用辣椒把四海楼抬起来，等辣椒用尽，抬得多高就会摔得多惨。
<br/>　　真狠的女人呐！
<br/>　　张员外忽然抽了抽鼻子，脸上又笑道：“哈，李大头你给我开玩笑，这不是辣椒的味道嘛，还想糊弄我。”
<br/>　　李大头也抽了抽鼻子，雅间内的确漂浮着一股辣椒的味道。
<br/>　　“这味道，不是我们店的啊？”李大头。
<br/>　　“那会是从哪里来的？”
<br/>　　说着，张员外便站起来，渐渐走到窗口，突然一推窗户，外面响晴薄日，阳光下，对面街上一副红字招牌闪闪发亮。
<br/>　　『湖底捞』
<br/>　　那味道，正是从对面酒楼传来。
<br/>　　“走，哥哥们，咱们去对面看看吧。”
<br/>　　“张员外，您别走，本店拿手的醋鱼给您尝尝。”
<br/>　　李大头追着张员外离开雅间，突然发现，客人正纷纷离去。小二忙的像如何似的，却也留不住客人。
<br/>　　这些人都是往对面去的，他们奔着四海楼的辣椒而来，既然这里没有，自然去了对面。而其他客人其实不如何爱吃辣椒，不过人的好奇心说也奇怪，见到这么多人去了，自然也想去瞧瞧热闹。
<br/>　　如此状况，李大头就眼睁睁看着客人走光，他使出八臂哪吒的劲头也留不住客人。
<br/>　　只能在门口巴巴看着这些客人进入对面酒楼，门口的廖乙穿着黑衣，如同一座黑铁塔般杵在哪里。
<br/>　　据李大头听说，那叫保安。
<br/>　　“去吧，去吧，吃不痛快了，砸了你们的店才好。”
<br/>　　不多时，小蝶慢吞吞从对面走来，站在李大头面前。二人原先都在苏家做活，互相也都认识。
<br/>　　“干嘛！”李大头颇有一种来者不善的感觉。
<br/>　　“向你们借几张凳子，我们店生意太好了，凳子都不够用了。”
<br/>　　李大头倒抽一口凉气，身边小二把头低下去，这些话，正是他上午去对面说的。
<br/>　　“我绝不会借给你们！”
<br/>　　“不借就不借呗，那么大声干什么。”小蝶撇撇嘴：“反正你们也没有生意。”
<br/>　　“呃……”一口凉气往冲，李大头眼前一黑，差点跌过去，这话，也是他早上让小二过去说的。
<br/>　　
<br/>(本章完)
