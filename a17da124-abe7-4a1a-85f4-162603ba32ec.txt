　　紫霄宫中，在众修士那充满炙热嫉妒的眼神之下，道祖鸿钧赐下诛仙四剑以及阵图给通天。
<br/>　　之后，鸿钧看向女娲，微微沉吟，沉默片刻，继续道：
<br/>　　“女娲为大功德者，吾此处有极品先天灵宝山河社稷图一张与上品先天灵宝宝莲灯一盏，今日便交由你执掌。”
<br/>　　“弟子谢过老师！”
<br/>　　女娲接过山河社稷图与宝莲灯，向着鸿钧叩谢道。心中却极为不满，三清得到的都是先天至宝，轮到她时，却只给了她一件极品先天灵宝与一件上品灵宝。
<br/>　　将山河社稷图给了女娲后，道祖鸿钧便闭上双眼，沉默不语。见此，接引与准提心中暗恼，只好发挥自己的神技，来换取灵宝，只听两人对着鸿钧哭喊道：
<br/>　　“老师，我西方贫瘠不堪，没甚灵宝，望老师垂怜！”
<br/>　　“哎，罢了，罢了！接引，吾此处有十二品功德金莲一座，可镇压你西方气运，另赐你六根清净竹，用作攻击。准提，吾赐你七宝妙树，先天之下，无物不刷，再赐你金刚神杵，为攻击灵宝，望你二人慎用之！”鸿钧有些不耐烦道，接着将一朵金莲，一根翠竹，一棵七色小树和一杆金杵交给二人。
<br/>　　如此，亲传弟子与记名弟子的差别就显现出来了。元始心中冷笑，通天目露不屑，老子面无表情，其他诸多修士除了有些好笑之外，更多的是心中难掩的火热，以至于他们看向道祖鸿钧，眼光火热无比。
<br/>　　“鸿钧不公，何至于如此待我西方？”接引与准提二人表面上感激涕零，连连叩谢，可心中却是早已怒骂不已，骂鸿钧不公，谁都能看得出来，他俩的灵宝与三清、女娲那里，根本没法比，仅有十二品金莲一样极品灵宝，还威力不强，主要用于防御。其他的都是上品先天灵宝，更是不堪。而三清这几个亲传弟子，不仅有先天至宝，更有诛仙剑阵这样的杀伐至宝。
<br/>　　元始心中冷笑，就你西方贫瘠之地，还想要鸿钧赐下灵宝？若不是当初为了偿还因果，说不定你们连个座位都没有？呵呵！
<br/>　　“禀告道祖。”
<br/>　　就在此时，坐在中间的东王公大声喊道，“洪荒之中，时有大战，众多修士遭劫，吾愿创立联盟庇护天下散修，还请道祖恩准。”
<br/>　　“既是如此，那么，东王公今后你为洪荒男仙之首，赐你先天灵宝天阳神剑。”也不知道祖是怎么想的，一挥手便将一柄赤色神剑交给东王公。
<br/>　　“西王母，今后你为洪荒女仙之首，赐你极品先天灵宝昆仑镜，此镜又名天机镜，能够洞察过去、未来天机，内蕴空间法则，能够穿梭虚空。今后你二人执掌天下散修。”鸿钧又对着东王公身旁的西王母淡淡道，之后将一面宝镜交给她。
<br/>　　“多谢道祖。”
<br/>　　两人满脸兴奋激动之色，对着道祖感谢道。他们却是不知，此刻有多少人已经将他们记恨在心了，天下男仙、女仙之首，那岂不是说，我们也要受他们管了？哼！
<br/>　　“弟子等修为浅薄，还请老师赐予些许灵宝，以作防身、斩尸之用。”
<br/>　　眼见分了灵宝之后，鸿钧便闭上双眼，不再言语，有修士顿时便大叫起来，其他人见此，也急忙开口，请求鸿钧道祖赐下灵宝。
<br/>　　鸿钧见此，只好无奈道：“罢了，紫霄宫外的分宝岩上，还有许多先天灵宝，汝等待之后贫道合道之后，便可前去，试试机缘。”
<br/>　　“啊！弟子等叩谢老师，老师慈悲！”众修士一听，还有灵宝？纷纷大喜，拜谢道。
<br/>　　“尔等可有疑问，趁如今贫道还未曾合道，吾为尔等解答。”鸿钧淡声问道。
<br/>　　众修士都看向元始，元始作为大弟子，理当由他开口提问。
<br/>　　“敢问老师，没有鸿蒙紫气，可否成道？”元始脸色淡然，问出了一个他已经知道，但众人都十分想知道的问题。
<br/>　　果然，此话一出，众修士纷纷将目光放亮，紧盯着道祖，看他如何回答。
<br/>　　“可！”道祖深深看了元始一眼，回答了一个字。
<br/>　　啊？真的可以？众修士兴奋不已，道路未绝，他们只觉得自己还有机会。可是他们却不知，没有鸿蒙紫气，想要成道，却比有鸿蒙紫气要难上千倍万倍。
<br/>　　“敢问老师，是否一切早有注定，不可改变？”元始继续问道。
<br/>　　鸿钧再次深深看了一眼元始，淡淡道：“小势可改，大势不变。”
<br/>　　元始若有所思，鸿钧这话可就值得自己深思了，何为小势？何又为大势？大小一般都是相对的，谁也分不清，只有找这两者之间的那个衡量点，然后把握住它，不要太过，一般就无事了。如今他要做的便是，找到那个衡量点。
<br/>　　“敢问老师，成道之法几何？”老子思索了一番，提问道。这一问一出，众修士再次提起耳边，仔细听道祖的回答，深怕错过什么。
<br/>　　“三千大道，条条可证混元。但吾总结了三条，一者，为最强者，以力证道，二者，为中等，斩三尸证道，三者，为下等，功德证至人，成就功德圣人，法力最为弱小。”
<br/>　　“敢问老师，以力证道可曾有人成功过？”通天急忙开口问道，以力证道，法力最强，他首选的，当然是以力证道了。
<br/>　　“盘古就是以力证道者，但他最后还是失败了，以致于身化洪荒万物。”道祖看了一眼通天，淡淡道。他可不会说盘古差点就成功了，要不是有人偷袭以及与三千魔神大战，盘古已经成功了。
<br/>　　啊？众人心中苦涩，连盘古都失败了，他们岂不是也注定没戏？看来，需要想一想第二条路，或者功德成圣，可是功德是那么好得到的吗？
<br/>　　“敢问老师，合道之后，老师是否为天道？”女娲身为女子，为人比较心细，问出了一个比较深奥的问题。
<br/>　　“合道之后，鸿钧为天道，天道不为鸿钧。”
<br/>　　鸿钧给出了一个模棱两可的答案，众人心中仍有不解。在元始看来，这话是说，鸿钧合道之后，鸿钧就变成了天道的一部分，所以说，他是天道。但天道又不仅只有鸿钧这一部分，它还有其余很大的部分，故而，天道又不为鸿钧。
<br/>　　“老师合道之后，吾等若是有事禀告，该如何面见老师？”一向不喜开口说话的接引在此时开口问道。
<br/>　　“量劫不出，鸿钧不现！好了，贫道要合道了，尔等能否抓住此次机缘，便看你们的了。”鸿钧回答了接引的问题之后，看到还有人想要提问，便直接打断，开口道。
<br/>　　“老师慈悲！”众人大拜道。
