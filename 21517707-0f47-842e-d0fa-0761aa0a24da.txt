　　第175章
<br/>　　可以兑换一个顶级人物了。
<br/>　　夜幕，山寨到处都是篝火和火把，在战时，白天和分界线并不明显。
<br/>　　程大雷一人来到点将台，揖手三拜，一拜天，一拜地，一拜……系统。
<br/>　　系统大大，你可怜可怜我吧，给来个猛人，让我撑过这场难关。
<br/>　　这次不是赌脸，干脆就是赌命。
<br/>　　“点将。”深呼吸后，程大雷口吐两个字。
<br/>　　点将台上的篝火像是亮了几分，星辰光芒像是此刻汇聚，嘟地一声，程大雷收到一条系统提示。
<br/>　　嘟，获得顶级武将——豹子头，林冲。
<br/>　　程大雷整个人一激灵：林教头要来了。
<br/>　　八十万禁军总教头，梁山第六把交椅，天雄星林，号称水浒马上第一。
<br/>　　程大雷拍着激动的胸口，果然是好人有好报，看来这次赌脸是赌对了。说不得，这次也能逢凶化吉，转危为安。
<br/>　　可突然间，程大雷又想起一件事，脸色变得有些难看了。
<br/>　　现在戎族围城，将山寨围得水泄不通，这林教头便是有滔天本事，他要怎么进来呢？
<br/>　　程大雷看天上，天上无有。
<br/>　　看周围，周围空荡荡的。
<br/>　　看地上，地上没有长出来。
<br/>　　这林教头该怎么出现在自己面前，若他从外面进来，岂不正面对戎族大营，林教头本事再大，面对千军万马也抓瞎。这要是闯营时被戎族杀了，自己岂非白白折损一员大将。
<br/>　　程大雷又在点将台等了会儿，没有收到山寨升级的提示，然后黯然离开。
<br/>　　林教头是很重要，但更重要的是随他而来，山寨将会生到三级。目前自己是空有三级山寨的建筑，却无三级山寨的附加属性。
<br/>　　就等着林教头来呢！
<br/>　　可程大雷千盼万盼的林教头究竟在何处呢！
<br/>　　……
<br/>　　北蛮部大营。
<br/>　　大帐内，各部落头领都在，赤眉坐在帅位，不再像往日那般慵懒无所谓，严肃的板起面孔。
<br/>　　连续两次攻打蛤蟆寨，结果都是无功而返，白白抛下许多条人命。有些小一些的部落，已经全员阵亡在这次战斗中。
<br/>　　这自然在军中造成不好的影响，有些头领已经开始怀疑赤眉的指挥能力。
<br/>　　北蛮王有诸多儿子，赤眉绝不是最有威信的一个，在北蛮部内，支持他的人可以说寥寥无几。
<br/>　　“早知道死这么多人，当初打完黑石城就该走，反正我们也抢了不少东西。”
<br/>　　“忽律雄，你现在说这话，难道怕了！”
<br/>　　“你才怕！”忽律雄腾地站起来：“我九岁就能杀人，你那个时候还骑着马崽子呢！”
<br/>　　“你说什么！”
<br/>　　两人剑拔弩张，当场就要在大帐内动起手来。
<br/>　　“好了，好了。”
<br/>　　赤眉无奈的挥挥手，手下都是这样的蠢货，队伍还真是不好带啊。他们平时就不能多读读书，当然，他们也不认字，戎族也没有文字。
<br/>　　怪不得帝国叫你们蛮夷……赤眉叹口气，若自己活在帝国的花花世界该有多好，不用每天见这些粗鲁的野蛮人。
<br/>　　可接下来的仗该怎么打，赤眉心中也没有办法。他忽然觉得，带兵打仗也不是一件好玩的事情。
<br/>　　咚咚咚。
<br/>　　营外忽有战鼓响，大帐内的诸人一惊：敌人打过来了。
<br/>　　这可是从未有过的事情，据城坚守是帝国作战的习惯，真正野战是戎族擅长的领域，他们真的敢冲过来。
<br/>　　只闻战鼓响，却不见有人冲过来。心里不知怎么回事，赤眉带着众部落头领离开大营，在距离城门六百步停在。
<br/>　　“十王子，不能再往前走了，小心他们的暗箭！”托蛮拦住赤眉。
<br/>　　赤眉点点头，远远望着城头，只见城头上有人用力捶鼓。
<br/>　　“他们干嘛呢？”赤眉困惑道。
<br/>　　这时听得城门上传来喊声。
<br/>　　“长生天的子孙，草原上的勇士们，素闻你们戎族有尚武之风，以骑术名扬天下。我家李……我家程将军说了，你们戎族的武技不过是野兽的厮杀战法而已，真正的堂堂武道还要帝国教你们。今日两军对垒，我家程将军不忍看两家士兵白白牺牲，程将军愿与戎族的勇士单枪匹马较量，阁下若能胜过我家程将军，我家程将军愿打开城门，举手投降。”
<br/>　　声音远远传来，赤眉的眉头皱起，令人向城头喊道：“你们是什么人？”
<br/>　　战到现在，赤眉还没弄明白与自己交手的是那方势力。
<br/>　　咦，原来戎族不是这么说话的呀，自己也不用费这个劲了。
<br/>　　“大当家，他们问咱们是什么人？”喊话的山贼道。
<br/>　　问这个呀，程大雷从城墙垛子下站起来，整了整腰上的三枚帅印。
<br/>　　“喂，戎族的渣渣们，你们听清楚了。”程大雷深吸一口气：“吾乃是青牛山陷阵营游骑将军，帝国落叶城城主，幽州王印执掌者，仙人传艺的神斧将，守卫青牛山的蛤蟆大王。”
<br/>　　程大雷呛得直咳嗽，这没有一个好的肺活量还真上不来。
<br/>　　赤眉一行人面面相觑：这货到底是谁！
<br/>　　“程将军斧下不死无名之鬼，你们是什么人？”
<br/>　　赤眉眼神示意托蛮，当真不可落了自家威风。
<br/>　　托蛮点头示意自己明白，高呼一声道：“赤眉！”
<br/>　　“……”赤眉。
<br/>　　没文化真可怕。
<br/>　　这时候，城门缓缓打开，一员大汉手持长枪，座骑五花马从城门内走出。
<br/>　　“秦蛮在此，那个枪下受死！”
<br/>　　两军对垒，猛将单挑的事情其实是很罕见的，但不是没有可能发生。看今天这状况，显然程大雷是想来这一套了。
<br/>　　“那个去斩了他！”赤眉看着手下众头领。
<br/>　　“让俺去杀了这贼人！”
<br/>　　“三个回合内，我要将他斩于马下！”
<br/>　　赤眉以眼神示意托蛮，这是最支持他的手下，他急需要一场胜利，来巩固自己的地位。
<br/>　　此时，托蛮还未搭言，就见一员虎将冲杀了出去。
<br/>　　赤眉一愣：怎么是他！
<br/>　　程大雷一愣：怎么是他！
<br/>　　这员将骑马冲到城下，举着狼牙棒大叫。
<br/>　　“姓程的，你给俺滚下城来，让我一棒敲碎你的脑袋！”
<br/>　　此人怒发冲冠，双目沁血，显然和程大雷有深仇大恨。
<br/>　　此人不是旁个，落叶城，杜茂。
