　　待欧阳紫嫣把御剑诀的心法牢记在心中，并且在体内运转一圈后，慕清霄才任由邪火滋生，直接饿虎扑羊的方式，将她扑倒在浴室中。
<br/>　　“嘤，老公，我待会要去集团。”欧阳紫嫣惊呼一声，哀求似的说道。
<br/>　　“晚一点没事。”
<br/>　　“可是上午还有一个商务会谈，我已经答应对方了。”
<br/>　　此时，慕清霄已经提枪上马，自然不可能中途停下，直言会语道：“推迟无妨......”
<br/>　　...
<br/>　　...
<br/>　　午时，慕清霄躺在游泳池边的木椅上，闭目养神，脑海中却是思绪万千。
<br/>　　通过位面的历练，他都会总结一些经验，战斗技巧，以及接下来自己所需要的东西。
<br/>　　他现在的修为是元婴六重中期，但想杀元婴六重的普通修士，却是轻而易举，哪怕是十位，数十位元婴六重修士围杀他，他都能全灭对方。
<br/>　　不仅是心法的原因，更是资质与肉身，血脉上的差距，他可是人形凤凰，而且还不是普通的凤凰！
<br/>　　战斗他能越级杀敌，但是战技却是一个都没有，也不是说没有，而是战技的等级，已经跟不上他的修为的提升，满足不了他心中的要求。
<br/>　　比如海贼位面所获得的‘剃’，也就是普通的瞬身术。
<br/>　　在低武位面倒是所向披靡，但到了仙侠位面，差距一下子就体现了出来，对于现在的他来说根本派不上用场。
<br/>　　要知道，他现在的修为，瞬息千米根本不是事。
<br/>　　低武位面的东西，其价值毕竟有限，还有火影位面的‘八门遁甲之阵’当初能增幅大半个境界，可现在的增幅效果却是微乎其微。
<br/>　　倒是‘永恒的万花筒写轮眼’，由于九霄琉璃焱的缘故，间接的成为了媒介，精神力量在不断变强，瞳孔也在逐渐随之变强。
<br/>　　或者说，他现在说拥有的，已经不能在称之为写轮眼，只是这样称呼，比较方便而已。
<br/>　　现在的写轮眼，虽然继承了其中的强大幻术，却已经进化为更高层次的眼瞳，而且随着九霄琉璃焱的成长，写轮眼也会不断的增强。
<br/>　　最让慕清霄感到诧异的是，在他踏入人剑合一的境界后，忽然发现，战技‘暴风剑’的威力，也已经满足不他了。
<br/>　　他现在一剑下去，轻轻松松劈开一座大山，剑意在提高，剑气随心念而生。
<br/>　　而暴风剑所展现的威力，远没有当初带给他的震撼大，照现在看来，这门剑法也只是让自己入门而已，并非什么高级剑术神通。
<br/>　　他就是这种洒脱的性格，修仙一途本就要往前看，这些功法已经没有任何价值，自然也没有在修炼的必要。
<br/>　　若是继续修炼下去，不仅是浪费时间，更会阻碍他前进的脚步。
<br/>　　当然，有些地方倒是能够利用起来，想到这里，慕清霄搅动灵气，‘八门遁甲之阵’修炼肉身的方式，便以功法的形式浮现在泳池水面上。
<br/>　　“系统，这门功法值多少掠夺积分？”
<br/>　　“叮，‘八门遁甲之阵’归为下等练体功法，价值300掠夺积分，功法回收完毕。”
<br/>　　系统提示声落下，慕清霄嘴角牵起一丝笑意，果然如他所料的一样，全当是废物利用，还能换取一些掠夺积分，何乐而不为呢？
<br/>　　于是，慕清霄照着刚才的方式，不断将心中所有的心法抄录出来，系统也报出一个个不等的价格。
<br/>　　当他将经历所有位面的心法抄录出来后，居然得到了整整三万掠夺积分，这可是经历一个低武位面的奖励啊！
<br/>　　然而，让他诧异的是，并不是所有的功法，系统都会收录，就比如低武位面的拳法，腿法等等，都是一文不值，这让他颇为失望。
<br/>　　而仙侠位面，他在青云门中所获得的心法，价值几百到几千不等。
<br/>　　将普通心法战技抛之脑后，慕清霄呢喃道：“现在战斗，光是剑气，剑意领域，幻术，九霄琉璃焱已经够了，至于强大的剑术就要看机缘了。”
<br/>　　躺在木椅上，沐浴着春风，享受着悠闲的时光，慕清霄忽然想到了什么，心念一动，一块完美的玉石出现在他的手中。
<br/>　　这块玉石，便是青云志位面结束后的奖励，三选一中的其一。
<br/>　　握着凉凉的玉石，慕清霄将它在手掌中翻来覆去，甚至划破指尖，滴上血液，玉石就是没有动静，再也没有出现，当初凭空浮现的熟悉气息。
<br/>　　琢磨了半年，他都没有琢磨出个所以然，苦笑一声，早知道如此，当初还不如选择圣灵丹，但事已至此，现在懊悔也没有任何的办法。
<br/>　　将玉石重新放入储存空间中，慕清霄全身心放松，躺在木椅上静静的沉睡过去。
<br/>　　此时，玉石漂浮在偌大的储存空间中，没有一丝瑕疵的表面，竟浮现一道模糊的划痕，划痕成十字架形状，划痕却长短不一，看上却有些像是......剑印！
<br/>　　剑印来的快，去的也快，转眼间就消失不见，玉石也恢复以往的模样。
<br/>　　然而，熟睡过去的慕清霄，并不知道储存空间中，玉石所发生的一幕。

<br/>　　...
<br/>　　下午的时间总是过的很快，待慕清霄睁开双眼，已是夕阳西下，落日的余晖照耀着大地，将天边的云彩染成色彩缤纷，极其绚丽。
