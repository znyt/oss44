　　慈心师太等一干人，傻傻地看着地上几人的尸体，有人流下了眼泪。
<br/>　　有人甚至想，如果重新选一次，自己一定与他们并肩作战，死在一起。
<br/>　　你是想做三分钟的英雄，还是一辈子的懦夫？
<br/>　　可惜，他们已经选了做懦夫，一念之差，便失去了当英雄的机会。
<br/>　　星月奴对他们的默不做声很满意，这才转身来到香炉前，看了一眼道：“还有半柱香。”
<br/>　　冲慕寒笑了笑，“你想清楚了没有？”
<br/>　　一时间，龙虎山上的邪物，还有那些弟子，都朝他看过来。他们没有动手，就是为了等一个答案，他毕竟是张天师的嫡传后人，是龙虎山的主人，他们忍到现在，一切都是为了龙虎山的基业，一切都是为了一个答案。
<br/>　　“如何！”
<br/>　　星月奴口吻严厉起来：“慕寒，龙虎山千年基业，就要毁在你手上了，灵气散尽，世间再无龙虎山！”
<br/>　　世间再无龙虎山……
<br/>　　天呐，这一切为什么要我来选择？
<br/>　　慕寒浑身战栗，眼睛望着那烧剩下的半柱香，内心做着艰难的抉择。
<br/>　　星月奴静静地等待着，她不着急。
<br/>　　她相信自己的判断。
<br/>　　“掌门师兄，她不敢的，灭了龙虎山的香火，会引起人神共愤，他们无法在龙虎山立足！”曲波上前抱住慕寒，用力摇晃起来，声嘶力竭地吼着，“龙虎山从没给人当过狗，若归顺于她，就算香火延续，龙虎山的精气也没了，掌门啊！”
<br/>　　慕寒不为所动，抬头望着星月奴，木然说道：“你要我怎么做？”
<br/>　　“路在前方，你自己选择。”
<br/>　　“我……为什么一定要选……”慕寒失神地望着前方。
<br/>　　“你是龙虎山掌教，你不选谁选！慕寒，香要烧完了！”
<br/>　　我是掌教，所以我要选……
<br/>　　慕寒突然收回目光，望着她，说道：“我若是普通人，就不用选了？”
<br/>　　这句话倒有点出人意料，星月奴皱了皱眉道：“可你不是普通人。”
<br/>　　慕寒突然笑起来。“我现在是了。”
<br/>　　所有人都疑惑地看着他，不少人心想他不是扛不住压力所以疯了。
<br/>　　慕寒从兜里摸出一方印章，冲星月奴摇了摇，然后塞到一旁傻站着的曲波手中。“我宣布，我退位了，我将掌教之位传给你，这一刻起，我不再是龙虎山掌教。你怎么选，都随你，与我……无关了。”
<br/>　　说完，他十分得意地大笑起来，冲星月奴大声说道：“我现在不是龙虎山掌教啦，大家都看到了，我现在是个普通人，你如果想杀一个什么都不是的普通人，你就动手吧，你不动手，那就再见了！”
<br/>　　说完大步朝山门走去。
<br/>　　星月奴愕然地望着他的背影。
<br/>　　她有点失望。
<br/>　　慈心师太一干人也很失望。
<br/>　　龙虎山的人，最失望。
<br/>　　如果他为了龙虎山的香火，屈服于星月奴，虽然会被人骂，但总归也是做了一件事，保全了龙虎山的香火，将来也还有忍辱负重从头报仇的机会。
<br/>　　如果他坚韧不屈，号召门人一起战斗，大伙也会义不容辞地跟着他上，哪怕粉身碎骨，哪怕世上从此再也没有龙虎山这个门派。
<br/>　　只有这两条路，每一条都不容易。
<br/>　　但他却选择了第三条路：他逃避了，放弃了本该自己担负的责任。他让大家都失望了。
<br/>　　他做到这个份上，星月奴不可能再对他动手，大伙就这么呆呆地看着他走出了道观的大门。他也没回头。没人知道他现在心里在想什么，也没人在乎了。
<br/>　　“混蛋，你个混蛋！”
<br/>　　曲波的声音穿透了寂静，望着他消失的背影大声喊叫起来。
<br/>　　“还有什么好说的，跟他们拼了！”
<br/>　　老正也忍不住了，大吼一声，扑了上去，身后数百邪物，也都一起冲上去围攻三人。
<br/>　　星月奴三人并不着急，三人一起结印作法，将法力汇聚在轩辕剑上，一时间放出数百道剑气，将所有邪物都逼到院子一个角落，然后剑气降落，如同一根根栅栏，在他们面前插下去，将他们尽数围了起来，形成了一道剑气组成的结界。
<br/>　　轩辕剑强大的灵力，被三人完全激发，结界之强，令这些邪物一时间无法破坏，一个个蓄力作法，有邪物对三人高声叫骂。
<br/>　　“那么……”星月奴转身面对龙虎山众弟子，目光从他们脸上一个个扫过去，最后还是落在曲波的脸上，道：“既然你手握掌门大印，你便是掌门了，至少是代理的，那便有你来决定吧。”
<br/>　　曲波怔怔地望着手中的掌门大印，猛然跳起来，结印朝星月奴打去。
<br/>　　星月奴抬手作法，一串符文飞出，将他击飞，撞在围墙上又坠落下去，刚爬起来，一把剑横在他脖颈上。
<br/>　　“你既然不从，便将掌门大印交出来，由本宫从你们龙虎山中再选一人，担任掌门，以保你万年香火。”
<br/>　　星月奴看也不看他，接着说道：“把大印交给我，我便撤掉搬山咒，否则很快山之精就要耗尽了，你若希望龙虎山在你手中毁掉，便拿着吧，我不强迫。”
<br/>　　曲波的表情因为痛苦而扭曲着，内心陷入了犹豫。
<br/>　　龙虎山千年基业……现在命运居然掌握在自己的这个小人物手中。
<br/>　　“师兄，给她吧，无论如何，不能让龙虎山毁掉啊，不然就算死了，怎么面对师父和师叔祖啊！”
<br/>　　曲波长叹一声，颤抖着将掌门大印从兜里摸出来，一咬牙，朝星月奴抛过去。
<br/>　　星月奴脸上露出满意的微笑。
<br/>　　虽然剧本的走向跟自己设想的不一样，但结果还算是好的，慕寒逃了，那就不要他，也省的真的大开杀戒，只要拿到掌门大印，从龙虎山众多弟子中随便找一个当个傀儡掌教并不是难事。
<br/>　　星月奴伸出丰腴雪白的玉手，去接大印。
<br/>　　然而，就在大印快要落到她掌心的瞬间，突然一道青光飞速射来，击打在大印上，直接砸飞了出去。
<br/>　　星月奴心下一沉，不由自主地顺着大印飞落的方向看去，一道人影从院墙外高高跃起，伸手抓住了大印，然后在众目睽睽之下，落在了道场中间。
