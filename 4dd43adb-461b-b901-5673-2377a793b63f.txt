        有这么一刻，童筱颖真的很想摇醒傅昼景，然后歇斯底里地质问他：
<br/>
<br/>        季思妍不是大哥的女人吗？
<br/>
<br/>        你和季思妍之间到底是怎么回事？
<br/>
<br/>        季思妍的那个孩子，真的是你的吗？
<br/>
<br/>        其实这些问题，她本应该在那天质问季思妍的，但是，她潜意识里不大相信季思妍，所以，才把那些问题全部积压在了心里。
<br/>
<br/>        想到这些，鼻子瞬间酸痛得格外厉害，眼泪就像断了线玉珠，从指缝间滑落。
<br/>
<br/>        童筱颖猝不及防地把掉下来的眼泪一一擦干，最终，她还是忍住了，没有叫醒傅昼景。
<br/>
<br/>        这一晚，她睡得浑浑噩噩。
<br/>
<br/>        翌日清晨，傅昼景醒来后，蹑手蹑脚地下床，生怕把童筱颖给吵醒了，扰了她的好梦。
<br/>
<br/>        童筱颖本想继续装睡，但是莫名其妙地就是睡不下去了，就在傅昼景从床边起身正在换衣服的时候，她也从薄被里爬了起来。
<br/>
<br/>        察觉到身后的动静，傅昼景一边换衣服一边转过身去，温和地问道：“是我吵醒你了吗？”
<br/>
<br/>        “没有，我去给你做早餐，好久没和你一起吃早餐了。”童筱颖强颜欢笑着掀开了被子。
<br/>
<br/>        “我……”傅昼景欲言又止，顿了顿后，转开了话题，“小爱怎么不见呢？”
<br/>
<br/>        小爱，三年前，他送给她的那只博美犬，而且还是一只小公犬，每次他出差不在家的时候，都是小爱陪伴在童筱颖的身边。
<br/>
<br/>        “本来和我大学同学约定好让小爱和她家的小可洞房，到时候她家小可怀了宝宝后就给我送一只。结果小爱提前出轨了，把邻居家的母泰迪狗给上了，还把邻居给咬伤了。邻居来找我麻烦了，我向邻居赔礼道歉，然后带小爱去做了绝育手术。小爱被我放宠物医院里了，医生说，他会找个合适的时机，给小爱做绝育，等小爱手术康复后，就打电话通知我，到时候再去接它回来。”童筱颖慢条斯理地说。
<br/>
<br/>        就因为小爱出轨，所筱颖沫把小爱送去宠物医院给阉了？！
<br/>
<br/>        傅昼景听着童筱颖的叙述，心里莫名其妙地感觉一阵毛骨悚然，总觉得哪里怪怪的，但又说不出是哪里奇怪。
<br/>
<br/>        其实，童筱颖也只是为了小爱好，小爱到了发情期，对人有攻击性而且还有些食欲不振。
<br/>
<br/>        “嗯，那把小爱接回来后，好好地安抚它。”傅昼景一脸尴尬地应了声。
<br/>
<br/>        童筱颖面无表情地点了点头，然后离开了房间，下楼去了厨房。
<br/>
<br/>        滨江城，傅氏集团大厦。
<br/>
<br/>        传闻，傅氏集团现任总裁傅夜沉，要特招一名女秘书，应聘那天竟然引来了全市两千多名醉翁之意不在酒、另存欲念攀上金山银矿的女人。
<br/>
<br/>        大家争奇斗艳，各式各样的衣着令人目不暇接，如同选美大会的竞选会场。
<br/>
<br/>        放眼望去，大体上净是一些长相娇艳无比的千金小姐、干练的女强人、傲慢自恃的富家女，偶尔夹杂着少之又少的平民百姓。
<br/>
<br/>        经过简单严格的初步筛选后，仍有十五名候选人抢夺这个小小的地盘。
<br/>
<br/>        落以柔预估自己在初次筛选当中铁定会被踢下来，却没有料到自己会雀屏中选，当天咧嘴笑了一整天。暗自感谢着，上天怜惜她这个孤苦无依的小孤女，特意赐与她这个好机会。
<br/>
<br/>        第二次面试这天，落以柔正站在电梯前和一群跟她一样来二次面试的女人们一起等电梯。
<br/>
<br/>        她们一个个妆容精致，让她这个素颜出镜的平凡女生显得有些格格不入。
<br/>
<br/>        “当——”的一声，电梯抵达一楼，正要开启的一刹那，后方传来急促奔跑的脚步声，随之而来的身影将她直接挤开撞到了一旁。
<br/>
<br/>        落以柔重心不稳跌扑在地上，左脚踝在重力撞击下呈现不太自然的扭曲状态，隐隐约约传来痛楚，她脸儿全皱成一团，额间沁出细微的汗珠。
<br/>
<br/>        身后陆陆续续有人疾步进入电梯，就在电梯门快要关起来时，落以柔吃力地站起身，刚挤进去，电梯里的警报声就响了起来。
<br/>
<br/>        这台电梯里全是女人，一个个用鄙夷而嫌弃的眼神睨着落以柔。
<br/>
<br/>        落以柔只觉自己后背凉飕飕的，不得不低着头，像只打了霜的茄子，只身走出了电梯。
<br/>
<br/>        电梯门随之无情地关上，那群女人一个个骄傲地上了楼。
<br/>
<br/>        为了赶时间，落以柔不得不去按旁边的那台电梯。
<br/>
<br/>        这台电梯是从地下室上来的，她是第一次来这种高大尚的大厦，不知道隔壁这台电梯是总裁的VIP电梯。
<br/>
<br/>        更加不知道，进这台电梯按楼层是要刷卡。
