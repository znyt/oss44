　　第439章
<br/>　　李行哉连日以来的苦闷心情，今日被冲淡了几分。虽然被当做傀儡看待，但李行哉一直想要培养自己的势力。
<br/>　　但想要做到这一点，真的很难。今日他突然意识到一件事：即便在崔相的羽翼覆盖之下，也并非铁板一块。他们为了共同的利益，要联合在一起。但为了各自利益，却也不在乎分裂。
<br/>　　何况，还有宋游渠这样的年轻人，对老一辈所追求的东西根本不屑一顾。
<br/>　　总之是好事，反正，在已经糟糕透了的情况下，任何变化都是好事。
<br/>　　李行哉在大帐内踱步，心思电转，压抑着内心的兴奋。
<br/>　　“你们有多少人？”李行哉忽然发问。
<br/>　　“我们有一个名字，叫青年营，都是由各家的年轻人组成。”说着宋游渠的声音低下来：“目前，人数还很少，只有二十一个。”
<br/>　　李行哉默然，随之道：“不怕，星星之火，足以燎原，眼前的一颗种子，日后未必不能成为参天大树。”
<br/>　　“下一步，殿下可知该怎么办？”宋游渠。
<br/>　　李行哉坐回到椅子上，手指轻轻敲着桌面。如今，各路诸侯的势力都涌入并州，为了同一个目的，就是消灭起义军，砍掉林少羽的脑袋。
<br/>　　封锁林少羽的包围圈越来越小，大家都明白，最后的结果也就在这一两天。但李行哉对这件事情，并无多少主导权。毕竟，他无兵可派，无将能用，也就是在别人的操纵下，亦步亦趋。
<br/>　　“依在下觉得，谁杀死林少羽，其实并不重要。”
<br/>　　“喔，怎么说？”
<br/>　　“谁能杀死林少羽，无非就是求个名正言顺，但现在的情况，名正言顺其实已不重要。重要的是谁手中握着的兵力更多。”宋游渠缓缓道。
<br/>　　李行哉当真要对宋游渠刮目相看了。林少羽很快就会被杀死，难道真是谁杀了林少羽谁成为九五之尊？怕那些手握重兵的各路诸侯，并不会心服口服。最后的结果，还是要打一场的。最后取决胜负的，不是名义，是谁手中的拳头大。
<br/>　　可眼下的问题是……李行哉手中无兵。
<br/>　　他长叹口气，沉默半晌，缓缓道：“如果可能的话，我还是希望能够亲手斩下林少羽的脑袋，这样，也算是给程大雷一个交代。”
<br/>　　……
<br/>　　夜，并州边境。
<br/>　　有三道人影沿着河流鬼鬼祟祟向前走。
<br/>　　“大当家……”徐神机轻声唤道。
<br/>　　“什么事？”程大雷压低声音：“小声些，别把人招来。”
<br/>　　并州与戎族草原接壤，这里地势平坦，几乎无险可守。但在最近，边关却布上重兵，时时有骑兵来回巡逻，以防范起义军有可能的突围。
<br/>　　现在程大雷三人想要潜入并州，便不能被人发现。好在三个人目标小，想要潜入也并非做不到。
<br/>　　“我累得实在走不动了，咱们能不能歇一歇吃点东西。”徐神机道。
<br/>　　程大雷白了他一眼，道：“再等等，等过了这条河，咱们就找地方歇会。”
<br/>　　程大雷不能不着急，眼下的情况，林少羽能活几天还不好说，必须抓紧时间赶路。草原上，遭遇帝国军的概率还小，可一旦进入并州，就要随时有遭遇帝国军的可能。
<br/>　　眼前这条大河拦住了三人的路，程大雷沿着河走，想要找一处浅滩，或一座桥通过这条大河。
<br/>　　明月在天，河水波光粼粼，三道黑影牵着坐骑往东行。
<br/>　　不怪徐神机叫苦，事实上，程大雷此刻也十分疲惫。他们需要通过河流，找个地方歇脚，打量起义军的去向，然后，吃点东西。走到现在，他们随身携带的干粮已经耗尽。
<br/>　　前方出现一座石桥，徐神机和程大雷眼中都是一亮，当然，银眸是不会有反应的。她就像个石头人似的，不知疲倦，不知痛苦，不知饥饿。
<br/>　　“喂，什么人！”
<br/>　　三人刚走到石桥中央，从对面桥头突然冒出一伙人，大概有十几个。
<br/>　　程大雷一惊，徐神机的手已经不知不觉摸住腰上的短刀。刚才程大雷着急了些，没有注意到桥对面还窝着一伙人。
<br/>　　“哈，我们是放牛的，想进城做点小生意，怎么，这条路不能走么，不能走我们就回去。”程大雷骑着黑牛，手摁在匹夫剑上，嘴里打着哈哈。
<br/>　　小队的头目盯着程大雷，微微皱眉，口中吐出三个字：“程大雷！”
<br/>　　程大雷一激灵，就见在月光下，对方骑着一匹黑马，身披黑甲，面目也十分凝重。
<br/>　　程大雷愣了好大一会儿，终于认出了对方是谁：柳轻名。
<br/>　　当初长安武举，有两个人是武状元的有力竞争人选，一个是将军府的柳轻名，一个是相府的屈九江。
<br/>　　但程大雷当初大闹武科场，将二人的好事都搅黄了，所以提起程大雷，二人都是咬牙切齿，刻骨铭心的难忘。
<br/>　　程大雷之所以可以认出对方，还是借着系统的功能。他尴尬的笑笑：“原来是柳大人呐，长安一别，已经三年有余，柳大人还是风姿潇洒，英俊不减当年。”
<br/>　　柳轻名眼角骤跳，忽然将手一挥：“与我杀！”
<br/>　　迅雷不及掩耳之势，十几位军中好手，向程大雷扑来。
<br/>　　当然，程大雷也并不是没有准备，从开始到现在，他的手就没有离开匹夫剑的剑柄。
<br/>　　“保护好徐军师。”
<br/>　　程大雷喝了一声，骑着牛冲了过去。他一人拦在敌人前，在两军相交的刹那，身子突然腾空跃起，跃过众人头顶，来到柳轻名面前。
<br/>　　柳轻名变招极快，一枪从下向上刺来。
<br/>　　程大雷在空中握住了他的枪，身子向下坠，手中用力一扯。
<br/>　　“给俺下来。”
<br/>　　咚！
<br/>　　沉重的声响，柳轻名连人带枪，直接被拽下马背，等他再回过神来。枪已经到了程大雷手中，枪尖压在柳轻名肩膀上。
<br/>　　这一幕鹊起鹰飞，只在刹那之间。柳轻名的手下根本没有反应过来，在他们想要扑过来时。程大雷忽然回头大喝了一声。
<br/>　　“谁敢动，不想要命了么？”
<br/>　　人群被喝住，程大雷目光落在柳轻名身上。
<br/>　　“都三年了，你还是一点长进都没有啊。”
<br/>(本章完)
