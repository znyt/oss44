　　“够了，够了。”
<br/>　　一尊武者看到这黑玄金，急忙的接了过来。
<br/>　　这黑玄金的价值，足足抵得上十个人的费用了，这一次他们二人克扣一下，也可以赚取不少。
<br/>　　其中一尊武者，犹豫了一下，突然从空戒中，拿出了一块精英的石头，递给了楚风眠。
<br/>　　这一块石头，四四方方的，大约一半拳头大小，极为工整，其中蕴含极为精粹的灵力。
<br/>　　“这是一块中品灵石，就算是你多余的钱了。”
<br/>　　那尊武者说着，便是将这一块中品灵石，递了过来。
<br/>　　“接着吧。”
<br/>　　白萱在一旁，悄然开口道。
<br/>　　听到白萱的话，楚风眠这才点了点头，接过了这一块中品灵石来，进入到了这星天峰之中。

<br/>　　对这一点，楚风眠倒是心中清楚。
<br/>　　虚空海毕竟是危险无比，一个人探索，陨落的几率极大，有着几个同伴，一起探索，遇到危险也好应对。
<br/>　　不过这种同伴，也是有利有弊，虽然可能会化解危险，但是在更多的时候，这些同伴却是危险的根源。
<br/>　　一旦是发现了极大的财富，互相厮杀，都有可能，想要探索，也最好找信得过的人才行。
<br/>　　“没想到你居然有着如此财富，拿出黑玄金作为费用，就连我都不会如此奢侈。”
<br/>　　白萱看着楚风眠，笑了说道。
<br/>　　“那两人倒是识时务，找回了你一块中品灵石，不过他们还是可以克扣一部分的财富的。”
<br/>　　“在加入到学院之前，略有奇遇罢了。”
<br/>　　楚风眠随意开口道，也不想解释。
<br/>　　楚风眠的财富，就算是除去那些九鼎丹，也要远远比起寻常巅峰圣者，甚至是生死境强者要多的多。
<br/>　　楚风眠毕竟可是掌控了秦皇国度大半的财富，更是灭了虚家，得到了虚家的无数财富，还有着之前夺花尊者，暗古魔尊的财富。
<br/>　　他的财富，简直是富得流油，无数奇物，这一块黑玄金对楚风眠来说，简直是九牛一毛罢了，他自然不会在乎。
<br/>　　令楚风眠有些在意的，只是这一块中品灵石，这块中品灵石，跟楚风眠从圣山中夺取灵力，凝聚的灵石极为相似。
<br/>　　不过论起其中的灵力品级，倒是要差得多，楚风眠凝聚出的灵石，比起这中品灵石，要精粹无数个等级。
<br/>　　“这中品灵石是？”
<br/>　　楚风眠看着旁边的白萱，不由的出声问道。
<br/>　　拓跋鼎都没有探索过虚空海，所以对着虚空海中的种种，楚风眠也只能是向着白萱询问。
<br/>　　“这灵石，其实是虚空海中的货币。”
<br/>　　白萱看着楚风眠，倒是没不耐烦，慢慢解释道，毕竟每一个刚来虚空海的新人，都会像是楚风眠一样，几乎什么都不懂。
<br/>　　“对我们这些实力达到巅峰圣者之上的武者来说，寻常丹药已经没有作用了，只有这种灵石，才对我们修炼有用。”
<br/>　　“你手中的这块灵石，只是中品灵石，价值不高，以你我的实力，耗费些时间，也可以凝聚出来，在往上还有着上品灵石，极品灵石。”
<br/>　　“这些灵石，按照之前北域中强者定下的规矩，便是都切成半个拳头大小的方块，作为货币，在极品灵石之上的，还有着王品，皇品，帝品等等。”
<br/>　　“这种高级灵石，就不是我们武者可以凝聚的出的，只有虚空海中的无数宝藏中，才会得到，每一块都是价值连城。”
<br/>　　白萱一边说着，一边手心一动，一块晶莹灵石，便是出现在她的手中。
<br/>　　这一块晶莹灵石的灵力，显然是比起那中品灵石，要精粹无数倍。
<br/>　　“这就是我曾经在一处远古洞府中，偶然得到的一块王品灵石，当时那一块灵石被切割开来，我分到的只有这一块，算是不错的宝物了，我准备留在度过生死大劫之前，冲击神力境极限用。”
<br/>　　白萱说着，倒是没有隐瞒，仔细的讲解这。
<br/>　　“这一块灵石，便是王品灵石吗？”
<br/>　　楚风眠看着那一块王品灵石，心中暗自想到，突然他心神一动，一块足有拳头大小的灵石，拿了出来。
<br/>　　“那这一块灵石，是什么品级？”
<br/>　　这一块拳头大小的灵石，正是楚风眠从圣泉之中利用剑意化为吞天神蟒凝聚出的无数灵石之一。
<br/>　　单单是论起灵力精粹程度，这一块灵石的灵力精粹，丝毫不比那王品灵石要差。
<br/>　　“这是？”
<br/>　　白萱看到楚风眠手中拿出来的灵石，眼睛都直了，一脸震惊。
<br/>　　“王品灵石！在王品灵石中都是品级极高的，差一步就达到了皇品灵石的地步，你你你哪里得到的，还是这么大一块，足以抵得上标准两块了。”
<br/>　　“这样的灵石，你是从哪里得到的，这种王品灵石，只有一些远古洞府之中，才可以寻找的到。”

<br/>　　“一块王品灵石，已经足以换取诸多财富了，你这一块，比起两块王品灵石还要大，足以换取到不错的奇物了。”
<br/>　　白萱开口之际，也是无比羡慕道。
<br/>　　“本来以为你刚刚加入到北邙学院，应该没有多少财富，没想到你身上的财富，已经足以让我羡慕了。”
