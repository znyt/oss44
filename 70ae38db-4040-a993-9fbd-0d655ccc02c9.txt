　　混乱虚空一处空间之中，杨柳青青，随风而动。
<br/>　　绿柳之下，两个道人正在论道，一人为扬眉老祖，另一人为道印天尊元始。
<br/>　　“成于混沌，先有规则，后有天地。寂兮寥兮，独立而不改，周行而不殆，可以为天地母……”
<br/>　　“夫道者，其成，先天地生，悟道乃天地相参，亿万合一，然后得以窥其成功。太初有云，其道与神同在，神即是道。夫道，天下一致而百虑……”
<br/>　　“万物以生，万物以成，命之曰道。乃曰：人法地，地法天，天法道，道法自然……”
<br/>　　“混沌无极，无极为道，道生一，一为元，一生二，二为阴阳，二生三，三为天地人，三生万物，故万物为道。万物负阴抱阳……”
<br/>　　“仙道元气者，元气之精者也，气道乃生。仙道者，盈而不溢，盛而不娇，劳而不矜其功……”
<br/>　　……
<br/>　　这一论道，就论了三千年。三千年之后，两道人才停了下来。
<br/>　　“道友，此次论道，贫道收获颇丰，在此多加感谢，贫道这便告辞，道友若有时间，可到东海永恒岛一游。”
<br/>　　讲道过后，元始便不再停留，想要告辞离开。
<br/>　　“彼此，贫道此次也收获巨大，需要闭关，在此就不便多留道友了。”
<br/>　　显然，不只有元始一人收获巨大，扬眉亦从元始那里学到很多东西，需要闭关参悟。
<br/>　　“无妨，那就在此别过。”
<br/>　　元始微微行了一礼，便告辞离开了，出门之后，直奔东海而去。
<br/>　　“这位元始道友果然不简单，即便吾有之前混沌魔神的记忆，他的境界亦毫不逊色于吾，甚至更胜一筹，而且，他的元始之道，确实不同凡响。”
<br/>　　杨眉老祖看着远去的元始感叹道，神色之间，闪过些许忌惮之色，显然，他感受到元始的强大之处，要他对付元始，不是那么容易的。
<br/>　　不再多想，杨眉老祖一个转身，身影便淡淡虚化于虚空之中，杨柳树神威大涨，立于虚空，之后，淡淡隐没虚空。
<br/>　　“这样扬眉老祖，果然不简单，不愧是空间魔神转劫而生，对于空间法则的领悟已经达到了一个不可思议的高度，就快要突破混元，踏入第三步了。”
<br/>　　出了混乱虚空，元始回头看了看，默默想到。
<br/>　　显然，此次论道，不仅扬眉老祖感受到元始的强大，元始同样感受到扬眉老祖的可怕，这扬眉老祖已经一只脚踏入混元之境了，显然，要不了多久，他就会突破，而且还要在元始之前突破。
<br/>　　“吾之元始一道还需要更深的积累，现如今吾虽然已不惧任何敌人，但想要突破混元，积累还是不够，看来此次回去，要尽快参悟更多元始印之中的法则了。”元始如是想到。
<br/>　　“不过，此次与扬眉道友论道，吾对于空间一道领悟却是更深了，已经脱离空间法则初期，快要进入中期空间法则之境了，此次回去闭关一番，定能突破。扬眉，不愧是空心绿柳所化，天生自带空间本源。”元始心道。
<br/>　　“不过，吾之分身混沌道君也有传承的混沌本源，他不需要参悟其他法则，只需要悟透那混沌本源，定能突破很多，实力提升倒是不用担心。况且，他也快要突破混元了。”
<br/>　　“混沌灵根，资质就是不凡。”
<br/>　　说到此处，元始都有些羡慕了，虽说自己是盘古元神清气所化，资质亦是不凡，但对于像混沌树这样的顶级混沌灵根来说，还是不够。
<br/>　　但羡慕归羡慕，元始并没有太多想法，他也并不会交换，混沌树是好，但也受于混沌本源限制，参悟混沌法则容易，但要是参悟其余法则，却是要比一般修道者要困难千百倍。这也算是有得有失了。
<br/>　　不过，混沌法则乃三大本源大道之一的物质本源大道，十分不凡，尽管只能修炼混沌大道，但对于大多数人来说，这是一辈子不敢想的事儿。他已经领先其余之人太多了。
<br/>　　回到永恒岛中，发现自己的四个记名弟子的修为都已经不弱了，四人都已经金仙修为了。尽管他们四人资质一般，化形之时只是真仙修为，但仅仅用了如此短时间就修炼到金仙，可见他们还是非常刻苦的。
<br/>　　夸赞四人几句，赏赐每人一件灵宝，吩咐几人刻苦修炼，不可生事后，元始就宣布再次闭关。
<br/>　　而就在元始闭关之际，洪荒却是风雨欲来。
<br/>　　龙凤麒麟三族自从上次大战之后，就深刻认识到自己的不足，都各自回去发奋刻苦修炼，为下一个大战做准备。
<br/>　　洪荒那么大，即便没有三族，洪荒中的争斗一样没有停止过。
<br/>　　洪荒之中虽然没有大战，但是小战却是不止。
<br/>　　特别是近来，洪荒之中的战争更是愈演愈烈，龙凤麒麟三族再次被卷如旋涡之中。
<br/>　　当然，这其中也有魔祖罗睺的推动。
<br/>　　魔祖罗睺需要用大劫之中的劫煞凶气来修炼，煞气越重，他的修炼就越快，如今洪荒安静已久，罗睺等不及了。于是，他便设计杀害龙凤麒麟三族子孙，并嫁祸到三族头上。
<br/>　　就这样，三族之间的战争变得越来越频繁，三族谁也不服谁，各种摩擦，利益之争，从不间断。
<br/>　　而天地之间的劫煞之气随着大战的爆发也越来越重，一些修为弱小的修士，早已经被这劫煞之气所影响，迷红了双眼，只知道杀戮。
<br/>　　天地间，每日不知道要死多少修士，这就是无量劫的厉害之处，修士减少了，天地负担也就少了，天道也就轻松了。
<br/>　　距离最终大战越来越近，天地间的天机再次被遮掩，天机混乱，龙蛇起陆，大劫将至。
<br/>　　同时，在不周山的一个隐蔽山洞之中，天地间无数煞气被吸引过来，形成一个巨大的煞气龙卷风暴，注入到山洞之中。
<br/>　　“砰！砰！砰！……”
<br/>　　山洞中，随着十二声砰咚的巨大响声响起，掀起一股莽荒苍茫之感。随着时间流逝，山洞之中的响动也越来越大。
<br/>　　“吼！吼！吼！……”
<br/>　　山洞中，有十二个巨大血茧，从里面不断响起巨吼声，仿佛有十二个巨大的蛮兽在对月长啸。
<br/>　　“砰…砰…砰……！”
<br/>　　十二个血茧就仿佛一个个巨大的心脏在不停地跳动，一会儿大，一会儿小。
<br/>　　“咔擦……咔擦……咔擦……！”
<br/>　　巨大的血茧出现一道道裂缝，而且裂缝越来越大，终于，
<br/>　　“嘭！！！”
<br/>　　血茧彻底崩溃，残余的波动卷起一阵阵灰尘，渐渐平息，十二个巨大的身影出现在山洞之中。
<br/>　　十二祖巫终于化形而出！
