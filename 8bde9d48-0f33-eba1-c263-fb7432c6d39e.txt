　　与此同时，华山之巅，
<br/>　　“神王大人，您为了区区一个世俗小子，如此大费周章，未免有些小题大做了吧？”
<br/>　　说话的，是一名身穿西欧中世纪风格服装的白人中年男子。
<br/>　　而他口中的神王大人，赫然正是那位神界之主宰。
<br/>　　那中年男子一开口，其余神界众人也纷纷道：
<br/>　　“是呀！我们既然已经和其他三界打成共识，何不集合四界高手，直接杀将过去，纵然那小子有化神境以上的实力，也不可能是我们四界之敌！”
<br/>　　“我们神界数十名强者，耗时整整七日，方才布置下这弑神大阵，用来对付区区一个世俗少年，未免有些小题大做了！”
<br/>　　面对众人的异议，神王却淡淡一笑，道：
<br/>　　“你们懂什么？集我们四界之力，对付区区一个世俗少年，自然是不再话下，可是如此一来，不就凸显不出我们神界的厉害了？”
<br/>　　“更何况，这段时间接触下来，你们也都看到了，四界之间一向互相猜忌，根本不可能彼此信任，想指望和他们合作，简直难比登天！”
<br/>　　神王说到这里，微微一顿，接着深吸了一口气，方才又道：
<br/>　　“那小子先后在妖、魔、仙三界肆意妄为，让三界丢尽了颜面，我们若是能一举将此子灭杀，那从今往后，我们神界便是名副其实的四界之首！其他三界还敢和我们叫板吗？桀桀桀！”
<br/>　　说到最后，神王忍不住哈哈大笑起来。
<br/>　　其余众人也纷纷面露喜色。
<br/>　　在他们看来，只要将叶尘引入弑神大阵之中，肯定必死无疑！
<br/>　　众人正在议论间，一名金发碧眼的美貌女子，摇晃着妩媚的腰肢，款款来到众人跟前，
<br/>　　“启禀神王大人，事情已经办妥，不出意外的话，那姓叶的小子，明日就会前来！”
<br/>　　神王顿时面露喜色，
<br/>　　“很好！黛丽丝，你果然没有让本神王失望！大家都好好准备，明天定要将此子一举拿下！”
<br/>　　……
<br/>　　叶尘虽然猜到有人要算计他，但是并没有因此退缩。
<br/>　　因为他清楚，自己先后得罪了神、仙、妖、魔四界，和他们对上也是早晚的事情。
<br/>　　既然对方主动送上门来，反倒省的他孤军深入了。
<br/>　　毕竟，华夏国可算得上是他的地盘。
<br/>　　当晚，
<br/>　　华夏国七大军区，足足数十万军队，分别从四面八方，悄悄赶往华山附近，形成包围之势。
<br/>　　要知道，叶尘如今可是华夏国的唯一一位元帅，调动军队自然不在话下。
<br/>　　甚至，华夏高层得知消息之后，连夜召开会议，最终一致决定，真要是到了危急关头，不排除动用大型核武！
<br/>　　……
<br/>　　第二日，天海大学生物学院的上百名学生，浩浩荡荡的从学校出发。
<br/>　　众人乘坐大巴，直奔华山方向而去，浑然不知道，自己已经卷入了一场前所有未的浩劫之中。
<br/>　　此处正值隆冬季节，而且又早已经进入北方地界，温度自然更加寒冷，天空隐隐飘起了飞雪，当真是风景如画。
<br/>　　“早就听闻，华山雪景，艳绝天下，没想到今日竟然有缘看到！”
<br/>　　穿上一身白色羽绒服的萧若曦，从车子上跳之后，仰头望着上方山上的雪景，俏脸上满是欣喜之色。
<br/>　　一旁的李虎，则愁眉苦脸道：
<br/>　　“这里风景确实不错，可下这么大的雪，还要爬山！哎呦，我的腰！”
<br/>　　陈东也跟着道：
<br/>　　“是呀，也不知道咱们那位陶班长是怎么想的，大冬天跑到这华山来旅游，华山可一向以险峻著称，真要是出点什么意外可如何是好？”
<br/>　　唐威却哈哈一笑，
<br/>　　“我倒是觉得挺好的，你们别这么娇气！再说有尘哥在呢，还怕护不住我们的安全吗？”
<br/>　　几人齐声称是，纷纷朝叶尘望来，却见叶尘一脸凝重之色。
<br/>　　萧若曦察觉到了异样，不由得拉了拉叶尘胳膊，
<br/>　　“怎么了？有什么不对劲吗？”
<br/>　　叶尘抬头望了一眼华山之巅的方向，方才露出一抹笑容，摇了摇头，道：
<br/>　　“没什么，待会你们记得，一定跟紧我，不要随便乱跑！”
<br/>　　众人见叶尘不像是在说笑，当即纷纷点头称是。
<br/>　　或许是因为下雪的缘故，游华山的人并不多，所以众人很快检完了票，正式踏足景区之中。
<br/>　　待众人真正来到华山脚下，一路上都一言不发的陶婉清忽然开口了，
<br/>　　“华山险峻，如今又下了雪，爬山风险太大，自认为能力不足的，可以选择做索道上山！”
<br/>　　说完这话，陶婉清有意无意的朝叶尘这边望了一眼。
<br/>　　李虎听到这话，忍不住笑道：
<br/>　　“尘哥，我看坐索道也挺好的，要不……”
<br/>　　还没等他说完，一旁的唐威直接朝他头上抽了一巴掌，没好气道：
<br/>　　“老四，你要是不嫌丢人，你自己去，可别拉着我们！”
<br/>　　唐威说完，直接纵身一跃，跨越数道阶梯，人已经到了十多米开外。
<br/>　　李虎叹了一口气，只好一脸幽怨的跟了上去。
<br/>　　“我们也上去吧！”
<br/>　　叶尘拉着萧若曦的皓腕，也缓步朝山顶的方向走去。
<br/>　　而另外一边，
<br/>　　陶婉清看到叶尘走了山道，嘴角露出一抹冷笑，转而走进了一旁的索道入口。
<br/>　　……
<br/>　　不一会，叶尘带着众人来到半山腰的位置，
<br/>　　“我们走这边！”
<br/>　　叶尘忽然指着一条羊肠小道，忽然开口，然后不由分说，带着萧若曦，便走了过去。
<br/>　　李虎、唐威和陈东三人，也没有多想，当即也跟了上去。
<br/>　　而其他众人，见那道路崎岖，则并没有跟随。
<br/>　　不一会，叶尘带着四人，越走越偏，越走越奇，彻底偏离了主道。
<br/>　　而此时，华山之巅上，神界众人也终于察觉到了异样，
<br/>　　“不对！这小子似乎发现了什么，竟然刻意避开了我们的弑神大阵！神王大人，这该如何是好？”
<br/>　　神王两眼微微一眯，望着远处半山腰上，那五道渺小的身影，眼眸中闪过一抹寒意，
<br/>　　“看来我们小瞧此人了！查尔斯，你去会会这小子，务必想办法将他引入弑神大阵之中！”
<br/>　　“是！”
<br/>　　神王的声音落下，顿时一道黑影在空中闪过，直奔叶尘等人的方向冲去。
