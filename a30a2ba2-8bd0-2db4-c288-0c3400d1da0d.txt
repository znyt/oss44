　　想不明白，他也就不再想了，距离离开还有接近一个月的时间，趁着这段时间，他想要先去做一些事情。
<br/>　　义庄那边是回不去了，所有人都知道他已经死了，如果回去的话，也根本无法解释。
<br/>　　虽然杀了鬼修老头，但任家的连环任务还没有完成，接下来的任务内容，是寻找鬼修老头的居住地。
<br/>　　对于后续任务，刘哔压根没有半点头绪，他想去任家看看，能不能找到什么线索。
<br/>　　之前的那栋房屋，距离任府并不是很远，稍微花费了一点时间，便到了任府门前。
<br/>　　看着朱红色的大门，刘哔眉头一皱，鼻子闻到了一股淡淡的血腥味道。
<br/>　　他心中猜想，或许是任家出事了也说不定。
<br/>　　对于任家人的死活，他根本就不关心，但如果任发死了，他也就断了一条线索。
<br/>　　稍稍犹豫了一下，刘哔确认四下无人，从墙头翻了上去。
<br/>　　和上次来时不同，落地之后，他根本没有多加掩饰，直接就上了任府的鹅卵石道路。
<br/>　　他没必要掩饰什么，他只是想要找任发问几个问题，当然，问完之后他也不会放过任发就是了。
<br/>　　他还活着的这个消息，最好还是不要让别人知道，否则流传出去的话，也是一个不小的麻烦。
<br/>　　任府的道路没有上次来那样整洁，地面上很多已经干涸掉了的血液，花圃中的花，也差不多都要凋落了。
<br/>　　和他猜想的一样，任府果然是出事了，但具体发生了什么，他也不太清楚。
<br/>　　来过一次，对于府上的布局构造刘哔心中也了解一些，在附近几个房屋内搜索了一番，却没找到一个活人。
<br/>　　甚至连尸体也没有，唯一能证明这家人存在过的痕迹，就是那一滩滩随处可见的鲜血。
<br/>　　紧皱着眉头，刘哔加快了脚步，向主厅走去，任府的气氛，让他心中有些不安，他怕停留时间长了，会发生什么危险。
<br/>　　来到主厅前，他看到了敞开的大门，几只原本挂在门口的灯笼，被风吹落在地上，显的有些颓败残破。
<br/>　　到了这里，血腥味更加浓郁了，被人血的味道刺激，刘哔的双眼覆盖上一层红色。
<br/>　　没过多犹豫，他直接从敞开的屋门中走了进去，绕过屏风，看到了后面如同炼狱一般的景色。
<br/>　　主厅的家具装饰，并没有多大变化，和上次不同的是，大厅中央，多出了很多尸体！
<br/>　　这些尸体大概有二三十个，服饰都差不许多，应该是任府的下人，还有一些穿着睡衣的，他也分不清哪个是哪个。
<br/>　　之所以认不清楚身份，是因为他们的头颅，都被人砍下，不知道放置到了哪里。
<br/>　　已经干枯了的血液，将四周地板浸成了暗红色，这些尸体以各种奇异的姿势，在大厅中央摆成了一个图腾一般的圆柱。
<br/>　　看着这幅景象，刘哔头皮有些发麻，他不知道凶手是否已经离开，也不知道继续搜索下去，会不会有什么危险。
<br/>　　但是如果找不到线索的话，下一轮的连环任务就很难完成了，一个鬼修的藏身地，岂是那么容易就能寻到的。
<br/>　　暗自咬了咬牙，最终他还是来到了密室的机关旁边，开启了暗道。
<br/>　　从血液的情况来判断，这些人死去的时间已经不短了，凶手有很大的几率已经离开，他不想放弃这次机会。
<br/>　　现在还没人发现任府的惨案，如果被人发觉，会对现场造成很大的破坏。
<br/>　　到那个时候，他再回来搜索，也很难找到什么有价值的线索了。
<br/>　　考虑清楚之后，刘哔便也不再犹豫，径直来到了大厅底下的密室中，但刚打开大门，就看到了一个熟悉的身影。
<br/>　　是任婷婷，她居然活了下来，但是此刻她却不知道被谁给绑了起来，全身赤裸的躺在密室床上。
<br/>　　粗糙的麻绳，将白暂滑腻的肌肤勒出一道道红印，却也将其完美的身材给勾勒了出来。
<br/>　　房屋内淡淡的胭脂味，让刘哔打了个喷嚏，其中还夹杂着一股腥臭的味道，有些像是杏仁，让他感觉有些恶心。
<br/>　　听到有人进来，任婷婷睫毛微动，睁开了眼睛，看到刚刚走进来的刘哔，眼神猛然变得惊喜，挣扎了几下，‘呜呜’央求着刘哔。
<br/>　　上前走了几步，拿下了任婷婷口中的袜子，刘哔一言不发，平静的看着她。
<br/>　　他想知道，任府到底发生了什么，这个女人又是怎么活下来的。
<br/>　　口中的臭袜子被扯掉，任婷婷微微喘息了几声，一根由口水组成的晶莹丝线，顺着嘴角滴落在她白暂的肌肤上。
<br/>　　此时她却再也没有了一丝千金小姐的优雅与端庄。
<br/>　　“父亲死了，叔叔也死了，整个任家二十多口人，都被杀了。”平复了一下心情，任婷婷语气有些悲切。
<br/>　　“谁干的？”抱着胳膊，刘哔语气平静，并没有多大波澜。
<br/>　　之前在大厅看到那些尸体，他就知道任家可能是完了，能在这里找到任婷婷，也是意外之喜。
<br/>　　“是一个老头，可能不是人，身子后面飘着几个死人头。”强忍住眼中的泪水，任婷婷开口回答。
<br/>　　听到这里，刘哔眼中的色彩才浮现出了一点变化，任婷婷说的，应该就是他之前杀掉的那个鬼修。
<br/>　　但他并没有表露出来，也没将鬼修已经死了的消息个告诉任婷婷。
<br/>　　“你怎么活下来的？”刘哔语气清冷，道。
<br/>　　似乎是勾起了不好的记忆，任婷婷眼中闪过一抹痛苦，回答：“是任山把我绑在了这里，藏了起来。”
<br/>　　顿了一下，任婷婷解释道：“任山是爷爷捡来收养在任家的，府上的大小事情，平时也都是他去负责的。”
<br/>　　微微点了点头，刘哔对这个名字没有印象，但是他也没追问的想法。
<br/>　　“任家为什么被屠，的罪过什么人？”看着任婷婷，刘哔继续问道：“那个凶手，你又了解多少？”
<br/>　　他觉得这件事情并不简单，既然是那鬼修动的手，其中自然会有什么原因，借着这个线索，他没准也能追查下去。
