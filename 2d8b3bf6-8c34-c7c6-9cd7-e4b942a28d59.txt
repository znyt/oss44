
大兴拳场见面后的三天，范二哥果然给常文书去了个电话，聊了几句后就直奔主题，对方一听就打了个马虎说王惊蛰出京了，要过几天才能回来，范二哥就告诉他讲人回来就跟他说一声。<br/>　　<br/>　　“你算的挺准啊，二哥还真就约你了”王惊蛰从东四环的那块地皮回来后，晚上常文书就把他约在了撸串摊子消遣。<br/>　　<br/>　　王惊蛰灌了一大口啤酒，抹了抹嘴巴子，淡淡的说道：“事关他升迁大事，这范二哥敢马虎么？我一句话就把他醍醐灌顶了，这几天估计他都未必能睡得好了，像他们这种人啊，有的时候站在十字路口不知道是往左还是往右，可能有时一步迈出去走错了，那以后也许就是错一辈子都挽回不了了，我那天说的那句话就相当于是在路口给他点了一盏灯，告诉他应该往哪走，他啊这是信了才给你打的电话，如果要是没信的话……”<br/>　　<br/>　　王惊蛰后半段就没在往下说了，有些事他自己憋在肚子里不说叫秘密，说出来那就是泄露天机了。<br/>　　<br/>　　常文书一听就明了了，他感慨的说道：“人啊真是命，我认识了你然后带着你去见了二哥，一句话把人给点醒了，这要是让永强早先认识你的话，可能醒的就是他了，二哥和永强都是后备梯队里的一员，到最后只有一个人有登顶的机会，现在看来是二哥领先了他一步啊，永强错过了”<br/>　　<br/>　　王惊蛰手拿着呲呲冒油的肉串，撸了一嘴后，淡淡的说道：“那个永强，他没有这个命的，如果放在古代的话，他最多就是王侯将相，想当一朝天子就难了，没那个命就别干这个事”<br/>　　<br/>　　常文书顿时一愣，搓了搓手后挺有兴趣的问道：“这你也能看的出来？从哪体现的啊……”<br/>　　<br/>　　“这种话我没办法和你解释，因为还涉及到一个变数的问题，只能说这范二哥是有大概率罢了”王惊蛰顿了顿是，忽然抬手指着晴朗的夜空说道：“紫微星号称斗数之主，古时人都把紫微星当成帝星，所以命宫主星是紫微的人就是帝王之相，但如果哪天出了变数，这人命宫紫微变淡那就是没有帝王之相了，姜子牙辅佐周武王是瞧出他有这个概率了，陈抟相面看出了赵匡胤和赵光义有皇帝面这也是概率的问题，只能说是他们看准了，但也有不少人失眼的，历史上就太多了，俗称选错人站错队了，所以啊现在说什么都是妄下定论，以后再看吧”<br/>　　<br/>　　王惊蛰说的话让常文书一阵云里雾里的根本就听不明白，等他说完了，就摊着手问道：“说白了，就是我押宝，还得押在范二哥身上呗？”<br/>　　<br/>　　“和永强来比的话，确实他的可能性要大很多……”<br/>　　<br/>　　午夜时分，两人撸完串之后就回去了，临走时王惊蛰和常文书说道：“能不能把东四环那块地的规划图，弄出来一份给我？”<br/>　　<br/>　　常文书点头说道：“这个可以，他们要开那块地的话，图纸得要报到规划局去审批，我托人可以要出来一份”<br/>　　<br/>　　王惊蛰点头说道：“注意点，别让人知道了”<br/>　　<br/>　　第二天下午，东四环那块地的规划图就到了王惊蛰的手里，这份图纸除了别墅内部的构造以外，整块地皮中所有的规划都在上面了，包括小区里的人行道行车道，地下车库甚至还有那些栽种的树木和绿化等等，全都一应俱全的展现在了上面。<br/>　　<br/>　　这些东西报上去了以后，基本就很难在更改了，更何况这块地里整个格局还暗合风水之道，一旦被玄门定下来，就更不会再变了，动一下就全都得要重新改动，这根本就不可能的。<br/>　　<br/>　　除非是他们知道，有人在暗中盯着这里呢。<br/>　　<br/>　　“这个世上最悲惨的事莫过于竹篮打水一场空了，但更悲惨的是发现水没打上来篮子还破了，可如果惨到家了呢……”王惊蛰拿起一支笔，在图纸上缓缓的勾画起来：“那就是，水都被别人取走，然后自己却没的喝了”<br/>　　<br/>　　两天后，东长安街上一栋挂着xx俱乐部牌子的大厦里，王惊蛰和常文书赴约，范二哥和书画今日一身正装，站在门口。<br/>　　<br/>　　从他俩的这身装扮上，就能品出来一个词了，态度。<br/>　　<br/>　　明显是，几天过后的范二哥已经把王惊蛰在他心里的角度，提到了一个偶遇江湖高手的层面上，态度无比的端正。<br/>　　<br/>　　“小文说你离京了，我还真怕你一离就不回来了，上次见面匆匆一别有点没聊到位，我可就等着你再回来的这天呢”范二哥一见王惊蛰就熟络的握着他的手，笑道：“今晚把酒言欢，行不？”<br/>　　<br/>　　王惊蛰是挺服这位老范的，对方这姿态拿捏的简直是恰到好处，既点出了我今晚有事和你讨教，又没显得自己饥渴难耐的程度，分寸上一点毛病都没有。<br/>　　<br/>　　更让王惊蛰无语的是，常文书在后面跟上来的一句话，把他的后路又给绝了。<br/>　　<br/>　　常文书说道：“二哥，那你可得小心了，惊蛰的酒量那都是酒经沙场的，我跟他喝过两次都不分伯仲，你还真未必能降服得住他呢”<br/>　　<br/>　　范二哥顿时哈哈一笑，点头说道：“那是得较量较量了，看谁先认怂哈”<br/>　　<br/>　　王惊蛰一阵无语的看着他俩说道：“不是，你俩这一唱一和的是不是都商量好了啊，直接把我给架起来了，我这要是不喝，就显得我不懂事了吧？”<br/>　　<br/>　　范二哥拍了下他的肩膀说道：“酒逢知己千杯少么……”<br/>　　<br/>　　其实吧，别管对方说话如何上纲上线，以王惊蛰的性格就是，我愿意交你那一切都好说，我要是不愿意的话，你说破天了也没用。<br/>　　<br/>　　总得来说就是，范二哥这个善缘，他并不排斥！<br/>　　<br/>　　(=)
