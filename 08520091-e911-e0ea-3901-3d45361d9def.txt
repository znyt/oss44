　　在虚空海中，这些虚空兽才是真正的霸主。
<br/>　　无数在虚空海中探寻的武者，都是死在虚空兽的手中。
<br/>　　这些虚空兽，都是以灵力为食的，圣者精血，正是最为精粹的灵力，所以许多虚空兽都会主动袭击武者的。
<br/>　　乱葬海中，拥有这整个虚空海中最为密集的洞府所在，许许多多的洞府，都是从无尽虚空之中，被卷入到了虚空海中。
<br/>　　最终全部在乱葬海中聚集。
<br/>　　许多的虚空兽，都是从中得到了大量的好处。
<br/>　　所以在乱葬海中的虚空兽，都是强大非凡，其中最为恐怖的几尊，甚至是有着生死境高阶的实力，是虚空霸主，无法抗衡。
<br/>　　遇上这样的虚空霸主，楚风眠只能是掉头就逃，根本没有抗衡的能力。
<br/>　　楚风眠敞开灵识，观察这周围的种种，虚空海中，危险重重，楚风眠都不敢丝毫大意。
<br/>　　楚风眠的灵识，就算是远超常人，但是在这庞大无比的虚空海中，所能够笼罩的，也只是很小的一番区域。
<br/>　　“这是？”
<br/>　　楚风眠的灵识，突然察觉到，就在前方不远处，有着一群生物，突然飞了过来。
<br/>　　这群生物，看起来应该是一种鸟类，他们在虚空海中，可以直接穿行。
<br/>　　说是生物，但是从他们的身上，却只能感觉到一种精粹到极点的灵力。
<br/>　　“前方出现了大批虚空兽！”
<br/>　　司空轩突然大喝一声，观察了一下，他的脸上露出了几分凝重。
<br/>　　“是虚空雷鸟！”
<br/>　　“虚空雷鸟，有多少？”
<br/>　　旁边的麟木，脸色也是凝重许多开口道。
<br/>　　在虚空海中，虚空兽也是极大的麻烦之一，每年都不知道有多少的天才，前来虚空海，都是死在了这虚空兽的手中。
<br/>　　许多虚空兽的实力，其实并不可怕，只有这神力境四重，五重左右的实力。
<br/>　　但是虚空兽的数量，向来都是无穷无尽，数量极多，就算是一尊生死境强者。
<br/>　　在数千数万的虚空兽围攻之下，也要陨落。
<br/>　　这虚空雷鸟，正是一种群居的虚空兽。
<br/>　　“整整一个族群，三千虚空雷鸟，其中最为强大的一尊，为生死境二重。”
<br/>　　司空轩打量了一眼，脸上露出几分凝重。
<br/>　　“我们要小心，这里已经接近虚空海边缘，找不到休息的地方，千万不要受伤，一旦是受伤，血迹便是会引来更多的虚空兽，倒时候可就真的危险了。”
<br/>　　虚空兽会察觉到灵力，直接从虚空海中急速敢来。
<br/>　　就像是在海中，血腥味会引来鲨鱼一样，在虚空海中，这些虚空兽，就是闻血而来的鲨鱼。
<br/>　　一旦受伤，后果不堪设想。
<br/>　　“杀！全力迎战，相互为盟，一起抵挡！”
<br/>　　司空轩大声喝道，顿时拔剑，直接冲了上去。
<br/>　　剩下的人，也是一起紧跟其后，冲了上去。
<br/>　　“嘶嘶！”
<br/>　　这些虚空雷鸟，似乎也是察觉到了楚风眠一群人的存在一样，各个都是兴高采烈起来。
<br/>　　对他们这些虚空兽来说，武者便是他们的补品，吞噬了武者的血肉，他们也会变得更加强大。
<br/>　　三千虚空雷鸟，同时一声咆哮，猛然冲了上去。
<br/>　　这每一尊虚空雷鸟的实力，只有神力境八重左右，对在场的人来说，随意一剑，都可以斩杀一只。
<br/>　　但是三千虚空雷鸟，可是截然不同，一起冲击过来，只怕是谁单独上，都会陨落。
<br/>　　“杀！杀了这些虚空兽，我们也可以得到不少的好处！”
<br/>　　白萱凝聚灵力，顿时冲了上去，她一掌之下，猛然打死了一只虚空雷鸟。
<br/>　　顿时这虚空雷鸟，直接破碎开来。
<br/>　　楚风眠察觉到，这死去的虚空雷鸟中，仿佛是有着一股灵力，融入到了白萱的身躯之中去。
<br/>　　这一股灵力，令白萱身上的实力，似乎是微微有着一些进展。
<br/>　　“杀了这些虚空兽，也可以夺取他们身上的灵力？这倒是一种互相猎杀的平衡。”
<br/>　　楚风眠看到这一幕，心中暗自想到。
<br/>　　虚空兽可以吞噬武者的血肉来增添他们的实力，但是武者也同样可以是去斩杀虚空兽，来夺取他们身躯中的灵力。
<br/>　　“这倒是正好，我现在已经是突破了瓶颈，只需要灵力足够，就可以凝聚远古战龙之血，为我增添实力！”
<br/>　　楚风眠心神一动，察觉到了这一点后，他便是直接冲了上去。
<br/>　　一个人顿时是杀入到了无数虚空雷鸟之中。
<br/>　　“什么？这小子是疯了不成？”
<br/>　　“一个人敢去闯入到虚空兽群去，这小子是在找死！”
<br/>　　“虚空雷鸟的实力虽然不强，但是三千虚空雷鸟，他一个人闯入群中去，每时每刻都会受到无数虚空雷鸟的攻击，真是愚蠢！”
<br/>　　“哼，想要夺取虚空兽的力量，结果自己找死的愚蠢每年都有，没想到这小子也是一个蠢货。”
<br/>　　“让他自己去死吧，就算是他引来其他虚空兽，我们离开便是。”
<br/>　　众人看着楚风眠的举动，都不由的冷笑起来。
<br/>　　尤其是那跟楚风眠有仇怨的罗辰，如今更是直接心中开始咒骂起来，诅咒楚风眠，想要让楚风眠直接死在了这虚空雷鸟群中。
<br/>　　楚风眠一下子冲入到虚空雷鸟群中，顿时便是无数虚空雷鸟，向着他攻击开来。
<br/>　　这些虚空雷鸟，不懂得什么高深武技，只有蛮力攻击，论起实力来说，远远不如同境界武者。
<br/>　　不过虚空雷鸟的数量繁多，无数攻击同时打出，寻常生死境武者，都很难抵挡得住。
<br/>　　“战龙之威！”
<br/>　　楚风眠冷喝一声，顿时在他的身边，战龙之威凝聚而出。
<br/>　　这些虚空雷鸟的攻击，明显在这战龙之威下，都是慢了许多。
<br/>　　这些虚空雷鸟，虽然是从虚空海中诞生的生物，但是有着兽灵，而远古战龙，便是万兽之尊。
<br/>　　感觉到战龙之威，这些虚空雷鸟不少都是心中开始畏惧，攻势减弱了许多。
<br/>　　随后楚风眠大手一挥，顿时在楚风眠周边的十几只虚空雷鸟，都是死在了楚风眠的手中。
<br/>　　同时这些虚空雷鸟中最为精粹的灵力，随着楚风眠手一抽，便是融入到了楚风眠的身躯之中去。
<br/>　　这十几只虚空雷鸟的灵力全吞噬，在楚风眠身躯中的远古战龙虚影中，又凝聚出了一滴远古战龙精血，令楚风眠的实力，又增添一分。
