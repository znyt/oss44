　　在漆黑的阴气之中，能够看到一些星星点点的发光的东西在山谷里跳动。
<br/>　　鬼火。
<br/>　　鬼火闪，魂灵出。虽然科学上有关于鬼火的解释——事实上解释得并不清楚——在法术界，一直认为鬼火是鬼灯笼，是鬼魂发出来探路的，鬼魂不一定都需要探路，但有大面积的鬼火闪烁，一定有鬼魂作祟。
<br/>　　“总算是出来了啊。”叶少阳看到这些鬼火，一点也不觉得紧张，反而有些兴奋，当即画了两张隐气符，跟毛小方一人贴了一张在身上，然后朝山谷贴过去。
<br/>　　来到那团阴云之中，叶少阳和毛小方用芭蕉叶擦眼，在阴气之中勉强能看见东西，定睛看去，在一片鬼火之中，出现了一道道鬼影，半透明的，在山谷里缓慢地漂浮着，看上去就好像轻飘飘的纸人被风吹着。
<br/>　　两人没有着急进去，站在谷口朝里面张望起来，因为目光只能看到山谷的一部分，看不到太远的地方，因此只看见这些鬼影在风中飘，越来越多，也不知道是从哪里出来的。
<br/>　　“对，我上次进来就是这个样子。”毛小方说道，“起初还没这么多，后来我走着走着，越来多多，好像是从地下冒出来的。”
<br/>　　“地下冒出来？”
<br/>　　叶少阳感觉不可思议，但也没有多问，毕竟真相就在眼前，有时间问还不如自己去看看，于是招呼毛小方一起进入了山谷。
<br/>　　因为隐气符的缘故，这些鬼魂看不见他们。两人从山谷里一路走过去，起初山谷还比较开阔，后面山谷变得狭窄起来，叶少阳总算看清了是怎么回事：
<br/>　　一股股阴气，从地下冒出来，在空中逐渐形成了人的形状，然后就开始在山谷里游荡。
<br/>　　这怪诞的一幕，就算是对什么都见识过的叶少阳来说，也是感到十分惊讶。
<br/>　　“这是什么鬼东西，你上次来也是这样？”叶少阳贴在毛小方耳边说道。
<br/>　　毛小方也很震惊，喃喃道：“上次没有月亮，我也忙着赶路，根本就没有看清楚……我也不知道这是怎么回事。”
<br/>　　两人越往前走，越是感受到山谷里阴气浓郁，就算两人是法师，一般不会被引起入侵肉身，也感到浑身寒冷，在强大的压力之中，有点喘不过气的感觉。
<br/>　　两人一直走到山谷的末端，这里是整个山谷之中阴气最浓郁的地方，从地下“冒”出来的阴魂也是最多。在往前走，出了山谷，顿时感到压力完全消失，身边没有一丝阴气蔓延出来。那些阴魂也在徘徊到谷口的位置就回去了，仿佛这里有一道看不见的屏障。
<br/>　　“少阳子……”
<br/>　　毛小方刚叫了一声，叶少阳就摆了摆手，“咱们先观察，有什么疑点就记下来，待会再总结。”
<br/>　　毛小方点点头，两人站在谷口朝里面望去，观察这些游荡的阴魂，这些阴魂都是半透明的，就好像一张图片被放大成了锯齿的形状，根本看不清脸，只能模模糊糊得看到身上的衣服，都是普通人的装束，看上去很古老。
<br/>　　“这应该是明朝时候的打扮，不是现代的，现代人穿的衣服，都是改良过的。”毛小方一眼就看出了区别。
<br/>　　叶少阳问道：“为什么不是清朝的？”
<br/>　　毛小方朝不远处一个鬼魂努了努嘴，道：“看不见吗，这些人头发都是盘起来的，没留辫子。”
<br/>　　叶少阳恍然，不由有些疑问，盯着从地下冒出的一缕阴气形成的人形，说道：“这些应该都不是真的鬼，而是魂气所化，因为某些原因，盘踞不散。”
<br/>　　所谓魂气，其实主要是指怨气，人死之后，鬼魂离开人间，本不会在人间留下什么魂气，就算有，也很快就散去了。
<br/>　　但如果人是横死，怨气就会很深，如果在一个封闭的地方，这种怨气就会集结，因此叶少阳之前每破掉一处阴巢鬼穴，都要找老郭帮忙清理现场，不如鬼魂的怨气会长存下来，滋生邪灵。
<br/>　　不过……在这种开阔的山谷里，叶少阳还是很难想象，为什么会有怨气集结。而且最不可思议的是，这些怨气居然强大到能够聚拢成人形……
<br/>　　只有一种可能，那就是这地下一定有着什么东西，不管能容纳这些怨气，而且还能保存这些死亡之人的一些神识……所以这些怨气能够凭着一种本能，仍然形成生前的形象。
<br/>　　叶少阳和毛小方观察了半天，这些怨气生成的魂灵，一直都在山谷里绕圈子，似乎没有什么特定的目的。
<br/>　　而且从地下冒出来的“魂灵”越来越多，说是百鬼夜行，但完全不止上百个，至少有好几百个之多。这幅场面，真的有点骇人了。
<br/>　　叶少阳反正是没有见过这样的场面。
<br/>　　观察得差不多了。叶少阳画了一张灵符，对着不远处的一只“魂灵”贴下去，将其收在灵符之中，想要研究一下这怨气是怎么回事，引燃了灵符，丢在空中。
<br/>　　灵符丢出去，在空中燃烧起来，叶少阳睁大眼睛看着，等着怨气被溶解，结果……等到灵符燃烧干净，怨气也没有溶解，从灵符里发出了一声凄厉的叫声，接着，从灵符的灰烬里飞出的……居然是一粒粒的精魄。
<br/>　　“什么！这是鬼魂！！”叶少阳吓的失声叫起来。
<br/>　　毛小方也是一脸懵逼。
<br/>　　叶少阳看着精魄飞走，内心的震撼，没有言语可以形容，再看山谷里这些数以百计的鬼影……居然，都是真的鬼魂，而不是怨气集结之后产生的影像……
<br/>　　这怎么可能！
<br/>　　这山谷如此空旷，又不是什么封闭的阴巢鬼穴，怎么可能有这么多的鬼魂徘徊不走？阴司也不管？再说这些鬼魂是从那里来的，为什么看上去这么模糊，一点也不像真实的鬼魂？不然的话，自己也不会把他们仅仅当成影像了。
<br/>　　这些问题，在叶少阳心中翻来覆去，同时想到刚才无意中被自己杀死的那个鬼魂，心中一阵自责，急忙又抓了鬼魂在手中，仔细看，那鬼魂也不挣扎，好像完全没有意识。
<br/>(本章完)
