　　这一群红衣人的打扮，虽然楚风眠没有见多，但是他们身上的气息，楚风眠却是无比熟悉的。
<br/>　　六道魔门的功法。
<br/>　　这一群红衣人，都是六道魔门的弟子。
<br/>　　只是不知道为何，六道魔门的衣着改变。
<br/>　　如今这一群六道魔门弟子，正在追杀这一群武者，这一群六道魔门弟子，几乎都生死境之上的武者。
<br/>　　几个为首的，都已经是踏入了生死台阶，成就皇者的人物，而这一群被追杀的武者，最强的不过是一名刚刚踏入生死台阶的老祖。
<br/>　　在这追杀之下，几乎是毫无还手之力，被连连杀戮。
<br/>　　斩杀了一名武者，几名六道魔门弟子便是都围了上去，不一会那被杀的武者，身躯就变的干瘪起来。
<br/>　　好像是一身精血都被抽空了一样。
<br/>　　楚风眠明白，这是六道魔门弟子，在施展魔功吞噬其他武者的精血，以增强实力。
<br/>　　这种魔功，就是六道魔门的根本，曾经的六道魔门统御九域的时候，不知道是有多少的强者，都是死在了六道魔门的手中，被他们所吞噬。
<br/>　　后来六道魔门被推翻，九魔城虽然嚣张，可也不敢如此大张旗鼓的去杀戮其他武者，施展魔功。
<br/>　　看来如今九域剧变，六道魔门的实力也发生了翻天覆地的变化，按照传闻之中。
<br/>　　单单是一个六道魔门，几乎就足以跟九域的其他六大宗门所对抗了。
<br/>　　几乎是成为了九域的半个霸主。
<br/>　　如此一来，大量的六道魔门弟子，也不在害怕什么了，大开杀戒，北域还好，这西域就是一个重灾区。
<br/>　　西域之中的各个宗门，都没少被六道魔门所屠戮，甚至是曾经称霸西域的终焉王朝，都在六道魔门的屠杀之下，崩塌离析。
<br/>　　终焉王朝崩塌，群龙无首，整个西域都是成为了乌合之众，几乎没有宗门可以抵挡得住六道魔门。
<br/>　　这样的景象，楚风眠也只是听说过，没想到这次居然是碰到了。
<br/>　　“好爽！”
<br/>　　“这一次出来杀戮！我的实力又足足增长了三成！”
<br/>　　“还要多谢魔帝的命令，让我们出来猎杀其他武者！这才只是西域的几个小势力，等我们杀到中域去，在吞噬精血，实力还会暴涨！”
<br/>　　那一群六道魔门弟子，一边屠戮，一边大笑道。
<br/>　　突然之间，几道目光，看向了楚风眠的遁光。
<br/>　　“有人，好快的遁光！”
<br/>　　“杀，杀了他！”
<br/>　　“杀了此人！”
<br/>　　这一群六道魔门弟子，都是入魔之辈，看向楚风眠，一个个当即咆哮一声，向着冲了过来。
<br/>　　施展出数道魔功，对着楚风眠轰击而来。
<br/>　　“找死？”
<br/>　　楚风眠眼神冰冷无比，对六道魔门他本来就没有好感，甚至是跟六道魔门的魔帝，楚风眠可是有着死仇。
<br/>　　这群六道魔门弟子居然还敢来找楚风眠的麻烦。
<br/>　　楚风眠大手一挥，凌空一拍，这漫天魔功，都被一掌打碎，几名生死境的六道魔门弟子，当即是被凌空打爆。
<br/>　　“怎么回事？”
<br/>　　“什么人？敢来跟我们六道魔门为敌！”
<br/>　　远处的六道魔门弟子，也察觉到了这里出现的战斗，一个个目光看了过来，正好看到楚风眠轰杀那几名六道魔门弟子的一幕。
<br/>　　“杀！”
<br/>　　“敢跟我们六道魔门为敌的，都要死！”
<br/>　　一群六道魔门弟子向着楚风眠冲了过去，其中两名为首，实力达到天命之境的六道魔门弟子，率先出手，两道魔爪直指楚风眠的喉咙抓了过去。
<br/>　　“死！”
<br/>　　楚风眠眼神都没有眨一下，手轻轻一挥，两道魔爪被当即震碎，随后恐怖的灵力席卷一切。
<br/>　　那两名六道魔门的皇者，被灵力席卷进去，当场被碾碎轰杀，凌空爆炸开来。
<br/>　　剩下的六道魔门弟子，楚风眠也一个没有放过，一掌之下，尽数轰杀，瞬息之间，刚刚还在想肆虐的一群六道魔门弟子，没有任何一人还活着。
<br/>　　那一群被追杀的西域武者，一个个都是站在原地愣住了，许久才是如梦初醒一般，向着楚风眠跪拜下来，来感谢楚风眠。
<br/>　　“想活命的话，就去北域吧。”
<br/>　　楚风眠看了一眼，开口道。
<br/>　　北域是剑道门的领地，楚风眠绝不会允许六道魔门，在北域肆虐。
<br/>　　说完，楚风眠便是身形一闪直接离开了。
<br/>　　楚风眠救了他们，也只是举手之劳，让他们去北域，也就是给他们一条生路。
<br/>　　至于这一群人能不能安全到北域，就不是楚风眠关心的事了。
<br/>　　在西域，这样的事每天每时每刻都在发生，楚风眠不可能全部照顾到。
<br/>　　不过一路上，碰到的六道魔门弟子，楚风眠几乎都没有放过，全部都是轰杀了。
<br/>　　魔帝如果知道楚风眠还活着，必然会跟楚风眠一战，甚至是跟剑道门一战。
<br/>　　杀了这些六道魔门弟子，也是在削弱着六道魔门的实力。
<br/>　　楚风眠一路杀戮，一路前行。
<br/>　　死在楚风眠手中的六道魔门弟子，至少已有数百了。
<br/>　　不知道是不是六道魔门已经是察觉到了这个消息，到后来，楚风眠就碰不到六道魔门的弟子了。
<br/>　　看来是这些六道魔门弟子，在刻意躲着楚风眠。
<br/>　　如此一来，楚风眠倒是也省了很多麻烦，索性直接向着归墟城进发。
<br/>　　曾经需要传送法阵，都需要折腾十几天才可以前往的归墟城，楚风眠凭借遁光，三个时辰，就已经是到来了。
<br/>　　楚风眠的身形，来到了归墟城外。
<br/>　　归墟城，本只是西域的一座小城，平日里默默无闻，只有每一次归墟天开启的时候，才会热闹一二。
<br/>　　不过现在，归墟城倒是大变样了，一座新城拔地而起，因为归墟天现在已经是彻底开启了。
<br/>　　进入归墟天的武者数量，也变得比起以前多得多了，不少武者进入归墟天，都碰到了好运气，得到了不少的宝藏。
<br/>　　归墟天之中的宝藏数量，楚风眠可是在清楚不过了，他当时进去的时候，一身实力也只是在生死境，就足以得到不少的宝藏。
<br/>　　更何况现在九域的实力整体提升，踏入生死台阶，甚至是踏入天命之境的武者，都不在少数了，在其中有所收获，也无比正常。
