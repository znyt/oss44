<br/>    一根紫色的藤蔓之下，悬着三十八块无字碑。
<br/>    
<br/>        陈明走到这三十八块无字碑之前，仔细看向这些无字碑，面板已经是师德开始降低，三十八块无字碑而已，也就三十多万师德顿悟到超脱的境界。
<br/>    
<br/>        陈明存了上千万的师德，如今终于是开始花了。
<br/>    
<br/>        其中有一块无字碑，凝结出的道种便是咫尺天涯。而其他的道种，还有不少很是神奇的，比如缩地成寸，掌中佛国，三寸人间，虚空破灭，掠空之刃，甚至于，其中有一颗道种，直接就是叫做神奇！
<br/>    
<br/>        而这一颗道种，同样是这神奇。
<br/>    
<br/>        而这神奇的效果，也是让陈明觉得匪夷所思，竟然是可以凭空出现某种东西！
<br/>    
<br/>        类似于空间转移位置类的法术，只是更加近乎空间大道。
<br/>    
<br/>        而且陈明凝聚了这么多的道种之后，竟然是感觉自己施展八步登龙的法力量可以减少一倍！
<br/>    
<br/>        这就是功法的强大之处了！
<br/>    
<br/>        对于道的领悟更高，施展相应的术法的时候，消耗的法力自然越少。
<br/>    
<br/>        将三十八块无字碑全部看完，陈明终于是开始布置这些无字碑的位置，苍空亲王屁颠屁颠地开始遵从陈明的安排，开始布置这些无字碑的位置。
<br/>    
<br/>        苍空亲王一边布置，一边小声问道：“上师，不知道是哪个地方可以立功？”
<br/>    
<br/>        月宫那边动了动，便平白得了一个星系，他太虚移动了这无字碑的位置，不知道将会有什么好处，陈明道：“妖族就那么大，若是想要侵吞，无冕者议会可不会轻易同意，我话就说到这里，天机不可泄露，泄露了就不灵了。”
<br/>    
<br/>        苍空亲王心想：妖族的地盘不够分，确实不够分，妖族太小了，四大仙国的胃口肯定不能满足，那就是要动无冕者议会的手了！
<br/>    
<br/>        苍空亲王当即点头道：“明白了，明白了，这么大的手笔，上师恐怕要布置完四大仙国才会开始吧。”
<br/>    
<br/>        陈明微微一笑，“孺子可教也！这里的事情安排完，送我去天
<br/>    
<br/>        庭吧，最后去瑶池。对了，你们的藏经阁最近没有失火什么的吧？”
<br/>    
<br/>        苍空亲王道：“没有，绝对没有！”
<br/>    
<br/>        陈明心中不喜，要是失火了，自己还能去看一看这太虚的术法啊，太虚的仙术，仙人指路，可是强大无比啊！但是面上却是只能说，“这就好，这无字碑虽然重要，但是修行了也只是修为高深，有了仙术，才是决定胜负的一环啊！”
<br/>    
<br/>        四大仙国修行无字碑的修行者不在少数，但是能够被允许修行大仙术的，却是不多，毕竟修行了无字碑，也就是修为高，耐揍，真正打起来，还是看谁的仙术厉害。
<br/>    
<br/>        能够修行仙术，只有亲王。
<br/>    
<br/>        像是修行无字碑的这种，大概就算是有了成为亲王的资格，等到立功，忠诚足够之后，或是攀上帝主，才有资格修行大仙术。
<br/>    
<br/>        这些大仙术，都是帝主亲自保管的，陈明压根没有任何机会接触到。
<br/>    
<br/>        仙界可不是燕山，人人都可以修行仙级功法，各种各样燕山有的仙术，只要你修不死，那都可以修。
<br/>    
<br/>        仙界对于大仙术的管制，还是比较严格的。
<br/>    
<br/>        毕竟诸位帝主都要维护自己的统治。
<br/>    
<br/>        就如同镇魂兵都在帝主手中一般，只有经过了帝主的允许，才会动用这种程度的杀伐重器。
<br/>    
<br/>        在太虚一行之后，陈明也是没有动虚空花的心思，这玩意带不走，跟太阴金桂一样，都带不走，唯有留在此处才有机会得到其中的造化。
<br/>    
<br/>        只能想办法弄到天庭的造化仙果和瑶池的蟠桃了。
<br/>    
<br/>        陈明在太虚结束了他的行程，接着出发向天庭，不过刚刚来到天庭，陈明再次驾临，二太子率领着上千的金仙恭迎，可谓是给足了陈明的面子。
<br/>    
<br/>        毕竟陈明在月宫的手段，也是让世人惊讶不已，对陈明只有两个字：拜服！
<br/>    
<br/>        二太子伸手揽住陈明的胳膊，惹得陈明觉得这二太子是不是有着别的嗜好，比如什么男上加男，知男而上啊！
<br/>    
<br/>        二太子讨好道：“我天庭，已经恭候上师久矣。”
<br/>    
<br/>        陈明笑道：“莫急，我这就去安排。小僧想了很久，此番动作太大，对于自身的安全，一直都是有着一个担忧，小僧四国行走，未免某些黑暗中人加害，所以想要组建一个卫队，为了保证这卫队的不会有奸细，所以小僧想要自己挑人，所以在这造化仙果的方面，不知道........”
<br/>    
<br/>        二太子思考了片刻，然后道：“上师，今年的收成本就不佳，加上众多的仙王，太子，公主都已经分了不少的造化仙果，所以，我这里，能够拿出来的造化仙果，可能不多。”
<br/>    
<br/>        陈明微微皱眉，然后道：“小僧身死没有什么，只是孑然一身而已，也算不上什么有名之人，只是，若是坏了我谋划的大事........”
<br/>    
<br/>        二太子咬咬牙道：“最多就是十颗大罗金仙的造化仙果，一千颗金仙的造化仙果，这便是天庭的最大魄力了，上师也知道，这一次百年大战，无冕者议会获得了全面的上风，我们有大量的造化仙果，都要卖给他们。”
<br/>    
<br/>        陈明心说：你管这个叫不多？
<br/>    
<br/>        还是说我太穷了？
<br/>    
<br/>        看看之前抢的那个天庭帝子，这完全不是一个级别的存在啊！
<br/>    
<br/>        本以为我燕山财团富有天下，现在才知道，什么叫富的漏油！
<br/>    
<br/>        这一次，陈明也是再次意识到了四大仙国，到底是何等的庞然大物，而无冕者议会，足以和这样的四个怪物抗衡，又是一个强大到了什么程度的怪物？
<br/>    
<br/>        陈明道：“唉，算了，也只能这样了，等到下一个百年大战，本上师亲自谋划，必定能够有更多的造化仙果。”
<br/>    
<br/>        二太子笑道：“那是当然，上师出手，必定能叫无冕者议会惨败，我这就去请示父皇，尽快给上师将这些造化仙果拿来！”
<br/>    
<br/>        陈明拱手道：“事不宜迟，带我去你们无字碑的放置之处，我施展妙手，给你们改改风水！”
<br/>    
<br/>        二太子笑道：“多谢上师，我这就带上师过去。”
