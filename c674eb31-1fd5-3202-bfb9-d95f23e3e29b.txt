
一夜无话，到了隔天的时候，清晨就迎来了黎明的阳光，今个看起来天气似乎不错，就连京城的雾霾天都在瞬间一扫而光了。<br/>　　<br/>　　这么好的天气，王惊蛰白日里却哪也没去，在潘家园的宝瑞福茶楼里喝茶，一壶陈年的白茶，透着淡淡的香气，闻着沁人心脾。<br/>　　<br/>　　王惊蛰闭着眼睛，摇头晃脑的品着茶嘴里哼起了小曲，茶间的网络电视上，放着一段节目，两个人在讲着段子。<br/>　　<br/>　　“话说那常山赵子龙啊还真是一员猛将，七进七出长坂坡，一杆丈八红缨银枪三十六套招数凶猛无比，杀的长坂坡是溃不成军，都水流成河了”<br/>　　<br/>　　“咦，不是应该血流成河么？”<br/>　　<br/>　　那人顿了顿，一本正经的说道：“但那长坂坡，也不是个省油的灯，不仅长得如花似玉，人也是膀大腰圆的，乃是正宗的头牌，她看着赵云媚眼如丝……”<br/>　　<br/>　　“噗”刚喝了一口热茶的王惊蛰突然就喷了，抹了下嘴巴子，无语的说道：“这都是什么和什么”<br/>　　<br/>　　茶间的门这时被推开了，贺三刀进来后看着桌子上的茶渍，挺心疼的说道：“您是真能糟蹋好东西，这白茶一年是茶，三年是药，七年就为宝了，我一共就藏了八两多的九年陈白茶，都多长时间没拿出来泡了，你行行好可别浪费了啊”<br/>　　<br/>　　王惊蛰拿着抹布擦了擦桌子，笑道：“我要是说全都要走了，你不也得舍得么？”<br/>　　<br/>　　贺三刀顿时笑了，坐在他对面也倒了一杯茶，点头说道：“别说这陈年白茶了，我这宝瑞福里有什么东西了，您看上眼了随便拿，关键是我还真怕你不要呢，您可不要担心我舍不得，尽管张口就是了”<br/>　　<br/>　　“得，拿人手短，拿了就是人情，我还是算了吧……”王惊蛰一口喝干了杯中的茶水，顿了顿后接着说道：“我还真有点事要和你讲呢，帮个忙”<br/>　　<br/>　　“嚯，这可新鲜了哈”贺三刀诧异的说道。<br/>　　<br/>　　“我也不是那七进七出长坂坡的虎将赵子龙，自然也有要求人的时候，有些事毕竟自己也办不来么”王惊蛰手指敲着桌子，寻思了寻思后说道：“找两三个眼尖手勤快，胆子又大，办事利索的人借我用用，最主要的是嘴要牢，我在京城也不认识什么人，就只能找你借了，然后再给我弄一辆车”<br/>　　<br/>　　贺三刀诧异的问道：“就这？”<br/>　　<br/>　　“你帮我把这些条件都凑齐，那就行了”<br/>　　<br/>　　“那您就请好吧，我在京城也不是什么大人物，不过就有一个优点，朋友多，什么三教九流的人都认识”贺三刀想了想，拿出手机说道：“我给你介绍两个吧，都是三十来岁左右，以前是跑江湖的，人很稳很托底……”<br/>　　<br/>　　贺三刀随后打了个电话，跟那边寒暄了几句之后就问他们有没有空，聊完后他朝着王惊蛰点了点头，说个把小时人就能到了。<br/>　　<br/>　　接下来两人喝茶闲聊等着人，十点来钟的时候，茶间门被敲响了，贺三刀打开门后外面站着两个青年，都是三十岁上下，穿着夹克牛仔裤，梳着短寸头，一看就挺精神的，王惊蛰打量下两人的面相，觉得确实都不是奸猾的小人，用着应该没啥问题。<br/>　　<br/>　　人来了之后，贺三刀介绍道：“惊蛰，这是大宝，这二宝”<br/>　　<br/>　　王惊蛰顿时一愣，拱了拱手说道：“哥俩啊？”<br/>　　<br/>　　“唉，对了，亲兄弟名字里都带个宝字，我平时就是这么称呼他们的，跟我认识能有七八年了”贺三刀又转而跟两人说道：“这是我恩人王惊蛰，你们哥俩跟着他去办事吧，看我的面子，小哥让你们干什么就干什么，出了什么问题由我兜着就是了”<br/>　　<br/>　　王惊蛰摆手解释了一句：“总归不是杀人放火的事，嗯，可能会行走在法律边缘吧，但我保证问题肯定没有”<br/>　　<br/>　　大宝和二宝直接干脆利索的说道：“贺爷介绍的，我们放心”<br/>　　<br/>　　随后，王惊蛰和两人就从宝瑞福出来了，茶馆外面停了一辆很普通的福特商务，挂着京牌。<br/>　　<br/>　　上了车后，大宝开车，二宝在副驾驶，就朝着后面的王惊蛰问道：“王哥，咱去哪里啊？”<br/>　　<br/>　　“往东四环那边走，到那里差不多也该中午了，在附近找个地方吃口饭吧……”<br/>　　<br/>　　车子开出潘家园，往东四环方向走，顺着王惊蛰的指示，在那块地皮的附近他们三人随便找了个小饭馆吃饭。<br/>　　<br/>　　吃完饭差不多就下午一点了，福特来到那块地皮前面，就停在了路边，这一停就足足是两个多小时的时间都没动，王惊蛰就靠着车窗一直望着那块荒地。<br/>　　<br/>　　大宝和二宝这哥俩都挺好奇的，不知道贺三刀安排的这个年轻人到底要干啥事，好像是专门来这发呆的一样，但他俩还挺守规矩的，尽管诧异也没问为啥。<br/>　　<br/>　　下午五点多钟，那块荒地里这才有人出来了，是三个中年男子，背着手往外走。<br/>　　<br/>　　王惊蛰这时坐直了，指着那三人说道：“你哥俩跟上他们，但别让人发现了，就一直跟着，等什么时候有人落单了，然后就给我打电话”<br/>　　<br/>　　“好叻，王哥，小事……”<br/>　　<br/>　　大宝和二宝说完就从福特里下来，尾随那三个人过去了，等到过去了一个半小时左右，二宝才给他打来电话，说是其中有一人独自走了，上了一辆出租车。<br/>　　<br/>　　王惊蛰当即就让大宝赶紧回来，然后让老二打车跟上对方，不要跟丢了。<br/>　　<br/>　　七点多钟，福特跟上的出租车开到一小区门口后就停了下来，对方下车了。<br/>　　<br/>　　福特停下来，等在一旁的二宝拉开车门就进来了，王惊蛰指着正往小区里走的人影，说道：“哥俩，车开过去后，把人按住，然后直接拖进车里来”<br/>　　<br/>　　大宝和二宝对视一眼，其中一人说道：“王哥，抓人我们倒是没问题，不过抓了之后会有点麻烦，您之前也没说要干这活啊，所以我俩开的车是干净的，这小区里又有摄像头，回头对方要是……”<br/>　　<br/>　　王惊蛰笑道：“去吧，没事，把人带上车来就行了，放心，他最后是不会报警的，没人会查你们”<br/>　　<br/>　　大宝皱了皱眉，略微思索了下，点头说道：“行，信您了！”<br/>　　<br/>　　随即，福特加快速度开了过去，当车经过那人身旁的时候，就“嘎吱”一脚急刹停住了，二宝直接拉开车门，探出半个身子，手脚非常利索的就一手捂住对方嘴巴，一手抓上了他的胳膊，将人给拽进了福特里。<br/>　　<br/>　　“哗啦”车门关上，大宝一踩油门，福特停顿了不到四五秒钟，就把人给劫上了车，然后开走了。
