　　“轰！”
<br/>　　雷霆炸响，九霄云动，一道巨大的雷剑贯穿一只火红色的狂暴巨兽，巨兽惨叫，吼声震天，声音传遍苍茫大地。
<br/>　　“畜生，受死！”
<br/>　　“杀！”
<br/>　　“它快不行了！杀啊！”
<br/>　　“……”
<br/>　　紧接着，数十道巨大的光华法术打在巨兽身上，神通爆炸，轰隆声不断，巨兽开始躲避，想要逃跑。
<br/>　　“让我来！”就在这时，只听一道冷峻的声音响起，紧接着，一道巨大的光芒笼罩天地，锋芒毕露，剑光璀璨。
<br/>　　“萧萧剑雨！”
<br/>　　“嗽嗽嗽！”
<br/>　　响声不断，如大雨倾盆，淅淅沥沥，散落大地，落下的是雨滴，斩出的却是无尽剑意。雨幕遮天，滴落下来，势无可当！
<br/>　　“吼！……”
<br/>　　那全身火红色的巨兽一阵嘶吼，终于不敌剑雨的锋利，轰的一声，倒在了地上，生机渐逝，显然是活不成了。
<br/>　　“呼……这火岩兽真是厉害，与我们同一境界，竟然如此厉害！”
<br/>　　“是啊，我们整整追杀了一天，才将它杀死，希望它能爆出点厉害的东西来！”
<br/>　　“是啊，我听说上次那个叫‘王者’的小队，人家可是找到了好几本仙法神通，更有一柄神枪，真是好运！”一人如此说道，说到这里，他眼中路露出无比羡慕的神色。
<br/>　　“我听说一个叫‘轮回’的小队，也是不得了，他们找到一个大怪，废了三天三夜将之杀死，然后更是爆出了一堆宝物，真是厉害。”
<br/>　　“我们什么时候能够像他们那样啊？”
<br/>　　说到这里，显然是所有人都十分渴望，渴望得到灵神界中的宝物。灵神界中虽然危险，但也机缘无数，若是能够得到一二，那就不下于一夜暴富。更可况，灵神界中又不会真正死亡，死亡之后还能复活，最多就是大病一场，生死都不怕了，还怕什么？
<br/>　　“这次还是得多谢萧萧公子，公子的萧萧剑雨，真是厉害，若不是萧萧公子，我们此次要对付这火岩兽，可就不容易了。”有人对着一个身穿青色衣衫的剑客感谢道。
<br/>　　“是啊，……”
<br/>　　“不客气！”男子抬起头来，看了一眼众人，淡淡地说了一句。
<br/>　　“轰！”
<br/>　　就这此时，那倒地的火焰兽突然爆炸开来，显然，那火焰兽是真的死了。谁也没有注意到，那火焰兽在爆炸后，一道神秘的力量升起，重新融入到天地之间，只等着吸收足够的能量，便会重新孕育出来。
<br/>　　“看，那是什么？”有人指着爆炸的地方，惊叫道。显然，大家对于打怪爆宝物还是很在意的。
<br/>　　众人看去，只见迷雾散开，光华流转，一道道赤色红光在闪烁着，于那红光之中，悬浮着一块寸许大小的玉简，它正在缓缓旋转着。
<br/>　　“玉简？这是神通？还是仙法？”众人大惊，之后便是大喜，显然，他们都知道，玉简记载着重要的信息，不会简单。
<br/>　　一道流光闪过，玉简便被萧萧公子抓到手中，他的神识一探，顿时，一股信息便冲入到他的脑海之中，演化出各种招式，最终汇成‘火神怒’三个大字。
<br/>　　“咔擦！”
<br/>　　紧接着，一声清脆的声响响起，玉简破碎，散落在地。显然，这玉简是一次性的。
<br/>　　“什么？这玉简竟然碎了！那我们岂不是……”
<br/>　　顿时，所有人大惊，紧接着便是大怒，大家废了九牛二虎之力，才将这火焰兽灭了，如今得到宝物，只能给一人用，这算什么事？所有人看向萧萧公子的眼神都变了。
<br/>　　“大家别急，这是一道神通，我会将它复制给大家的。”萧萧公子见此，心中一惊，连忙说道。
<br/>　　“嗯？”听了他的话，众人停了下来，看着他怎么做。
<br/>　　萧萧公子轻轻松了一口气，虽说他的实力比这些人要强一些，但要让他一人对十多人，还是有些不及的。他连忙取出一块玉简，想要复制这‘火神怒’的神通。
<br/>　　很快，他便复制了两份。紧接着，他便发现，他复制的‘火神怒’并没有先前直接传到他脑海之中的好，好像差了一些什么。他立刻就知道，有些东西是自己这等修为无法刻录的。
<br/>　　没有多管，他立即刻录第三块。可是，当他想要复制第三块之时，却发现自己什么也复制不了，或者说，就算复制了下来，也不能看。顿时，他心中大急。怎么回事？怎么还有这种限定？我怎么不知道？该死！
<br/>　　他心中一阵怒骂，看着眼前虎视眈眈的十几人，他心思微微一转。连忙将手中刻录好的一块玉简扔到人群中，大喊一声：
<br/>　　“神通在此，给你！”
<br/>　　之后，他转身便向着远处飞去，复制不了，要他怎么说啊，就算别人知道，但也不会甘心啊，此刻不跑，更待何时？至于扔出一块玉简，当然是要制造混乱了，好让他逃走。至于为何只扔一块，因为此刻扔一块与两块，效果都是一样的。
<br/>　　“是我的！那是我的！”……
<br/>　　果然，几人见到玉简，连忙争抢了起来，众人一下子便乱了，争抢不断。
<br/>　　“该死，我们被他骗了，赶紧去追！”有几人立即反应过来，连忙向着萧萧公子跑去的方向追去。
<br/>　　“追！”
<br/>　　……
<br/>　　“暗影族的贼子，竟然敢偷我宝贝，拿命来！”
<br/>　　两道流光呼啸而过，却是两人在半空中追逐不断，抢夺宝贝。
<br/>　　“炸天帮的成员们，速来集合，龙傲天带人来要与我们开战了！”
<br/>　　一名人族，举着一面大旗，旗帜上写着“炸天”两个嚣张大字，在高声呼喊道。
<br/>　　“人族宵小，可敢与我暗魔族一战？”
<br/>　　一个全身散发着魔气的怪物在半空中叫嚣不止。
<br/>　　“咻！”
<br/>　　一道剑光闪过，斩在了虚空之中，斩破天际，将那暗魔族怪物斩成两半，跌落下来。
<br/>　　“魔神队与火灵族开战了，就在魔龙山脉，天啊！”
<br/>　　“青花信物与飞翔真人大战于摩天海，谁胜谁负？三日之后，便分结果！”
<br/>　　“……”
<br/>　　灵神界此刻却是彻底热闹起来了，约战高手，种族对战，私人恩怨，杀人夺宝，队伍争锋等，各种各样的战斗，全都在此刻发生着。
<br/>　　万族大战，道法之间的触碰，神通之间的对决，仙诀之间的较量，大道之间的争锋，智慧之间的碰撞，剑客刀客，仙人武者，究竟谁能独领风骚？
<br/>　　而灵神界之中的天道‘道我’，则是在不断地吸收着他们的战斗经验、大道神通等，领悟出更多的神通，汇聚到元始大道之中。
<br/>　　灵神界中，死亡之人，便会残留一部分精神力，被灵神界吸收，这种变化是极其细微的，谁也没有发现，或许圣人已经发现了，但他们并没有多说。在灵神界之中的每一个人，都会自然而然的对灵神界产生一种信仰念力，而这种力量，也会被灵神界吸收，无数种族的念力汇聚起来，那是一种无法想象的力量。
<br/>　　修士们，在不断战斗着，灵魂不死，争斗不止！
<br/>　　灵神界，在不断蜕变着，信念不灭，蜕变不停！
<br/>　　……
