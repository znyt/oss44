　　听着北北的话，叶帝这心里也是稍稍有些自责，自从OG战队被他们收购回来之后，貌似自己等人根本就没有去管理过这个战队，只顾着忙着自己的了。
<br/>　　“行，北北你开免提吧。”叶帝开口说道，等到那边北北回话之后，叶帝再次开口。
<br/>　　“首先我要和大家说一声对不起，战队被收购过来之后，我这边只顾着忙着荣耀战队的事情了，倒是忽略了大家，今天如果不是北北来电话，恐怕我都忘了你们线下赛的事情了。”叶帝开口说道。
<br/>　　另一边听着叶帝的话，六七名小伙子连忙说道没关系，听起来倒是有些局促、又有些激动的样子。
<br/>　　而这个时候一旁的蒋念等人也是知道是谁给叶帝打的电话了，听着叶帝和他们的对话，几人也都是无言。
<br/>　　“哥，你还是先忙着白队的事情吧，你们拿下冠军后，到时候再来管理我们，那我们岂不是更有面子，你现在要过来的话，很容易分心的，到时候在拿不到好名次，那就真的不好了。”另一边北北连忙开口说道。
<br/>　　在北北说过话之后，其余几人也都是跟着开口应道，表示并不希望叶帝在这个时候分心，他们能够听到叶帝鼓励两句就已经非常知足了。
<br/>　　“那这样吧，之前我准备了一首歌，过两天我送给大家，希望你们能够取得好的名次。”听着他们的话，叶帝开口说道。
<br/>　　因为歌曲他心里都有草稿，加上有林奇他们，这个肯定是非常省时间的，所以叶帝才这般开口。
<br/>　　果然听到叶帝这么说，OG战队的几人顿时激动了起来，一个个都是有些不敢置信的在那里问是真的么。
<br/>　　“当然是真的，我什么时候说过谎，周二吧，周二我会开直播，到时候你们来看，这首歌是给你们的，也是给所有努力的职业选手的。”叶帝开口说道。
<br/>　　双方又聊了一会后，叶帝这边便是挂断了电话。
<br/>　　“哥，你这脑袋是怎么长的，又有新歌了？”看到叶帝挂断电话，香蕉一脸惊奇的说道。
<br/>　　“人家需要你鼓励，你的歌曲是这个方面的么，别来一些情呀爱呀的，那可就偏离主题了。”林雅楠跟着叮嘱道。
<br/>　　“就是，你今天已经把牛吹出去了，到时候可千万别丢人啊。”柳清月同样点头，一脸的不相信。
<br/>　　之前杨小艺将柳清月带到亚洲区大师组后，柳清月就开始自己玩游戏了，没有继续在跟杨小艺双排，若不是这样的话，也不可能出现小艺和清浅双排的局面。
<br/>　　当时柳清月感觉自己到了那个分段已经是极限了，如果在继续下去的话，恐怕小艺都上不了分，那样的话她就拖对方的节奏了。
<br/>　　而且能够从国服钻石上到洲区大师组，这对于柳清月来说已经是一件非常开心的事情了，就因为这个，他直播间里的人气倒是又上涨了不少。
<br/>　　“放心吧，我什么时候丢过人，叶帝出品，必属精品！”叶帝自信的开口说道，他拿出来的这首歌，绝对非常适合赛场上的选手们。
<br/>　　结束了和对方的对话后，叶帝便是点开了自己的排位赛，依旧是单排，而且用的还是他的大号。
<br/>　　因为这段时间的懈怠，世界区王者组第一的位置重新被金相赫给夺了回去，而叶帝则是位列第二。
<br/>　　大概过了十多分钟的样子，随着‘砰’的一身金属撞击声，叶帝这边已经是排到了队友。
<br/>　　想要保持一个非常好的竞技状态，每天打这种高端局排位那是必不可少的，因为很多时候在这种高端局的排位当中，大家打的比之比赛还要严谨。
<br/>　　能够打到王者组前列的，这肯定都是各个战队的当家队员，他们自然是知道该怎么玩。
<br/>　　而且大家的水准都非常高，你见过哪个战队拥有王者组前20的选手的，这个恐怕除了当初的荣耀战队，就再也没有其他战队有那个水平了。
<br/>　　就算是现如今的白蓝两队都不行，比如白队这边萝卜和茄子他们两人就没在前20之列，而蓝队那边飞刀也没在这里面，他们的王者排名都在30到50名之间。
<br/>　　不出意外的排到的都是熟人，他们这边的每个位置都是各个战队的大将，就比如叶帝这边的上单，正是ST战队的matin。
<br/>　　双方飞快的ban人，各自将三个英雄ban掉后，便是飞快的选起了人，很快双方的阵容确定了下来。
<br/>　　叶帝他们这边选择了：上单武器、打野挖掘机、中路流浪、下路则是拿出了轮子妈和时光老头这对组合。
<br/>　　而对面则是选择了：上单鳄鱼、打野猪女、中路蛇女、下路则是女警和扇子妈这对组合。
<br/>　　看到双方的阵容，叶帝的眼睛顿时一亮，就是连叶帝身后观战的李景禹、秦霄几人的眼睛也都亮了起来。
<br/>　　“果然啊，现在高分段的上单选手因为香蕉的那一波节奏都开始选这种暴力型的上单了，反而让自家的打野选这种坦型英雄打野。”秦霄开口说道。
<br/>　　“像武器这种英雄，如果取得优势，那么到了中后期单带可就没人能够制得住他们了，而打野选择这种肉型打野，即使到了后期，他们这边也是有着能够充当前排的决色，以后的比赛真的会越来越有意思了。”李景禹跟着开口。
<br/>　　“是啊，相比于那些坦型上单发育到后期的比赛，大家肯定都喜欢这种暴力对决的比赛的，什么锐雯对战武器啊、或者是武器打鳄鱼，这都是非常值得期待的。”叶帝也是跟着笑了起来。
<br/>　　这样的比赛不光是观众愿意看到，就是连他这个选手也是非常乐于见到的，上单并不是被放弃的一个位置，如果总是玩肉，那么他们一战封神的机会可就要少了很多。
<br/>　　在这个赛场上能够秀起来的英雄，永远是那些暴力的输出型英雄。
<br/>　　“这一局小叶子他们这边的下路也很有意思啊，轮子妈和时光老头这对组合，这要是打到了后期，感觉小叶子真的是要飞起来了。”看着叶帝这边的下路组合，一旁的蒋念也是跟着开口说道。
<br/>　　片刻后，双方便是进入到了游戏界面。
