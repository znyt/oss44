        傅夜沉抱着童筱颖直冲三楼，一脚踹开了连华生的实验室大门，将正在做实验的连华生给吓了一跳。
<br/>
<br/>        “我去！阿沉，你吃错药了啊？”连华生额冒青筋地吼了声，他差点就把手里的试剂给洒了，这可是他最近两个月的心血啊！
<br/>
<br/>        当连华生吼完，再定睛一看时，只见傅夜沉怀里抱着的童筱颖，她的大腿上有一丝鲜血骨碌而下。
<br/>
<br/>        而童筱颖的脸色惨白，柳眉微蹙，双手握拳紧紧地护住了腹部。
<br/>
<br/>        傅夜沉的怀里紧紧地抱着童筱颖，大步走到连华生的面前，焦虑不安地说道：“筱颖有点不对劲，你快给她看看。”
<br/>
<br/>        “她来例假了，痛经，又犯困，我给她开点药就行了。”连华生一脸淡定地说。
<br/>
<br/>        傅夜沉顿时松了口气，但又心疼了起来。
<br/>
<br/>        他是不清楚女孩子这时候到底有多疼，所有这方面的知识仅源于初中时候的生理课。
<br/>
<br/>        可傅夜沉记得，上一次，在傅家大庄园里，筱颖因为穿不惯高跟鞋，而扭伤脚腕，脱臼了，也没有见她脸色有这么差过。
<br/>
<br/>        刚刚，她好像连说话都很艰难的样子。
<br/>
<br/>        连华生转身走到药柜前，打开玻璃柜门后，从里面取出了一个小药瓶。
<br/>
<br/>        他又去倒来了一杯温开水，拿着小药瓶朝傅夜沉走了过去。
<br/>
<br/>        傅夜沉见状，连忙在一旁的椅子上坐下，将童筱颖轻轻地放在自己的大腿上抱着。
<br/>
<br/>        连华生给童筱颖喂了药，忍俊不禁地说道：“去找刘婶给大小姐换衣服吧！”
<br/>
<br/>        童筱颖已经痛到无力气说话，更加顾不上自己此刻尴尬不已的心情。
<br/>
<br/>        本来连华生还想调侃傅夜沉把这弟妹真的是当心肝宝贝了，后来心里又觉得这种调侃只会让傅夜沉的脸色更加难看，索性就笑而不语了。
<br/>
<br/>        傅夜沉抱着童筱颖回了房间，并把刘婶叫了过来，吩咐刘婶好好照顾她后，才离开。
<br/>
<br/>        童筱颖面对刘婶都觉得尴尬，更何况刚刚还在两个大男人面前……
<br/>
<br/>        此刻的她，简直无地自容。
<br/>
<br/>        吃过药后，腹部的痛感得到了缓解，童筱颖自己换下的衣服和床单，本来想自己洗的，结果这活还是被刘婶给抢了去。
<br/>
<br/>        就连晚饭，都是刘婶端到她房间里来给她吃的。
<br/>
<br/>        翌日，童筱颖本想去上班，结果被大哥傅夜沉给强制留在家里休息。
<br/>
<br/>        苏漫雪从刘婶那儿得知，童筱颖又被傅夜沉接到私宅里去住了，于是不等傅夜沉主动去接她，她就自己收拾行李让司机从临海城的傅家大庄园赶到了滨江城这傅夜沉的私宅。
<br/>
<br/>        私宅里，傅夜沉和连华生今天都出去了。
<br/>
<br/>        童筱颖坐在餐桌前，正和刘婶一起吃童筱颖自己做的云耳红枣煲黄鳝。
<br/>
<br/>        刘婶一边吃一边赞不绝口地说道：“大小姐，你这汤味道很鲜。是怎么做的？”
<br/>
<br/>        “黄鳝两条、云耳二朵、红枣十颗、生姜二片。云耳、红枣分别洗净、稍浸泡并撕开、去核；黄鳝宰洗净，用热水烫去黏液，洗净、切段，一起与生姜放进瓦煲内，加入六碗清水，大火煲沸后，改为小火煲约一个小时，最后调入适量食盐便可。这汤养阴活血、调经补血。我十二岁第一次来例假的时候，我外婆教我做的。”童筱颖微笑着回答道。
<br/>
<br/>        刘婶笑道：“你外婆一定很疼你！居然教会了你如何用膳食来调理身体。”
<br/>
<br/>        “因为，我外婆说，等哪天她不在了，我也能自己把自己照顾好！”童筱颖微微一笑，只不过，昨天她倒是把自己来例假的日子给忘了。
<br/>
<br/>        这段时间，她每天都过得浑浑噩噩的。
<br/>
<br/>        就在童筱颖和刘婶聊得正起劲的时候，可视对讲电话响了。
<br/>
<br/>        刘婶起身去接的电话。
<br/>
<br/>        她接完电话，按了院门的开锁键，然后转身回到餐桌前，对童筱颖瘪着嘴说道：“大少奶奶也来了！”
<br/>
<br/>        “大嫂和大哥的感情真好！”童筱颖微笑着说。
<br/>
<br/>        大哥傅夜沉对苏漫雪的爱，已经好到，苏漫雪不在的时候，就拿她童筱颖当替身。
<br/>
<br/>        就因为她和苏漫雪现在长得一模一样，大哥傅夜沉才会有个时候傻傻分不清吧！
<br/>
<br/>        而殊不知，傅夜沉把所有的温柔，都给了她童筱颖一个人。
<br/>
<br/>        因为苏漫雪要进来，刘婶直接把家门敞开了。
<br/>
<br/>        童筱颖正劝刘婶再多喝点她煲的汤时，门口突然传来了苏漫雪的微微颤抖的声音。
<br/>
<br/>        “以、筱颖……救、救我啊！”
<br/>
<br/>        苏漫雪这一反常态的呼救声，让童筱颖和刘婶定格了手中的动作，彼此面面相觑。
