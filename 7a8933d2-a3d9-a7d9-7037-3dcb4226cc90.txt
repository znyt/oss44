　　浓郁的天地灵气涌入体内，萧逸沉浸在修炼当中。
<br/>　　入夜，萧逸感受到体内一阵力量爆发，他知道，这是突破的前兆。
<br/>　　一会儿后，他睁开了眼睛，“凡境六重了，好快的速度。”
<br/>　　萧逸也没想到，自己竟然这么就突破了，毕竟，前些天他才刚刚突破凡境五重。
<br/>　　凡境六重是一个分水岭，与五重的真气差距了五、六倍以上，突破所需的灵气与瞬间也大幅度增加。
<br/>　　但如今借着紫云洞内浓郁的灵气，竟在一天内就突破了，速度不可谓不快。
<br/>　　半晌，萧逸忽然再次睁开了眼睛，眉头微微皱了起来。
<br/>　　“五十步内的修炼速度尚且这么快，若是再往前走，速度岂不是更快？”萧逸心里暗暗想道。
<br/>　　“但再往前走，威压将超过凡境武者所能承受的地步。”萧逸皱了皱眉头。
<br/>　　若是其他人，肯定不会纠结这个问题。既然二长老已经严肃地嘱咐过了，超过五十步会有生命危险，就不会有人胆敢涉险。
<br/>　　但，萧逸从来都是个不肯安分，且胆大的人。
<br/>　　他站了起来，试探性地微微伸出了一脚。
<br/>　　左脚刚刚踏出五十步之外一丁点，顷刻间，一股庞大的压力笼罩了左脚，如负重石。
<br/>　　“还好，这个压力在我的承受范围内。”萧逸低声自语。
<br/>　　事实上，五十步距离，是家族长老多年来勘测出来的结果。这个距离，既能让家族子弟承受住威压，又能让子弟们安心修炼。
<br/>　　当超过这个范围，就算家族子弟能勉强承受威压，但那股负重力，已经很难让人入静，更别说安心修炼。
<br/>　　只是，萧逸心智过人，无论多恶劣的情况，多大的干扰，他都能稳住心神，并快速进入修炼状态，这是他远强于其他人的优点。
<br/>　　很多时候，武者的心智远比其他很多东西更重要。
<br/>　　同一时间，不远处的二长老猛地睁开了眼睛，紧紧地盯着萧逸。
<br/>　　其他家族子弟都在安心修炼，没有发现萧逸的异状；倒是二长老，他看似在修炼，实则一直在注视着各位子弟的情况。
<br/>　　“这小子想干嘛？”二长老皱起了眉头。
<br/>　　当看到萧逸在不断往前走时，他顿时急了，“臭小子，不要命了吗？”
<br/>　　他之前的警告并非开玩笑。
<br/>　　当萧逸走了大概五步后，灵气变得更加浓郁，可威压却比之前大了十倍都不止。
<br/>　　这里的威压，就算是后天境武者都会变得举步维艰。
<br/>　　萧逸也一样，心智归心智，身体承受不住也是枉然。
<br/>　　“噗，好大的威压。”萧逸嘴中竟溢出一丝鲜血。
<br/>　　“看来不能再前进了。”萧逸咬了咬牙。
<br/>　　然而，正当他有这个想法时，体内冰鸾剑竟再次一阵躁动，那股锋利的气息再次笼罩了全身。
<br/>　　这个气息，竟直接与威压抵消，让他的身体变得轻松无比。
<br/>　　萧逸笑了笑，知道这是体内冰鸾剑武魂的功劳，立马再次前进。
<br/>　　果然，在冰鸾剑那股锋利气息的帮助下，自己一连走了百多步，一直走到妖兽雕像面前也没有感到任何不适。
<br/>　　而这里，天地灵气的浓郁度达到了一个骇人的地步，比之前强了数十倍都不止。
<br/>　　萧逸没有浪费时间，立马盘膝坐下，开始了修炼。
<br/>　　另一边，二长老目瞪口呆地看着，暗暗咋舌。
<br/>　　原本他怕萧逸有危险，准备强行将萧逸揪出来。可看着萧逸竟一步步靠近妖兽雕像，而且样子轻松无比时，他便暂时打消了想法。
<br/>　　当最后看到萧逸竟直接在妖兽雕像前坐下来修炼后，他不禁倒吸一口凉气。
<br/>　　他很清楚那里的威压强到什么地步，在妖兽雕像面前，就连先天武者都无法站立身子。也只有他这个先天九重，才勉强能做到抵抗那股威压。
<br/>　　“这小子，到底怎么回事？难道威压对他没有影响？算了，只要他没有危险就好。”二长老看了萧逸一眼后，摇了摇头，没有再管。
<br/>　　......
<br/>　　所谓山中无岁月，修炼不知时。
<br/>　　转眼间，第九天已经来到。
<br/>　　在第二天时，萧逸的修为在毫无预兆的情况下，突破到凡境七重。
<br/>　　最靠近妖兽雕像的地方，修炼速度果然恐怖。
<br/>　　然而，武道修炼，越到后面，越是困难。
<br/>　　到了第五天，修为才堪堪突破到凡境八重。
<br/>　　如今第九天，修为突破到凡境九重，只差一步便是后天境。
<br/>　　“只剩最后一天了，抓紧修炼吧。”萧逸睁开眼睛休息了一会儿，打算再次入静修炼。
<br/>　　忽然，他转头看向了背后的萧若寒。
<br/>　　别的子弟，修炼时都是安安静静的，从不间断；他倒好，老是有各种小动作。
<br/>　　不过，正是他这种性格，却发现了萧若寒如今的情况。
<br/>　　“气息飙升，灵气吸入体内而修为不涨，好家伙，萧若寒这小子居然在凝结气丹。”萧逸冷笑一声。
<br/>　　不错，那边儿，萧若寒正在凝结气丹。
<br/>　　不过，他就没有萧逸那么快的修炼速度了。
<br/>　　他的位置，在五十步之内，距离妖兽雕像极远，灵气浓度也大大不如。
<br/>　　他进入凡境七重已经很久了，就算不在紫云洞修炼，也用不了多久时间就能突破到凡境八重。
<br/>　　刚进紫云洞当天，他就突破到凡境八重。
<br/>　　一直到今天，他才刚刚突破到凡境九重。
<br/>　　如今只剩一天时间，他当然赶紧借着紫云洞浓郁的灵气，赶紧凝结气丹。
<br/>　　另一边，萧逸冷笑一声，“想凝结气丹，突破后天，我偏不让你如愿。算计了我这么多次，也该轮到我来报答你了。”
<br/>　　说罢，萧逸毅然站了起来，往回走。
<br/>　　远处，二长老再次睁开了眼睛，看向萧逸，“这小子，又在搞什么鬼，修炼就修炼，咋总走来走去的。”
<br/>　　当看到萧逸走回之前五十步范围之内坐下修炼，二长老才无奈地叹了口气，再次闭上了眼睛。
<br/>　　萧逸坐下，从始至终，除了二长老外，其他子弟都没有发现他这些天跑到别的地方修炼。
<br/>　　“冰鸾剑，给我吸。”萧逸心中低喝一声。
<br/>　　顷刻间，周围的灵气被他快速吸收着。
<br/>　　而此时，原本正心存美好愿望，凝结气丹中的萧若寒，睁开了眼睛，并紧紧地皱起了眉头。
<br/>　　萧若寒满脸疑惑之色，“怎么回事，身旁的灵气怎么全部消失了。今天应该才第九天，紫云洞内的灵气不可能这么快被吸收完阿。”
<br/>　　疑惑之际，他转头看了眼后方的其他家族子弟，发现他们都在闭目修炼，顿时更加疑惑起来。
<br/>　　“怎么回事，他们都还在修炼，为何我却无法修炼了？”萧若寒心里疑惑到极点，同时凝结气丹过程中被打断，让他很是难受。
<br/>　　他怎么都想不到，在他前面，那唯一的背影，萧逸正暗暗窃笑。
<br/>　　间隔五步，是家族武者测算出来的距离，彼此不会影响到其他人修炼。
<br/>　　但，他们测算的标准只是橙阶武魂，顶多就是橙阶武魂。
<br/>　　而萧逸的冰鸾剑武魂，乃是最强的紫色，将能力开到极致，一下子就将萧若寒身边的灵气都给吸到了自己体内。
<br/>　　也就是说，其实萧若寒周围还是有灵气的，只不过这些灵气还未来得及被他吸收，已经率先被萧逸吸走了，造成了他以为灵气已经被吸收殆尽的错觉。
<br/>　　整整一天，萧若寒都在极度纠结中渡过。
<br/>　　而到了第十天，他的情绪几乎能用抓狂来形容。他和父亲的计划之一，就是他必须要在紫云洞中凝结气丹，突破到后天境。
<br/>　　若是他无法突破，将大大影响后续的计划。
<br/>　　“阿...该死，到底怎么回事！”萧若寒快要气疯了。
<br/>　　.....
<br/>　　直到最后一刻，二长老吩咐道，“紫云洞修炼结束，众子弟随我离开。”
<br/>　　这时，紫云洞内的灵气已经被十位子弟吸收殆尽。
<br/>　　紫云洞修炼名额，之所以限制十个，就是因为紫云洞内的灵气只够十个家族子弟吸收十天，并从中获得莫大收获。
<br/>　　走之前，萧逸明显感觉到，妖兽雕像正以缓慢的速度，不断地吸收着天地间的灵气，而后束缚在周边。
<br/>　　如果没有猜错的话，这个妖兽雕像拥有着类似‘聚类阵’的功效。
<br/>　　三年之后，这里将会再次变回灵气浓郁的地方。
<br/>　　当然了，三年之后，将是新的一代家族子弟前来修炼。
