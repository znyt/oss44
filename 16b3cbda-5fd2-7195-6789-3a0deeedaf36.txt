　　地魔界，遍地魔意，这些魔意，足以影响武者的灵识，就算是灵识远超常人的楚风眠，在这地魔界中，也觉察不到方向。
<br/>　　幸好如今楚风眠斩杀了雷绝，将雷绝的记忆全部读取，这才可以寻找到离开地魔界的路线。
<br/>　　几个时辰，楚风眠带着莽通三人，这才离开了地魔界。
<br/>　　等回到了北邙学院之中后，楚风眠才终于松了口气。
<br/>　　在地魔界中，楚风眠可是一直提心吊胆的，生怕突然出现一尊魔头，就将楚风眠灭杀了。
<br/>　　现在楚风眠的实力，虽然不错，但是一旦是遇到了地魔门中的强者，达到生死境高阶的强者的话。
<br/>　　这种生死境高阶的强者，想要灭杀楚风眠，并非难事。
<br/>　　“这地魔界中，虽然危机重重，但是其中肯定也蕴含无数奇物，等到我踏入神力境后，倒是可以再去闯一闯。”
<br/>　　楚风眠一边向着北邙学院飞了回去，一边心中暗自想到。
<br/>　　只要是楚风眠能够踏入神力境，便是面对这那些生死境高阶的强者，虽然无法战胜，却也足以自保。
<br/>　　他这一次前往地魔界，不单单是完成了北邙学院交代的任务，更是得到了一枚龙丹，这龙丹，至少也是属于一尊生死境的真龙的。
<br/>　　这一次楚风眠的收获巨大，龙丹可以助他突破，同时楚风眠还得到了万古不败真魔体的修炼方法。
<br/>　　这万古不败真魔体，对楚风眠来说无用，因为楚风眠身居战龙血脉，跟这万古不败真魔体，正是力量相冲，楚风眠无法修炼。
<br/>　　更何况以战龙决的强大，楚风眠再次突破，这战龙血脉的效果，不会比起万古不败真魔体会差，甚至是还要强大的多。
<br/>　　不过这万古不败真魔体，楚风眠传授给了这莽通三人，同时楚风眠还将一些记忆中的功法武技，传授给了他们。
<br/>　　在北邙学院，一人难敌四手，楚风眠便是要培养一些武者，来为他做事。
<br/>　　以楚风眠的速度，毫不休息的情况之下，大约五六个时辰，楚风眠便是来到了北邙学院的外围。
<br/>　　这个时候楚风眠才将速度缓缓的降了下去，也放开莽通三人，四个人缓缓的，回到了北邙学院之中去。
<br/>　　根据雷绝的记忆，楚风眠便是跟这莽通三人一起，来到了北邙学院的任务殿中去。
<br/>　　这北邙学院的一切任务，都是从任务殿中接取进行的，这一次宗门考核，完成了任务，也要回到任务殿去禀告。
<br/>　　一尊任务殿弟子，看着楚风眠四人步入其中，不由的问道。
<br/>　　“哦？你们居然回来了？完成了任务没有？没有完成任务，擅自回到学院中，可是重罪。”
<br/>　　楚风眠听着这任务殿弟子的话，也不惊讶，平静的说道。
<br/>　　“我们已经完成了，还请师兄去请长老出来。”
<br/>　　“好快的速度，你就是这一届的新人王吧，果然不错，有着几分实力。”
<br/>　　那任务殿弟子，微微有些惊讶，不过还是笑了笑道。
<br/>　　“长老就在里面，各位请随我来吧。”
<br/>　　说着，便是由他带路，将楚风眠一行人，带入到任务殿的一座房间之中去。
<br/>　　“长老，这几位新人弟子，完成了任务，还请长老前来检验。”
<br/>　　任务殿弟子，对着房间之中盘膝而坐的一尊老者恭敬的说道。
<br/>　　不一会，这老者便是缓缓的睁开眼，站了起来，他一眼扫过楚风眠一行人，猛然开口道。
<br/>　　“仅仅两日时间，就完成了学院交代的任务，不错。”
<br/>　　这任务殿长老，打量了楚风眠四人一眼，突然眉头一皱道。
<br/>　　“负责带领你们的内门弟子是谁？怎么不在？”
<br/>　　这任务殿长老的问话，令莽通三人的心中不由的有些紧张起来，雷绝的确是叛变的宗门，但是实际上却是被楚风眠杀死的。
<br/>　　楚风眠的实力，现在还不能暴露，如今他们三人也不知道，该如何解释。
<br/>　　“禀告长老，雷绝师兄，为了救我们，跟地魔门中的魔头同归于尽了。”
<br/>　　不过楚风眠早就料到了这任务殿长老的询问，他的脸上，露出了几分悲伤之痛，低声开口道。
<br/>　　“这一次若是没有雷绝师兄，我们四人，也都要死在地魔界了。”
<br/>　　“什么？雷绝陨落了？”
<br/>　　听到楚风眠的话，这任务殿长老的双眼，立刻瞪大，一脸不可置信。
<br/>　　一身灵力，猛然暴起，突然化为了无穷无尽的威压，向着楚风眠四人压迫而来。
<br/>　　“说！雷绝是不是你们害死的！”
<br/>　　“雷绝都陨落了！你们四人怎么可能活得下来！”
<br/>　　这任务殿长老开口一刻，便是蕴含无尽杀意，笼罩下来，这一股威压，令那莽通三人，都是有些喘不过气来。
<br/>　　眼前这尊任务殿长老，实力虽然不如苏凌长老等人，但是也是生死境三重的强者，比起那暗古魔尊都要强大一些。
<br/>　　他的威压，绝非是一些圣者可以抵抗的。
<br/>　　“禀告长老，雷绝师兄，的确是死于地魔门的魔头手中，而我们能够活下去，只是因为我的身上，有着一枚真龙一族的玉符，利用玉符的力量，我们才逃了出来。”
<br/>　　楚风眠在这威压之下，一字一字的开口道。
<br/>　　“请长老明察。”
<br/>　　“哦？你们以为巧舌如簧，就能骗得了老夫吗？”
<br/>　　“内门弟子，各个都是我们学院中的精英，谁死了，都要彻底追查！要是他的死跟你们有关，那你们也一样都得死！”
<br/>　　任务殿长老，眉头一皱，冷喝一声。
<br/>　　突然大手一挥，一股灵力，便是他的面前，化为了一道水镜，从这水镜中，突然显现出的便是地底洞窟中的场景。
<br/>　　“不好。”
<br/>　　莽通三人的眼神，皆是一变，要是重现曾经地底洞窟中的场景，那楚风眠的秘密，可就彻底暴露无遗了。
<br/>　　唯有楚风眠的脸色，平静无比，心中冷笑连连。
<br/>　　这水镜刚刚凝聚出的一刻，画面还未浮现完成，一声龙吟，突然响彻而起。
<br/>　　随后一道巨大无比的战龙之爪，突然对着水镜拍了过来，这水镜，顿时破碎。
