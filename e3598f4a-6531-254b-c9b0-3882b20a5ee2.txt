　　密室之中，苏莫盘膝而坐，面色古今无波。
<br/>　　从今以后，他的身份和行踪，都不再是秘密，必须更加的小心了。
<br/>　　好在，如今他有龙腾的庇护，在龙族是绝对的安全。
<br/>　　当然，龙帝此人，绝对不是善茬，也不可能帮他，他能动用的力量，或者是龙腾能动用的力量，都极其有限。
<br/>　　良久，苏莫叹了口气，目前，只能待在龙族了，让龙腾尽量拉拢更多的长老。
<br/>　　嗡嗡~~
<br/>　　这时，苏莫储物空间中的传讯符震动，是绫裳传来了消息。
<br/>　　苏莫取出传讯符，意念一动，便得知了对方传来的消息。
<br/>　　对方，早已经将天王宫派回去的弟子击杀，并且回到了云龙城，同时，对方还询问玉虚宗的事情。
<br/>　　随即，苏莫传讯给人德宫主，询问玉虚宗的情况。
<br/>　　“苏莫，老夫并没有进入玉虚宗，但是，老夫查探了一番，玉虚子从未与人交手，也很久没有离开玉虚宗了！”人德神王很快便传来了消息。
<br/>　　“果然有诈！”苏莫闻言，面色沉重如水，这应该是终极神王殿，或者是天王宫、地王宫为了找到绫裳，而设下的计谋。
<br/>　　只不过，具体是哪个势力的阴谋，就不得而知了。
<br/>　　“人德前辈，您老辛苦了，此事不用再查了！”苏莫向人德神王传讯道，现在，他的身份和行踪已经暴露，想必这阴谋也会自动消除了，只要绫裳不去玉虚宗便好。
<br/>　　随后，苏莫又将情况，告知了绫裳，让对方切勿前去玉虚宗，就在云龙城常住下来。
<br/>　　随后，苏莫再次开始了修炼，他要尽快达到四品的境界，离这一步距离已经不远了。
<br/>　　……
<br/>　　龙族的酒宴，举行了整整三日，三日之后，各大种族各大势力的强者，纷纷离开了龙都城。
<br/>　　随着各大势力离开龙都城，苏莫的情况，也在整个神域沸沸扬扬的传开了。
<br/>　　天下间还有第三具轮回之身，本名苏莫，化名秦陨、屠霄，已经加入了龙族，和龙族苍穹祖龙龙腾是至交好友。
<br/>　　苏莫的天赋无与伦比，丝毫不比帝一魂和拓跋青云差，修为三品高级，战力可以碾压四品高级的万古天骄。
<br/>　　这一系列的消息传出，让得神域震动，无数势力为之讶然。
<br/>　　而其后，关于苏莫和人王宫关系匪浅的消息，不知道是何人透漏而出，在神域不胫而走。
<br/>　　轮回神朝，魔王城。
<br/>　　魔王城坐落在轮回神朝的西部区域，靠近亘古魔国的地界，掌控轮回神朝方圆数百万里的地域。
<br/>　　魔王城建立在一片山峦之上，建筑延绵起伏，长达数百里。
<br/>　　魔王大殿之中，此刻，有三位老者端坐。
<br/>　　主座之上的老者，一身黑袍，体型高大，面色漆黑如墨，宽大的脸庞之上，两道漆黑的眉毛如同两柄利剑，给人一种极度的凶煞之气。
<br/>　　此人不是别人，正是魔王宫的宫主，修罗剑王。
<br/>　　下首，一左一右，端坐的两人同样是老者，左边的老者穿着一声古朴的灰袍，面容狭长，面色青紫，给人一种平易阴森之感，而右边的老者，体型极为雄壮，五大三粗，满脸络腮胡略显花白，给人一种霸道非凡之感
<br/>　　这两人，灰袍老者乃是鬼王宫宫主阿鼻鬼王，以及兽王宫宫主冰雪狮王。
<br/>　　“两位，想必你们也能猜到，今日邀请两位前来的目的。”修罗剑王目光环视阿鼻鬼王和冰雪狮王，声音低沉的问道。
<br/>　　“那苏莫的身份，已经确认无疑了，不知两位有何想法？”修罗剑王见两人沉默不语，再次开口道。
<br/>　　阿鼻鬼王沉默了片刻，道：“现在，天王宫支持帝一魂，地王宫支持拓跋青云，人王宫显然是要支持苏莫了！”
<br/>　　“不知人德哪来的底气，为何会帮助苏莫？”冰雪狮王疑惑的问道，虽然苏莫的天赋很高，但是，他完全看不到苏莫有最终胜利的希望。
<br/>　　“苏莫的天赋毋庸置疑，但是，根本不可能翻盘，除非龙族全力助他！”修罗剑王道，这是苏莫唯一能翻盘的可能，但是，也仅仅的是可能而已。
<br/>　　因为，无论是帝一魂还是拓跋青云，目前都有了高级神王的实力，基本上很难被杀死或者擒拿。
<br/>　　换言之，就算是吞虚龙帝亲自出手，也不一定能彻底镇杀帝一魂或者是拓跋青云，两人的实力虽然远不如吞虚龙帝这等强大的高级神王，但是逃命或者保命的本事还是有的。
<br/>　　而且，帝一魂和拓跋青云，还有天君河和地渊神王相助。
<br/>　　“根据这次祖龙大典的情况，吞虚龙帝不可能相助苏莫。”阿鼻鬼王声音低沉的说道，祖龙大典之日发生的情况，他们都一清二楚了。
<br/>　　以当时龙帝的态度来看，不可能帮助苏莫，苏莫想要借助龙族的力量翻盘，可能性不大。
<br/>　　“两位，老夫想问的是，我们还要静观其变吗？”修罗剑王眸中精光一闪，郑重的问道。
<br/>　　这些年，无论是天王宫还是地王宫，都在不停的拉拢他们，但是，他们一直没有表态。
<br/>　　“你的意思是，我们也开始插手？”冰雪狮王眸光一凝，紧紧的盯着修罗剑王。
<br/>　　“不错，上次帝一魂和拓跋青云交手，帝一魂略胜一筹，明显比拓跋青云潜力更大！”修罗剑王沉声道。
<br/>　　“你是想要帮助帝一魂？”阿鼻鬼王面色凝重了下来。
<br/>　　“略胜一筹也说明不了什么，你能确定拓跋青云不会先一步达到高级神王的地步？”冰雪狮王不置可否的说道。
<br/>　　修罗剑王闻言沉默不语，他是想要帮助帝一魂，但是，也不确定拓跋青云是否会下一步达到高级神王。
<br/>　　“二位，若是我们三人全部辅助帝一魂，便能以雷霆之势灭杀拓跋青云，到时候帝一魂融合拓跋之魂，一切都水到渠成。”修罗剑王眸光如电，紧紧的盯着阿鼻鬼王和冰雪狮王。
<br/>　　若是两人同意，那么，一切都能尘埃落定，就算融合了拓跋青云之魂，帝一魂不能立刻晋升高级神王，也相距不远了。
<br/>　　最主要的是，拓跋一死，帝一魂众望所归，苏莫就是粘板上的鱼肉。
<br/>　　“老夫和地渊私交不出，出手杀他爱子，这未免……！”冰雪狮王有些拿不定主意，他知晓修罗剑王此言在理。
<br/>　　只要帝一魂融合了拓跋青云，那么，再融合苏莫问题不大，一切都能尘埃落定。
<br/>　　“的确是这样！”阿鼻鬼王叹了口气，他们六大王宫，在轮回神朝建立之初就存在了。
<br/>　　他们六大王宫之主，也俱是当年跟随天尊之人，六人之间，可谓是在一起共事了无数的岁月。
<br/>　　现如今，让他们出手击杀地渊神王之子拓跋青云，还真有些拿不定主意。
<br/>　　事实上，他们早已看好帝一魂，帝一魂在各方面都强于拓跋青云，之所以他们一直没有表态，也与拓跋青云是地渊之子有关。
<br/>　　“两位，到了如今的地步，你们居然还在乎和地渊的私交？”
<br/>　　修罗剑王面色冷漠，声音低沉的说道：“只要帝一魂融合了拓跋青云和苏莫之魂，天尊大人就会复生，说不定真的能更近一步，到时候天尊会带着我等，横渡永恒神尊构建的时空通道，离开荒界，从而超脱纪元，永生不死。”
<br/>　　听闻修罗剑王之言，阿鼻鬼王和冰雪狮王陷入了沉默之中，在思考此事的利弊。
