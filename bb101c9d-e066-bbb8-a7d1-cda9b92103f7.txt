　　第430章
<br/>　　经历过死亡的恐惧，最终迎来胜利，如此放肆一下也是应当。
<br/>　　但山寨里有人醉得一塌糊涂，可以需要人去防备来犯的敌人。程大雷独自离开街道，走到城头检查城防。
<br/>　　城头有卫兵巡逻，时刻防备来犯的敌人，见到程大雷出现，他们立刻挺直腰杆，握紧手中的长矛。
<br/>　　“大当家！”
<br/>　　程大雷点点头，替对方将帽子扶正，如今还是冬天，立在城头上要抵抗一阵阵朔风。这自然不是什么好差事，幸运的是，每个人都保持着极高的警惕，没有因为艰苦便怠慢。
<br/>　　“今天是庆功会，你们辛苦些，等换了岗，也去喝一杯。”程大雷。
<br/>　　“是！”一支小队十个人，同时挺起胸膛。
<br/>　　其实程大雷也不知道他们叫什么名字，不知是自己召唤出来的，还是迁徙此地的流民子弟，但终究不太重要，既然来到这里，便是自家兄弟，程大雷现在也要为他们负责。
<br/>　　“大当家，咱们什么时候再打仗？”
<br/>　　“你喜欢……打仗？”程大雷表情有些困惑。
<br/>　　“嗯，咱们什么时候打到长安去，也给那些官老爷点颜色看看。”卫兵兴奋道。
<br/>　　程大雷无奈的摇摇头，问道：“你今年多大了？”
<br/>　　“禀报大当家，十七。”
<br/>　　“才十七岁呐。”程大雷叹了一声，用手拍了拍他的肩膀：“好好干活，有你出力的时候。”
<br/>　　“是。”
<br/>　　程大雷独自走下了城头，没有回城，而是走到城外的荒原上。是冬春相交的时候，一眼望不到头的荒原，虽有草木在黑土地里冒出新绿，可天气依旧冷得瘆人。
<br/>　　城外，有一座蛤蟆城的公墓。
<br/>　　战死、老死、病死、受伤而死的兄弟火化后，骨灰葬在这里，地面上竖着墓碑，黑色花岗岩上刻着他们的籍贯、姓名。许多人的尸体根本无法找到，也就是埋几件他们生前的衣服。只要和兄弟们葬在一起，便算是回家了。
<br/>　　这次蛤蟆城又打了胜仗，同样，墓地里又竖起了新的墓碑。为了稳定军心，程大雷从没有过分渲染这件事，但有时间总也会过来看看。
<br/>　　荒原上，是一眼望不到边的黑色碑林，上面刻着的一个个名字，就代表一个或年轻或苍老的生命。和三年前相比，蛤蟆城的确越来越强大，与之相随的，则是竖起的黑碑越来越多。
<br/>　　程大雷在碑林遇到了刘悲，他裹着羊皮大袄，一个人默默饮酒。程大雷走过去的时候，他也没有吭声，只是把酒葫芦递了过来。
<br/>　　程大雷啜了一口，又递还给他。二人并肩坐了一会儿，刘悲打了个哈欠，欲言又止。
<br/>　　“大当家……”
<br/>　　一个声音打破了碑林的沉默，和珅快步过来，来到程大雷身边。
<br/>　　有人在庆功宴上喝得酩酊大醉，但同样有人时刻保持着清醒。刘悲、程大雷、和珅都是这种人。
<br/>　　“我一猜大当家就在这里，所以特意找过来了。”和珅道。
<br/>　　“呃……有什么事么？”程大雷有些心不在焉。
<br/>　　“大当家，我已经派人清点过战利品的数目，这次咱们可以说大获丰收。还有另外一件事，我一定要向大当家汇报。眼前，可是难得的机会……”
<br/>　　“什么机会？”
<br/>　　和珅来得很着急，明显此刻才喘匀了呼吸。
<br/>　　“大当家，宋伯康提了十万人去长安，如今凉州城留守的兵力不过五万，而经过这一战，被我们消灭了不少。如今，凉州城防守空虚，我们若趁机进攻，定能一举将凉州城拿下。之后，肃清凉州其他势力，整个凉州便是我们的天下。”
<br/>　　啧……
<br/>　　程大雷倒抽一口凉气，如果不是和珅提醒，他倒是忘了这件事。
<br/>　　天下大乱，朝廷这两个字已经是名存实亡，各个势力间斗争不止，现在哪有什么王法朝廷，谁手下兵多，谁就是王法。
<br/>　　现在程大雷手上的兵不少，而且经此一战，战斗力也是不弱。假若想要拿下凉州城，眼前的确是最好的机会。
<br/>　　“凉州城是座大城，想要打下来怕是不太容易。而且，其他城池的人，未必不会驰援，到时候我们腹背受敌，可就陷入了险地？”程大雷。
<br/>　　“这件事大当家不用担心，具体情况我们已经打听过。凉州共有七家城池，与戎族接壤地带有五座关，当然，原先的琴川关也是其中之一。这十二家势力，按道理来说，都是由凉州牧管辖。但事实不是如此，其中还有将军府的人。我们先派人潜入凉州城，有白元飞在，这不是太大的问题。然后发起强攻，短时间解决战斗，其他城池未必会反应过来。”
<br/>　　和珅说得很详细，看来他之前已做过不少准备，和珅的野心不小，很早就开始打整个凉州的主意。
<br/>　　但问题是，程大雷没有这样的野心，他还是第一次开始考虑这件事。和珅的办法未必不可行，当然需要冒一些险，但风险越大，同时收获也是越大。
<br/>　　不过，程大雷一时间还没有拿定主意。
<br/>　　“大当家……”和珅唤了一声。
<br/>　　恍然失神的程大雷这才反应过来，沉思片刻道：“这件事宜缓不宜急，一旦打下凉州城，凉州的势力平衡必然被打破。到时候，一个混乱的凉州未必是我们愿意看到的。难民、其他势力的反应都要考虑到，这件事要从长计议，从长计议。”
<br/>　　和珅读懂了程大雷的犹豫，他沉默片刻，道：“事到如今，大当家究竟想要什么，是不是该告诉我们了，我们也好有条路走。”
<br/>　　程大雷一怔，他自然听懂了和珅的话。事实上，在一开始和珅就拿这个问题问过程大雷。
<br/>　　乱世已在眼前，程大雷是要逐鹿天下，或是择主而事……他有了方向，手下人才有方向。如今的蛤蟆城已经可以吃饱饭，但吃饱饭后，就开始想些有的没的，封王拜相，开疆扩土……
<br/>　　问题是，程大雷自己也没有想好。他没有兼济天下的情怀，想的是独善其身。
<br/>　　可乱世本就是个零和游戏，要么，你拿走全部，要么，你什么都没有。
<br/>(本章完)
