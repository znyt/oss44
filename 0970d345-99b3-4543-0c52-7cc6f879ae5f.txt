　　萧子木一棍轰来，萧逸明显从中感觉到威胁，当即脚步一动，躲开了这一棍。
<br/>　　火旋棍轰到地面，当即‘轰’的一声，将比武台砸出一个大洞。
<br/>　　萧逸瞳孔一缩，暗暗惊讶，“好强，凡境六重与五重，果然有着极大的差别。”
<br/>　　在凡境中，六重是一个分水岭。
<br/>　　六重与五重虽然只差一重，但彼此体内的真气却足足相差了五、六倍都不止。
<br/>　　越是浓郁的真气，就能将武魂发挥出越大的威力。
<br/>　　像萧子木这一棍，足以将箫星扬一棍重伤。
<br/>　　虽然二者都是橙阶武魂，但因为真气的差距，就造成了萧子木这一棍的威力，比箫星扬的战力强了五、六倍以上。
<br/>　　比武台下方，箫星扬也是暗暗惊骇，“真不愧是家族排名前五的子弟，若是我对手这萧子木，恐怕单单这一棍就能将我秒杀。”
<br/>　　比武台上方，萧子木冷笑一声，道，“萧逸，知道我的厉害了吧，现在求饶还来得及。”
<br/>　　“求饶？”萧逸嗤笑一声，随即握了握拳头，认真道，“看来，要动真格了。”
<br/>　　动真格？
<br/>　　萧子木愣了愣。
<br/>　　其他族人也是有些反应不过来，甚至认为萧逸太过狂妄了。
<br/>　　“凡境五重和六重的差距太大了，萧逸很可能连几招都接不下，还谈什么动真格呢。”
<br/>　　“在一个凡境六重面前谈动真格，这跟自取其辱有什么区别。”
<br/>　　族人们纷纷摇头，仿佛已经看到萧逸落败的下场。
<br/>　　萧子木更直接，再次发动火旋棍，打算用行动来证明自己的强大。
<br/>　　“萧逸，你败了。”萧子木大喝一声，一棍攻出。
<br/>　　萧逸冷笑一声，怡然不惧，身子陡然间化为了一阵虚影。
<br/>　　而在这需要当中，一股火焰笼罩全身。
<br/>　　萧子木的火旋棍打出，轻松被这道虚影躲开。
<br/>　　“怎么回事？”萧子木一惊，不信邪地将火旋棍再次快速打出，化作一窜连绵不断的棍影。
<br/>　　然而，棍影虽快，却始终打不中虚影。
<br/>　　“怎么回事，那到底是什么东西？奇怪的身法吗？”族人们瞪大了眼睛，满脸惊讶。
<br/>　　在他们眼中，萧逸仿佛变成了一条灵动的，浑身冒着火焰的火蛇。
<br/>　　“是我眼花了吗？”有些族人更是使劲地揉了揉自己的眼睛，试图看清那道虚影。
<br/>　　只不过，无论他们怎么看，始终看不明白。
<br/>　　不错，萧逸施展了形意五绝之一的蛇形。以形意五绝的精妙，岂是外人能随便看透的。
<br/>　　而且，萧逸还附加了他的武魂力量，控火兽的火焰。
<br/>　　这是他的底牌之一，也是这半个月来幸苦修炼领悟出来的招式。以形意五绝，配合武魂力量。
<br/>　　火焰本身就具有爆发性，且控火兽本身就精通操控火焰，在这爆发性的加持下，他的蛇形变得更加千转百柔，更加灵动，更加让人捉摸不透，虚无缥缈。
<br/>　　这时，萧子木不断攻击，却始终打不中萧逸，让他极为着急。明明自己的境界更强，却连萧逸的身体都碰不到，这种感觉让他抓狂。
<br/>　　紧接着，他这般快速攻击，体力和真气都消耗得极快，让他满头大汗。
<br/>　　“阿...该死，到底怎么回事？”萧子木咬了咬牙。
<br/>　　连执事，乃至长老们都看不透的形意五绝，他就更加别想看透了。
<br/>　　“怎么，累了吗？”这时，萧逸冷笑一声，“累了的话，轮到我了。”
<br/>　　话音刚落，萧逸的脚步动了，化为一窜幻影消失在原地。在幻影之中，还冒着熊熊火焰。
<br/>　　嗖的一声，萧子木根本反应不过来，甚至连萧逸是怎么消失地都不知道。
<br/>　　“好快的速度。”族人们惊呼一声。
<br/>　　而且，更让他们惊讶的是，在他们眼中，萧逸竟化作了一条浑身是火的猎豹。
<br/>　　猎豹奔袭，速度奇快。
<br/>　　“该你尝尝我的攻击了。”萧逸低喝一声，忽然出现在萧子木背后，一掌打出。
<br/>　　萧逸的掌中，竟虎虎生风，仿佛出现了一声虎啸山林之音，‘吼’！
<br/>　　“这...”族人们惊骇至极，在他们眼中，萧逸再次变了，变为了一头猛虎。
<br/>　　猛虎下山，势大力沉。
<br/>　　一掌出，如震啸山林，地动山摇。
<br/>　　“轰”的一声，萧子木连反应的机会都没有，直接被萧逸一掌震飞。
<br/>　　结合了武魂力量的形意五绝，威力强悍如斯。
<br/>　　‘噗’，萧子木口吐鲜血，直飞台下。
<br/>　　当他跌落地面时，已经彻底晕厥过去，背后衣衫尽碎。
<br/>　　眼尖的族人们明显可以看到，在他的后背上，正有一个醒目的，被火烧严重灼烧的掌印。
<br/>　　“排名前五的萧子木，竟然败了？”
<br/>　　“而且萧逸仅仅出了两招，这是完胜吧？”
<br/>　　族人们的表情已经不能用惊讶来形容了，而是震惊，震惊到极点那种。
<br/>　　角落边缘，三长老萧重流下了一滴老泪，“好，很好，逸儿，你终于长大了。”
<br/>　　当初他只是受家主支托照顾萧逸，可是家主失踪多年未回，萧逸是他一手养大的，他早已将萧逸视如己出。
<br/>　　这些年来，他一边要忙着家族里的大小事务，一般要应付五长老等人的暗中为难，一边又对萧逸百般忧心，要为萧逸的长大后的日子操劳。
<br/>　　偏偏以前的萧逸是个性格窝囊，又觉醒了废武魂的人。
<br/>　　谁能知道，他这些年来承受着多少的压力，为萧逸操碎了多少的心。
<br/>　　现在见萧逸竟然完胜家族最出色的子弟，他的心情根本无法言喻。
<br/>　　狂喜？欣慰？自豪？满足？他不知道，他只来得及无法抑制地留下了一滴老泪。
<br/>　　另一边，五长老‘咔’的一声，将手中的茶杯一把捏碎，脸上尽是阴寒和愤怒。
<br/>　　萧若寒也是脸色如霜，“小废物何时变得如此厉害了？该死！”
<br/>　　.....
<br/>　　比武台上，萧逸负手而立，淡淡地看向其他子弟，道，“还有谁要上台和我切磋一番？”
<br/>　　被他那淡然目光扫视过的子弟，纷纷转过了头，不敢直视。
<br/>　　哪怕是排名靠前的凡境六重子弟，也哑口无言。他们很清楚，他们绝对做不到完胜萧子木，而萧逸做到了，证明萧逸的实力绝对在他们之上。
<br/>　　若还上台和萧逸切磋，只会是自取其辱。
<br/>　　裁判看了一眼，见没人应答，高声宣布道，“此战，少家主，萧逸胜。”
<br/>　　裁判竟连称谓都改了，主动称呼一声少家主。
<br/>　　要知道，在萧家已经很多年没有人愿意以这样的尊称叫萧逸了。
<br/>　　这个世界就是这样，以武为尊，萧逸表现出了足够的实力，足够的天赋，便会受人尊重。
<br/>　　“萧逸，你可以下去了。”裁判和颜悦色地说道。
<br/>　　萧逸摇摇头，道，“不急。”
<br/>　　而后，萧逸看向不远处的长老席，那里是长老和执事们观看比武的地方。
<br/>　　萧若寒也坐在那里，在五长老的下方。
<br/>　　“萧若寒。”萧逸傲然说道，“还记得我们的打赌吧，你输了！”
<br/>　　输了，萧若寒的打赌输了！在这大庭广众之下，所有族人都在的情况下，这个年轻一辈第一人，第一次听到‘输’这个字。
