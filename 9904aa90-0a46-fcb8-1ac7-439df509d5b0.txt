<br/>    天庭和瑶池，可以容忍孽龙星河之中有着众多纷杂的势力，也不管这些势力互相争斗，厮杀得天下大乱，也不管这些星辰上的资源如何分配，他们只关心一点，那就是鬼门。
<br/>    
<br/>        鬼门之中灵魂，才是真正能够影响到他们根基的东西。
<br/>    
<br/>        因为这些灵魂，都是用来供养英灵树的。
<br/>    
<br/>        唯有英灵树，才是一个大势力能够延续的保障，这些灵魂，绝对不能出任何事。
<br/>    
<br/>        人寿有时尽，人终究会有一死，而灵魂的增长，是没有限制的，所以，这些灵魂，必须要消除，否则，便是一场鬼祸。
<br/>    
<br/>        他们接过了天道的权柄，执掌星空，便是要替天道做事，收割这些灵魂，以免鬼祸降临。
<br/>    
<br/>        而英灵树，便会因为他们的所作所为，给予馈赠。
<br/>    
<br/>        亘古以来，都是如此。
<br/>    
<br/>        鬼门即将开启，天庭和瑶池都在蠢蠢欲动，而就在这时，瑶池进驻鬼门星的所有人，几乎在一夜之间被斩杀殆尽。
<br/>    
<br/>        整个星空，一时之间风声鹤唳。
<br/>    
<br/>        瑶池震怒。
<br/>    
<br/>        玉龙河府大殿之中，一座高塔之下，匍匐着众多的天仙，“禀报将军，我们在鬼门星上的人，一夜之间都被斩杀殆尽，根据调查的探子回来禀报，似乎是抢劫，将所有的财物都抢走了。”
<br/>    
<br/>        轰！
<br/>    
<br/>        整座高塔颤动，从中传出一个老迈的声音，“放屁，你等，可曾见过抢劫杀光所有人，还毁尸灭迹的！”
<br/>    
<br/>        天仙们不发一言，他们知道，是将军大人怒了。
<br/>    
<br/>        高塔之中传出一个愤怒的声音，“十年，还有十年才是决战的时刻，但是单清风，他已经忍不住了吗？”
<br/>    
<br/>        有天仙疑惑道：“大人，这是天庭动的手吗？”
<br/>    
<br/>        高塔之中继续传出气急败坏的声音，“蠢货，如今鬼门将要开启，天庭和瑶池之间的战争一触即发，能够在一夜之间杀尽瑶池的人，有这个实力，还有这个胆量的，除了天庭，难道还能是已经消失了三百年的无面者不成？”
<br/>    
<br/>        “这件事，单清风，必须给我一个交代！”
<br/>    
<br/>        瑶池的质问很快传到了天庭，单清风现在正烦着追查他兄长的身死，哪里会管那么多，“瑶池自家出事了，
<br/>    
<br/>        问本尊作甚？”
<br/>    
<br/>        这就是单清风的态度，众多天仙齐聚赤龙星河河府，拜见新来的将军。
<br/>    
<br/>        金碧辉煌的大殿之中，一团金光如同烈日一般落下，落在大殿唯一的大座之上，下面，跪着一地的天仙，“将军，瑶池驻扎鬼门星的人，一夜之间被斩杀殆尽了。瑶池那边，让我们给他一个交代。”
<br/>    
<br/>        金光之中传来一阵笑声，看热闹不嫌事儿大，他知道跟自己没关系，但是这并不妨碍他奚落瑶池一番，就算没这个事，还不是要打。“哈哈哈哈，告诉他们，连劫匪都没办法，他丁况，没有资格问本尊！”
<br/>    
<br/>        “哦，对了，顺便告诉他，听说他被抢了，我决定送上一枚灵石，以表安慰！”
<br/>    
<br/>        “说实话，虽然不知道是谁干的，但是，干得漂亮！”
<br/>    
<br/>        瑶池，玉龙星河河府高塔之下，一枚灵石被送了过来，一位天仙颤巍巍地举着这枚灵石，禀报道：“禀报将军，天庭将军说我们连劫匪都抓不住，没有资格问他，我们瑶池损失惨重，他送来一枚灵石，以表安慰。”
<br/>    
<br/>        轰！
<br/>    
<br/>        尘封三十年的大门在这一刻突然破碎，木屑横飞之中，一个白眉白发的老人拄着一根桃木杖走出，“单清风，老朽闭关了三十年，今日，老朽出来了，来人，摆驾鬼门星！”
<br/>    
<br/>        有天仙道：“天庭正在接管我们瑶池的控制范围，现在只怕不好过去啊！”
<br/>    
<br/>        丁况怒道：“滚开，老朽亲自前去，倒要看看，谁能挡我！”
<br/>    
<br/>        玉龙星河府，城门大开，一辆白玉雕琢而成的马车驶出城门，在后面，跟着三十个天仙，那是丁况的亲兵。
<br/>    
<br/>        大道之旁，一座酒楼之中，有着一个戴斗笠的修行者，他摊开面纱，正是步留痕，“丁况老儿，已经离开了吗？”
<br/>    
<br/>        旁边一个天仙问道：“大人，我们何时动手？”
<br/>    
<br/>        步留痕问道：“东西到手了吗？”
<br/>    
<br/>        天仙恭敬道：“已经到手了。”
<br/>    
<br/>        步留痕道：“走，回房间，这酒，暂时不吃了，我要看看你们取来的东西。”
<br/>    
<br/>        来到房间之中，步留痕道：“将东西取出来。”
<br/>    
<br/>        天仙恭敬地将背后的木匣子取下，轻轻地放在木桌之上。
<br/>    
<br/>        而后，两人捧来一个香炉，恭敬地点上两炷香，而后道：“我等非是冒犯神兵，只是想要一睹神兵风采，英魂莫怪！”
<br/>    
<br/>        两人焚香完毕，这才打开盒子，其中一柄血红色的重弓沉睡其中，在旁边，还有着十三根狼牙箭。
<br/>    
<br/>        步留痕感受了一番这弓的威势，他虽然从来没有这么近距离看过英魂兵器，但是他能感受到这柄弓的暴戾。
<br/>    
<br/>        那是他从来没有感受到过的威势。
<br/>    
<br/>        步留痕微微点头道：“没错了，真不知道，我这一次要去救的，是什么人，能够拿起这柄弓的人吗？”
<br/>    
<br/>        关上盒子，又是弯腰行了一礼，道了一声，“得罪了！”
<br/>    
<br/>        天仙这时才问道：“执刀人大人，我们现在动手吗？”
<br/>    
<br/>        步留痕道：“我去请示一下那位神秘的存在。”
<br/>    
<br/>        天仙恭敬道：“是的是的，应该请示的。”
<br/>    
<br/>        步留痕闭上双眼，在心中道：“尊敬的无面者，您的执刀人在此刻呼唤您.......”
<br/>    
<br/>        陈明正在关注着事态发展，抬头看了一眼星空，像是天庭和瑶池这般大势力坐镇一方的将军，身上都是有着气运的，在天象之中，陈明自然是能够看的到。
<br/>    
<br/>        “丁况这老东西，莫非没有上当不成，居然还没有动身？”
<br/>    
<br/>        正在这时，步留痕忽然在呼唤他，呼唤他的媒介便是无面者面具，“步留痕在这个时候呼唤我做什么？去看看吧。”
<br/>    
<br/>        陈明回房，戴上无面者面具，出现在大殿之中，步留痕随之出现，陈明道：“怎么回事？”
<br/>    
<br/>        步留痕道：“丁况乘坐着他的马车，已经带着他的亲兵离开了玉龙星河河府，我们现在动手吗？”
<br/>    
<br/>        陈明心中冷笑，这才明白了过来，原来这个老东西还有着一手防备啊，若是真的以为他走了，这个时候出来营救老杀才，一定是被他一举镇压。毕竟是天庭镇守一方的将军，还真是老谋深算啊。
<br/>    
<br/>        陈明微微一笑道：“没有我的命令，不能动手，他还没有离开。”
<br/>    
<br/>        步留痕道：“可是属下亲眼看着丁况的车架还有他的亲卫出城啊！”
<br/>    
<br/>        陈明道：“过几日，过几日你就明白了，去吧，没有我的命令，不可轻举妄动。”
