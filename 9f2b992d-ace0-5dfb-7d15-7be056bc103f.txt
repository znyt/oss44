　　紧追着留在地面上的越野摩托轮胎印，对从矿洞中逃跑的布克尔特进行着追击。
<br/>　　两辆越野摩托的轮胎印距离矿洞出口一百多米的位置却是分了开来，分成了两路。
<br/>　　罗昊弯腰伸手在留在地面上的轮胎印触摸了两下，说道：“左侧的轮胎印相较于右侧的较深，佟千雪应该是被往左侧的那辆越野摩托带走了。”
<br/>　　“右侧那辆车交给我，你们负责去追左侧那辆车。”齐凡对罗昊说道。
<br/>　　“注意安全。”
<br/>　　罗昊没有把太多人的分出去交给齐凡去追击往右侧逃跑的那辆越野摩托，他们的主要目标是营救佟千雪！
<br/>　　齐凡紧追着地面上的轮胎印，追击着向右侧方向逃跑的诺林顿，追出去了差不多有两三百米的距离，齐凡看到了那辆倒在地上的越野摩托。快步走到越野摩托旁边，伸手在摩托车的排气管上摸了一下，还带着温度，这辆越野摩托放在这里的时间还很短，摩托车的主人应该还在这附近。
<br/>　　齐凡把挂在身上的紧凑型HK416自动步枪摘了下来，放在地上，右手缓缓拔出斜固定在战术背心上的龙牙军刀……
<br/>　　就在此时！
<br/>　　一道漆黑的刀芒从齐凡的背后斜劈而来，直逼他的后背。
<br/>　　齐凡敏捷的做出一个闪躲，身体在地上一滚，顺势拉开跟偷袭自己的诺林顿之间的距离，右手龙牙军刀反握，做出防御的姿态。
<br/>　　铿！
<br/>　　诺林顿这一刀砍在了越野摩托的油箱上，越野摩托的油箱被砍出了一道口子，油箱中的柴油立刻流了出来。
<br/>　　诺林顿手中握着两把疯狗战术突击刀，上前一步，踩在从摩托车油箱中流出来的柴油中，耸动着肩膀，显得跃跃欲试。
<br/>　　齐凡注意到诺林顿握在手中的那两把疯狗战术突击刀，眼睛一凝，因为那不是使用高碳钢所打造出来的疯狗战术突击刀，诺林顿那两把疯狗战术突击刀是用黑曜石打造而成！
<br/>　　黑曜石砍刀，最早是由阿兹特克人所发明。阿兹特克人崇拜美洲虎，因而他们把最精锐的武士叫做“美洲虎武士”，勇士自然要陪宝刀，但遗憾的是，当时的阿兹特克人还不懂得冶铁，所以他们就为自己的勇士们打造了另外一种可怕的武器——“马夸威特”（Macuahuitl）。
<br/>　　马夸威特就是世界上最早的黑曜石砍刀，黑曜石是一种火山岩，它的材料硬度很高，而且十分锋利，但是在17世纪时，随着西班牙人的入侵和阿兹特克帝国崩溃，“马夸威特”随着它的主任们一度堙没在历史尘埃中。
<br/>　　黑曜石刀再次出现，是一位眼外科医生发现用钢制的手术刀在给病人做完手术，出现很严重的感染情况，那名医生在尝试使用黑曜石作为材料制作了手术刀后，发现手术后的感染情况明显减少，黑曜石用作手术刀刀刃材料也随之被得到广泛应用。
<br/>　　黑曜石不管是材料硬度和经过打造后的锋利程度不输于高碳钢，但是其重量要比高碳钢重，所以黑曜石基本没有被用作打造军刀的材料。
<br/>　　可现在站在齐凡面前的诺林顿就拿着两把使用黑曜石所打造出来的
<br/>　　诺林顿在“绿色贝雷帽”特种部队服役时的行动代号叫“螳螂”，因为擅长使用双刀进行近战，所以又被叫做“双刀螳螂”。诺林顿能使用黑曜石疯狗刀作为自己的近战武器，主要还是依赖于他双臂强大的力量，在阿富国执行任务的时候，他在双刀折断损毁的情况下，用手臂抱住一个“圣战”组织成员，硬生生挤断了对方的肋骨，让断裂的肋骨刺入对方肺部，杀死对方。
<br/>　　“中国小子，来啊。”
<br/>　　诺林顿在言语上对齐凡进行着挑衅。尽管相较于当年第一次参加委内瑞拉“猎人学校”时有了翻天覆地的变化，可是固有的观念，还是让西方很多军事强国的特种部队对中方特种部队的轻视，认为他们还是上不得台面的“童子军”。
<br/>　　没有理会诺林顿语言上的挑衅，齐凡试图寻找出诺林顿在防御上的漏洞，然后发动攻击。
<br/>　　唰！唰！
<br/>　　诺林顿双手的黑曜石疯狗刀挥舞了两下，右脚在地面上的狠狠一踏，留下一个足有半寸深的脚印，朝着齐凡进攻而来。
<br/>　　诺林顿身高足有一百九十公分，体格也十分健壮，可是这并不足以影响他的速度，他就像是一辆踩足了油门了的土方车，朝着齐凡冲撞而来。
<br/>　　齐凡没有选择跟诺林顿硬碰硬，快速向后拉开跟诺林顿之间的距离。
<br/>　　唰！
<br/>　　诺林顿左手的疯狗刀对着齐凡横切而来，齐凡翻身滚地躲开，右手龙牙军刀挥出架住诺林顿右手对着劈砍而来的疯狗刀，右脚在地上一个蹬踢，向后滑出一段距离。
<br/>　　诺林顿左手疯狗刀切空，砍在一棵树干上，把树干砍断了一半，从树干中把疯狗刀收回，看了眼已经接连两次躲开自己进攻齐凡，心中很是恼怒，冲着齐凡怒吼道：“胆小鬼！！”
<br/>　　仅仅只是跟诺林顿短暂的接触，齐凡已经感受到了诺林顿手臂力量的强大，刚才的挥刀格挡，强大的力量震得齐凡手心发麻，手中的龙牙军刀也是差点脱手而出。
<br/>　　诺林顿左手挽了个刀花，再度对着齐凡发起攻势；而齐凡在刚才闪躲的时候，左手已经扣住一枚手雷，咬住保险销上的圆环，拉住了保险销，甩手朝着诺林顿的脚边扔去。
<br/>　　“Fuck！”
<br/>　　看到落在自己脚边的手雷，诺林顿大惊失色，迅速朝着旁边闪躲开来，可还是被手雷的爆炸冲击波给震飞出去撞在那棵刚才被他砍断了一半的树干上，把树干撞断。
<br/>　　齐凡在丢出手雷的同时迅速朝着旁边的树干翻滚过去，龙牙军刀交到左手，右手拔出手枪，等到手雷爆炸后，从树干背后闪身而出，对着还没从地上爬起来的诺林顿连续射击。
<br/>　　尽管诺林顿身上的防弹衣为他抵挡住了一些子弹，可还是有一颗子弹击中了他的眉心。
