　　第194章
<br/>　　下雨的这些日子，程大雷也没有闲着。他每天所做的，也不只是站在城头跳广场舞。
<br/>　　程大雷在城头吸引敌人主意，暗地里悄悄将山寨的兄弟送到外面。山道狭窄，又因为下雨的缘故，更加难以攀登，有时候一夜只能送上去几十人。经过这么长时间，山寨的老弱妇孺已经完全送到外面，苏樱和柳芷也在其中。
<br/>　　如果没有什么意外的话，这些人当在高飞虎和高飞豹的保护下，坐上了南下的船。
<br/>　　那里风调雨顺，国泰民安，当能过上几年好日子。
<br/>　　现在山寨里剩下的，不过是程大雷、徐神机、秦蛮、林少羽、林冲等百人而已。
<br/>　　山谷，可以说已经是一座空城。
<br/>　　程大雷本来是想趁着今日撤离的，可还来不及撤退，他便知道了戎族将全力攻城的消息。
<br/>　　“大当家，时间来不及了。”
<br/>　　徐神机心惊胆战，以现在蛤蟆寨的兵力，也别舔着脸说守城了，戎族可以直接长驱直入，以手中的刀收割众人的生命。不，他们或许不会一刀砍了大家，以戎族对蛤蟆寨的恨意，估计是砍掉四肢，养在坛子里做一个活死人他们才痛快。
<br/>　　“大当家，你们先走，我带兄弟去守城，为你们争取世间。”秦蛮握紧铁枪。
<br/>　　程大雷把蛤蟆寨的大旗叠起来收在怀里，道：“别废话，让弟兄们先走，我去守城。”
<br/>　　“你一个人怎么守！”众人一惊。
<br/>　　“如此，看来本当家只能为他们奏一曲空城计了。”
<br/>　　程大雷身背大斧，腰间佩剑，大步流星的向城门走去。
<br/>　　城门外，大军欲发。
<br/>　　正在此时，只见程大雷背大斧，佩宝剑，携琴一张，登上城门，凭栏而坐，焚香三柱。
<br/>　　当要进攻的大军停住了，众人心里同时冒出一个想法：这货又搞什么幺蛾子。这一瞬，他们都想起被程大雷恐惧支配的日子。
<br/>　　城门上只有一琴，一人，守兵全无。
<br/>　　程大雷看着面前的瑶琴愣住了，伸出去的手落也不是，收也不是，原因是此刻他才想起：自己根本不会弹琴。
<br/>　　若是吉他程大雷还是能凑活着来一曲的，当然水平也不如何。这就是崇洋媚外，不潜心研究国学的苦果啊。
<br/>　　此刻想cosplay丞相，可自己却没有丞相的手艺。
<br/>　　怎么办，在线等，挺急的。
<br/>　　铮！
<br/>　　琴音忽起，程大雷赶鸭子上架，用弹吉他的手法弹瑶琴，反正都是哆瑞咪发嗦啦西，大概……差不了多少吧。
<br/>　　只听琴音时断时续，有时停顿好长时间，听声音仔细辨别，大概节奏是：
<br/>　　『……哆瑞咪哆，哆瑞咪哆，咪发嗦，咪发嗦……』
<br/>　　程大雷所弹奏的，正是那首经典不衰，渊源流传的——《两只老虎》。
<br/>　　大军已兵临城下，此时突然停住。
<br/>　　“贼人故作此态，大哥，我看不如杀进去。”
<br/>　　“对，杀进去。”
<br/>　　“不可妄动。”大王子摇摇头，侧耳倾听着琴音。
<br/>　　过得半晌，他脸上忽然挂出一抹冷笑。
<br/>　　“这铮铮之音如惊涛拍岸，风卷残云，指端私有雄兵百万。”
<br/>　　“咦，大哥我怎么听不出来。”二王子道。
<br/>　　大王子冲他抿嘴一笑：“你听，似山间小溪，清澈见底，非心旷神怡着不能为之，程大雷定然是胸有成竹。”
<br/>　　“几根琴弦岂能如此传神？”
<br/>　　“心乱则音噪，心静则音纯，心慌则音误，心泰则音清，听此人弹琴，如观其肺腑也。”大王子抚须缓缓道：“传我军令，退兵。”
<br/>　　“……”二王子。
<br/>　　“……”五王子。
<br/>　　“……”金问道。
<br/>　　所有人心头一群草泥马奔过，哪里有铮铮之音了，哪里有雄兵百万了……大哥，你脑袋不会秀逗了吧，好不容易赶上一个好天气，你却要退兵。
<br/>　　“杀呀！”
<br/>　　突然一声咆哮，只见一员大将手持大锤，挥舞着冲杀了出去。此人真是北蛮部六王子雷怒。
<br/>　　“嗨，回来，回来。”大王子马上大叫：“小心中计。”
<br/>　　雷怒领着一群人冲杀出去，大王子无力的看着这一幕，只能鄙夷的长叹一声：没文化真可怕呀。
<br/>　　他并不知道，此刻城头之上的程大雷发出了和他一样的感慨。
<br/>　　程大雷正弹得愉悦呢，忽然喊杀声响起，他吓得一跌，心脏差点从腔子里跳出来。想丞相当年，为何能够悠哉悠哉的城头一曲退仲达，而赶上自己，他们怎么就冲了过来，难道自己cosplay得不像。
<br/>　　程大雷恍然明白，丞相和司马都是聪明人，聪明人的计谋只对聪明人有效。这就像给聪明人一个密码箱，聪明人自然会用各种高深的法子破解密码。而若把密码箱给一个莽汉，然后，就看到他拎着一柄斧头过来了。
<br/>　　特么的，没文化真可怕。
<br/>　　程大雷一脚把琴踹翻，转过身就向山谷逃，口中大叫：“逃啊！”
<br/>　　这一幕也让大王子愣住了，和自己所想得好像有些不一样。
<br/>　　“杀呀！”
<br/>　　戎族十万大军，一窝蜂的扑了过来，冲破城门，未遇抵抗。
<br/>　　身后大军紧紧追赶，这要是被追上了，十万大军，估计把程大雷熬成汤，每个人都分不到一口汤喝。
<br/>　　程大雷不顾一切的向前狂奔，此正应了那四个字：狼奔豕突。
<br/>　　此时戎族大军已经突破山谷，只见山谷内空空荡荡，一个人影也无。
<br/>　　他们什么时候走了，莫非这程大雷会五行遁不成。
<br/>　　不得不说，程大雷真的把这些人搞出了心理阴影。
<br/>　　“莫要放箭，我要抓活的。”大王子狂吼。
<br/>　　其实不用他开口，也没人会放箭。所有人心里都憋着一口气，只等着把程大雷活捉，用各种各样的法子折磨他。
<br/>　　“我有一千种法子，一千种！”金问道咬着牙大喊。
<br/>　　此时，徐神机几人已经爬上山头，冲着程大雷紧张的大喊：“大当家，快上来！”
<br/>　　程大雷气喘吁吁的跑到山谷底部，一条绳索悬在那里，死亡激发了程大雷的身体潜能，在筋疲力尽的状态下，他依旧保持着极快的速度。
<br/>　　“莫要让这狗贼跑了，我有一千种法子折磨他，一千种。”金问道牙齿摩擦得咯吱咯吱响。
<br/>
