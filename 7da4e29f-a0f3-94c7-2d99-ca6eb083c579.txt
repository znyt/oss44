　　钦陵带着那支由苏定方打造数年的骑兵离开了，向着心中胜利的方向。
<br/>　　辽东的战事也在浑河边一战之后陷入僵持阶段。
<br/>　　而长安城，此时却已经翻了天。
<br/>　　铁甲战舰终于完工了，整艘战船除了内部装修其他全部由钢铁打造，这是历史上第一艘钢铁战船，它用钢铁的事实向这世界上所有的土鳖们宣布，并不是比水重的东西都会沉底！
<br/>　　李二陛下来了，长孙皇后也来了，包括那些世家的族长、勋贵家族也全都来了。
<br/>　　整艘铁甲舰上张灯结彩，宛如节日般热闹。
<br/>　　而在河岸的两侧，无数大唐百姓聚集到了这里，观看大唐第一艘钢铁战船的试航。
<br/>　　……
<br/>　　“父皇，人都已经上来了，启航么？”李承乾穿着一身和老头子款式一样的军礼服，手中擎着一杯红酒，凑到老爹身边。
<br/>　　“你看着办，今日朕就是来凑个热闹。”李二似乎很享受这种自助餐式的聚会，手里同样端着一杯红酒，正在与房玄龄和长孙无忌两人聊着什么，随意的摆了摆手便把李承乾打发了。
<br/>　　不过想想也是，整艘战舰从开始建造一直到竣工，老头子根本连问都没有问过，现在自然也不会在这上面指手画脚，专业的事情交给专业的人来办，这一惯就是老头子的宗旨。
<br/>　　当然，更有意思的是，李承乾为了让这次初航更有意义一些，还在船上办了一个类似与自助餐的宴会，左右这些上船的大佬们也是来玩的，不如让他们玩的尽兴一些好了。
<br/>　　所以当老头子示意李承乾可以自行决定之后，李承乾果断的撤了，跑到船长室找到了小唐，兴奋的说道：“走了走了，可以开船了！”
<br/>　　“好，我去通知！”小唐正在船长室里没头苍蝇一样的乱转，听到可以开船了，立刻打起了精神，来到角落中一排铜制的管子跟前，对着其中一根吼道：“轮机组，最大火力输出。”
<br/>　　片刻之后，船长室中一个大喇叭传出声音：“轮机组明白。”
<br/>　　“甲板组，起锚！”
<br/>　　“甲板组明白！”
<br/>　　“各战斗组，准备！”
<br/>　　”战斗组明白！”
<br/>　　“……”
<br/>　　用铜管传递声音，这是李承乾从电影《地道战》里学来的，当然，可能初中物理课本也有过，但没有《地道战》的记忆清楚。
<br/>　　不过也正是得益于这一点，使得船上的消息传递要轻松许多，不必向以前一样吵吵嚷嚷，毕竟外面还有许多大佬在呢，乱七八糟的十分丢脸。
<br/>　　……
<br/>　　下面人的动作十分讯速，想来是知道了皇帝就在船上想要表现一下吧。
<br/>　　不到一刻钟，已经陆续有人通过送话的铜管回报，准备就续。
<br/>　　唐善识与李承乾对视一眼，嘴角微微抽动，但最终没说什么，转头对两个月前从水师调来的一位船长问道：“你的训练应该都完成了吧？接下来应该怎么做记得没有。”
<br/>　　那船长已经不知道在这船上待了多长时间，每天就是摆弄船上的这些东西来练习，早就已经练的滚瓜烂熟，就算是闭着眼睛都可以操作，所以在听完小唐的问题之后斩钉截铁的回答道：“大人，下官记得！”
<br/>　　“那就开始吧，不要丢脸！”小唐点点头。
<br/>　　“是！”船长兴奋的吼了一声，吓了小唐一跳。
<br/>　　不过现在已经丫已经顾不上这些了，回答过后，讯速观察起船上的各种仪表，然后又对那一排送话用的铜管下了几个命令，比如：减压百分之三十；打开节流阀等等。
<br/>　　……
<br/>　　庞大的钢铁战舰顶上有两个硕大的烟囱，这代表着它是由两台巨大的蒸汽机来驱动的。
<br/>　　当所有准备工作完成之后，随着船长的一声令下，两只硕大的烟囱突然喷吐出滚滚浓烟，巨大的气笛声中，战船的船身有了那么一丝轻微的抖动。
<br/>　　片刻之后，船上众人已经明显可以感觉到战船在缓缓移动。
<br/>　　要知道，眼下可是逆水流啊，一只没有船桨，没有风帆的钢铁战舰竟然在逆水而上？
<br/>　　所有略微懂行的人几乎都惊呆了，尤其是船上的那些世家的族长之类。
<br/>　　他们在来之前已经向有行船经验的人打听过，一至认为铁甲舰就算是能浮起来，但是动力却是最大的问题，在没有船桨，没有风帆的情况下，根本不可能移动。
<br/>　　当然，如果顺水而下的话或许有可能，但最终的结果也只能是漂到哪里算哪里，船上的人最后都需要坐着马车回来。
<br/>　　可事实证明，他们全都错了，这船真的可以开动，而且随着两只烟囱的浓烟滚滚而上，速度竟然越来越快，渐渐的已经可以超过官道上跑来跑去的马车。
<br/>　　所以水师的将军全都不说话了，也不管那些摆在露天的食物上落满了黑灰，呆呆的往嘴里塞着，眼睛却全都盯在了高高在上的船长室。
<br/>　　世家的老怪物们也都不说话了，一个个只觉得老脸火辣辣的疼，就好像被人狠狠抽过几巴掌一样。
<br/>　　当然，这并不是说战舰上便没有人说话了，至少一直被众人贬低，认为家门不幸出了一个不孝子的唐俭开始得意了。
<br/>　　整艘战舰上，除了汽笛的声音之外，就只有这老货的笑声存在。
<br/>　　逮住一个人也不管是谁就开始嚷嚷：“怎么样，怎么样，这船就是我儿子造的！哈哈……”
<br/>　　这特么真是太尴尬了，整的小唐本想下到甲板上装个逼最后都没敢下去。
<br/>　　不为别的，因为怕下去之后挨揍。
<br/>　　毕竟那些老货现在可都被他家老头儿给得罪了，一个个正在气头上，下去之后免不了被当成出气筒。
<br/>　　可事与愿违，就在李承乾和小唐躲在船长室里装熊的时候，一个小太监跑了上来，敲开门后说道：“太子殿下，唐大人，陛下要你们两个下去，说是要给您二位庆功！”
<br/>　　庆功？庆什么功？老头子不是来玩儿的么？开始也没说庆功之类的事儿啊？而且看外面这情况，根本也不像是庆功的举动，要啥没啥庆个屁啊？
