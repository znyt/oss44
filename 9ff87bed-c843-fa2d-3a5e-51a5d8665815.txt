　　听到叶帝这么说了，众人便是也不再有其他的意见，现在直播间里的人数才刚刚过4亿，按照以往的惯例肯定还会有很多人要进来的。
<br/>　　而打一局游戏的时间，应该是足够让这群人进来了，反正他们也很喜欢叶帝直播打游戏，看看倒是也无妨。
<br/>　　今天之所以是这样，实在是他们被叶帝的这首新歌给勾的不行。
<br/>　　而且最关键的是叶帝直播的时候大部分时间都是在玩游戏的，出新歌的次数虽然较之其他歌手快了很多，但是和他打联盟的时间可是就要少上很多很多了。
<br/>　　物以稀为贵，所以在听到叶帝今晚要发新歌的时候，这些观众自然是比较期待这个了。
<br/>　　大概过了十分钟左右的时间，叶帝这边终于是排到了选手，排到的也大都是熟人。
<br/>　　看到是排到了叶帝，叶帝这边的几名队友纷纷选择将中单这个位置让了出来，而轮到叶帝选人的时候，他则是选了一个自己玩的很少的狐狸。
<br/>　　“小叶子这是要玩狐狸？这个英雄很少上场啊。”看着叶帝选择的英雄，蒋念的眼睛一亮小声的说道。
<br/>　　直播间里的众人看到叶帝的选人，一个个眼睛也都是亮了起来，对于这种稀奇古怪的英雄，大家还是非常好奇的。
<br/>　　而叶帝这次选择这个英雄最主要的原因就是，他发现了这里的冥火居然没有取消，事实上这个装备他早就发现了，只不过之前冥火的属性太垃圾了，他一直没有出过而已。
<br/>　　而在前几天冥火这件装备加强了，提升到了他所知道的那个版本的冥火之拥的属性，所以叶帝自然是想要体验一下了。
<br/>　　至于昨天的比赛为什么没出，只能说是习惯性原因，因为长此以往的就出那几件装备，所以叶帝几乎每次回家都是秒出装备然后出城的。
<br/>　　狐狸这个英雄虽然如今上场率不高，但是有着冥火这件装备之后，狐狸确实已经是具备了秒人的能力了。
<br/>　　【物品属性】：+120点法术强度，+10%冷却缩减
<br/>　　【唯一主动】:造成目标最大生命值15%的魔法伤害，并在接下来的4秒里，使目标英雄所受的魔法伤害提高20%。冷却时间90秒。施法距离750。
<br/>　　只看看这件装备的属性，就知道它有多么的暴力了，不仅加法强，减冷却和最大生命值百分比的伤害以及寺庙内提高的百分之二十输出，这对于法师来说都太关键了。
<br/>　　在S3那个时期，Faker能够以法师刺客英雄证道，其实这件装备也是立功不小。
<br/>　　很快双方的阵容便是确定了下来，叶帝这边的上单是武器、打野盲僧、中路狐狸、下路则是ez和娜美。
<br/>　　而对面则是选择了上路剑姬、打野蜘蛛、中路机械先驱、下路则是女警和娜美这两个英雄。
<br/>　　看着双方的阵容，叶帝的眼睛再次一亮，这个时候他终于是想出了另外一个套路。
<br/>　　之前他总感觉自己好像忘了些什么，直到今天看到这个阵容的时候，他才想起来。
<br/>　　奶妈这个英雄现在可以打上单啊，他的技能还都没有做过改动的，w技能依旧是回血还能增加护甲值，e技能依旧是给人回蓝，对敌人释放则是会造成沉默。
<br/>　　当然了这个英雄之所以能够打上单，最关键的还是她的q技能，奶妈的q技能可是全等级2.5秒的，而且q技能还会降低对方的魔法抗性，做多能够叠加五次。
<br/>　　你可以想象一个给自己奶血，提升护甲值之后的奶妈，在上去对别人一直释放q技能的样子，如果对面真的是一个类似武器这种的上单，那肯定是要被奶妈无情制裁的。
<br/>　　“哥你怎么了，玩个狐狸至于把你激动成这样么。”身后的香蕉忍不住伸手拍了拍叶帝的肩膀提醒道。
<br/>　　“咳咳，你小子有福气了，哥哥我刚刚想到了一个无敌上单的套路，等直播结束我交你，到时候保证你分分钟虐暴小南。”叶帝斜倪着香蕉，一脸你占便宜的表情。
<br/>　　“哥，你真是我亲哥，我早就想虐楚南了，哈哈哈哈！”听着叶帝这么说，身后的香蕉刹那间愣神之后，直接得意的笑了起来。
<br/>　　“卧槽，真的假的啊，我现在怎么感觉背后凉飕飕的。”一旁的楚南有些心虚的开口说道。
<br/>　　“我看挺像的，要不然他至于高兴成这个样子么。”秦霄在一旁也是倒吸了一口凉气。
<br/>　　如果真的被叶帝想出那样一个无敌的套路的话，那他们在夏季赛决赛相遇的时候，恐怕还真的有些难办了。
<br/>　　直播间里的众人听到叶帝的这句话后，一个个也都是满脸的楞然，选了一个狐狸，然后就想到了一个无敌的套路，要不要这么悬啊。
<br/>　　一旁的蒋念这一刻也是瞬间换了一副讨好的表情，在叶帝身旁和他套着近乎，不过任凭他怎么说，叶帝这边就是不透露丝毫消息。
<br/>　　“你傻还是我傻，不要跟我套近乎啊，咱来不是一个战队的。”叶帝鄙视的说道。
<br/>　　“说什么呢，你这就是不把我们当兄弟了，我们不都是荣耀战队的么，我们可都是一个队伍，一个血统的啊，想当年我们五兄弟一起叱咤战场的时候，那是多么的潇洒啊，今天咱们依然还是，有套路自然也要共享。”蒋念一脸正义凛然的说道。
<br/>　　屋子内，直播间里的众人都是再次被蒋念的无耻给刷新了三观。
<br/>　　不过叶帝这边根本就没搭理他，在游戏开始过后，他便是已经将目光放到了自己的电脑屏幕上，仿佛已经自动屏蔽了蒋念一般。
<br/>　　而等到时间到达16分钟的时候，当众人看到叶帝做出冥火之拥这件装备一套技能直接将对方中单机械先驱秒杀，甚至都没给对方释放净化的机会的时候，无数的观众再次呆住了。
<br/>　　而那些观看着叶帝直播的职业选手还有教练，此刻也是都有些发呆，感觉狐狸这个英雄的伤害有些高的离谱了。
<br/>　　至于另外一些人则是看着冥火之拥这件装备久久无法回过神来。
