　　但是没有一个人敢来抢劫，因为林峰故意放出自己的气势，就是为了敲山震虎，这群人虽然羡慕林峰吃得起辣条，但是却没有任何歪主意，甚至害怕林峰抢劫他们，提前就躲得远远的。
<br/>　　哒哒哒哒哒。
<br/>　　也就在这个时候。
<br/>　　一架直升机从远处飞了过来。
<br/>　　“所有幸存者，都可以前往位于江城郊区的曙光基地，在那个地方，我们已经建立起了绝对安全可靠的安全区域可以让幸存者们避难！”
<br/>　　“所有幸存者，都可以前往位于江城郊区的曙光基地，在那个地方，我们已经建立起了绝对安全可靠的安全区域可以让幸存者们避难！”
<br/>　　在这直升机上挂着一个喇叭，不断的在重放着这么一条消息。
<br/>　　“什么？我没有听错吧老公，是幸存者基地？”
<br/>　　“老婆，你们没有听错，真的是，我们得救了！”
<br/>　　所有幸存者都是哈哈大笑起来，甚至激动的老泪纵横，感激涕零的。
<br/>　　还有人感谢上帝感谢耶稣感谢伟大领袖。
<br/>　　吼吼吼
<br/>　　但下一刻他们感谢不出来了。
<br/>　　因为许多丧尸被喇叭的声音给吸引了，纷纷躁动的冲了出来。一些丧尸甚至从楼上跑下来有的直接摔死了有的指示摔断了腿但却依然在地上爬。
<br/>　　“跑！”
<br/>　　不过现在大部分人都已经是黑铁战士了，倒也足够可以自保。
<br/>　　林峰杀了几个不长眼的丧尸，第一时间回到了学校。
<br/>　　“林峰，你听到了吗？是幸存者基地！我就知道，军警肯定没有那么容易就被打败的。”
<br/>　　陈勾他们自然也听到了，一个个都无比兴奋。
<br/>　　林峰也兴奋，“终于等到了，既然如此，我们赶快收拾一下，然后前往幸存者基地！”
<br/>　　“嗯。”
<br/>　　每个人都立即去收拾东西。
<br/>　　实际上也没有什么东西可以收拾的，就是一些还没吃完的东西拿个大号登山包一装，然后再带些武器就可以了。
<br/>　　林峰等人从学校里出来时，不少幸存者也都大包小包着从小商铺里出来，这一幕就好像是大家春节回家过年了一样。
<br/>　　林峰等人自然是开着车的，不少幸存者也都开着车，但更多的还是徒步走。甚至还有美团小哥骑着电瓶车的。
<br/>　　这么多人自然吸引了丧尸，有的人走的好好的被忽然出现的丧尸扑倒在地，有的人反应快和丧尸厮杀起来。
<br/>　　也有的人开着车去撞丧尸，也有的人被丧尸从车子里拿出来，大卸八块。
<br/>　　“前往基地的路上不知道要死多少人。”
<br/>　　苏云感慨道。
<br/>　　“没办法，生活就是这样。为了达到某个目的，往往都要付出许多。”
<br/>　　林峰道。
<br/>　　他们开着两辆车。
<br/>　　他和陈勾吴三外加苏云坐在宝马X5内。
<br/>　　李广和他的手下坐在一个金杯面包车内。
<br/>　　两辆车直接朝江城郊区驶去。
<br/>　　不过没走多远，林峰他们就不得不弃车徒步而行了。
<br/>　　一路上直接被堵死了，许多丧尸在车子之间怒吼着。
<br/>　　“妈的！”
<br/>　　林峰等人只能下车行动。
<br/>　　“峰哥！”
<br/>　　李广等人下了面包车追了上来。
<br/>　　“嗯，注意丧尸，也要注意其余幸存者，我们背着大包小包的，很容易被其余人盯上！”
<br/>　　林峰道。
<br/>　　“放心吧峰哥，那群家伙敢来打我们主意，我手中的砍刀肯定是白刀子进红刀子出！”
<br/>　　“嗯。不发生冲突最好，若是发生冲突，那我们只能下狠手，没必要手下留情。”
<br/>　　“走。”
<br/>　　林峰大手一挥，一群人立即朝基地奔去。
<br/>　　一路上，不少幸存者，其中有男有女有老有少，有单独行走的独狼，也有成双结对的情侣，也有三三两两的队伍。
<br/>　　诸如林峰这样几十个人一起大规模的还真的不多见，毕竟每个人都是活人，都有自己的想法，很难把一群人组织在一起。
<br/>　　“老婆，你怎么了。”
<br/>　　走在林峰身边的幸存者，其中一个十七八岁的女孩子忽然晕倒了。
<br/>　　看样子似乎是因为长时间没有喝水，脱水导致的。
<br/>　　“啧啧，苏老师你看看，这男的怎么说也八十多岁了吧，老婆居然这么年轻，才十七八岁啊，和我差不多大。”
<br/>　　林峰扫了眼那个男的，一看就知道是个总裁。
<br/>　　在他身边还跟着一个男的，似乎是保镖的样子。
<br/>　　“和我说这些干嘛。”苏云也看到了。
<br/>　　“没什么，只是我觉得，像老师这么漂亮而且还独立的女生特别少见了，明明可以给人包养非要当老师。”林峰道。
<br/>　　“胡说什么呢。”苏云瞪了眼林峰。“小小年纪不学好，社会上的那些负面新闻虽然存在，但也不能以偏概全，不是所有女生都喜欢钱的。”
<br/>　　吼吼吼
<br/>　　忽然前面爆发出一阵咆哮。
<br/>　　众人就看到了十几头变异大狗冲了上来，一口就是把一个人给吞了。
<br/>　　“啊有怪物啊。”
<br/>　　“三阶的怪物？快跑！”
<br/>　　“儿子，快走！”
<br/>　　“爸爸！我的爸爸呢！啊！我找不到爸爸啦！”
<br/>　　瞬间原本还算平静的幸存者队伍一下子就好像是那沸水一样哗然一片。
<br/>　　人群的尖叫声，怪物的嘶吼声，还有各式各样的声音无比喧杂。
<br/>　　面对怪物，有人立即抱着自己老婆掉头就跑的，也有把自己老婆丢下独自逃跑的。

<br/>　　林峰等人不为所动，通往幸存者基地这条路最近，十几头三阶的还不足以让他们放弃走近路而去绕远路。
<br/>　　“狗子，李广，你们两个跟着我去把那一群变异狗砍了，其余人原地警戒！”
