　　在通道中约莫走了半个小时，一抹光亮终于出现在通道的尽头，看见光亮，通道内的几人不约而同的加快步伐。
<br/>　　片刻后，终于抵达通道尽头，刺眼的阳光自天际挥洒而下，使得身后几人下意识的闭上双眼。
<br/>　　慕清霄望着映入眼中的景象，饶是已经猜到一些，心中也是极其的惊讶。
<br/>　　转身看了一眼身后，空中有一口漆黑色的洞口，根本看不到任何障碍物，也看不到任何一幢迦南学院中的建筑。
<br/>　　若是猜的不错，此地估计是一处小型空间秘境。
<br/>　　斗尊级别就能掌握空间领域，开辟出属于自己的领域，而迦南学院的院长，是半圣级别，到了那种层次，谁也不知道有多恐怖。
<br/>　　创造出空间秘境，并非没有可能。
<br/>　　身后几人睁开双眼，顿时惊得合不拢嘴，看周围的景象，明显是一处山谷凹槽。
<br/>　　那陡峭的山壁一路向上延伸，最后直至视线的尽头，在三面峭壁的中央处，正好有着一个占地极为宽阔的空地。
<br/>　　而在空地上，一幢极其庞大厚重的古朴楼阁，正矗立其上。
<br/>　　视线缓缓扫过空地上的古朴楼阁，最后停留在楼阁上一块显得极为古老的匾额上，三个被岁月摧蚀的有些模糊不清的字体，正若隐若现的显露而出。
<br/>　　藏书阁！
<br/>　　古老的字迹，虽然历经岁月摧残，但慕清霄依旧能感觉到，字体中存在着某种意境，意境玄妙，难以领悟。
<br/>　　不愧是斗气大陆最古老的学院，仅仅是这一块匾额，便将它悠久的历史与底蕴，完全的体现了出来。
<br/>　　琥乾带着八名学员缓步走向藏书阁，在即将进入其二十米距离时，却是陡然挺住脚步，随后对着藏书阁方向抱拳，道：
<br/>　　“这一届内院选拔赛前五名已经诞生，其中一名自愿待在外院，另外两名为特招生，按照规定，我带他们来此处，还请诸老开门！”
<br/>　　顺着琥乾的方向，慕清霄的视线落在盘坐于大门口处，两个灰袍人影身上。
<br/>　　此时，他的心中震惊无比，刚进入这里的时候，完全没有察觉到两人的存在。
<br/>　　但现在，这两个灰袍老者，却是宛如原本就坐在那里一样，这般诡异的场景，多半已经领悟一丝空间领域。
<br/>　　就好比如慕清霄自己，在领悟空间领域后，能够完全的融入这片空间中，哪怕有人经过，都仿佛没看到一样，当然，实力必须比他低一些。
<br/>　　而现在的场面，也是一样的道理。
<br/>　　两人皆是斗宗巅峰，但与他前些天击杀的魂殿斗宗比起来，后者连渣渣都算不上，根本不是一个层次的。
<br/>　　两人给慕清霄的感觉，就是深不可测，不可力敌，最起码现在的他不可能是对手。
<br/>　　琥乾的声音在山谷中回荡，随后消散，而盘坐在楼阁下的两位灰袍人却是没有一丝动静，仿佛没有听到一般。
<br/>　　见到自己的话语没有回应，琥乾也未再度出声，一直保持着抱拳的姿势，安静的等待着。
<br/>　　慕清霄不说话，小医仙也安静等待，古薰儿面色平淡，至于身后的五名男学员则面面相觑。
<br/>　　琥乾在迦南学院中的地位，没有几位能够相比，然而面对楼阁下的灰袍人却如此恭敬，足以见得两人有多么恐怖！
<br/>　　此时，慕清霄在思索，他可没有自信能够在两人面前不露马脚，若是两人看破，不知会不会对他出手。
<br/>　　若真对自己出手，哪怕化身黑凤，恐怕也无法逃离此处。
<br/>　　寂静的气氛在山谷中持续了将近十几分钟，两名盘坐在楼阁门口处的灰袍人影，袍服终于是抖动了一下。
<br/>　　旋即，灰袍下有着两对犹如老僧般古井无波的视线抬了起来，淡淡的在琥乾身上扫过，最后却是停留在了慕清霄身上。
<br/>　　灰袍猛的一颤，有着细微嘶哑惊疑声传出：“真是妖孽，还有一股令我二人都感到余悸的气息，远古？不对，或许更远...”
<br/>　　话语刚落，慕清霄紧绷的身躯也放松下来，视线中没有敌意，果然瞒不过这两个老怪物，毕竟实力相差太悬殊了。
<br/>　　细微的声音，缓缓在寂静的山谷中回荡着，惊疑的话语也传入所有人耳中。
<br/>　　琥乾若有所思，古薰儿盯着慕清霄，眼底掠过浓浓的震惊，看来后者的身份，比她推断的更加不简单。
<br/>　　至于身后五名路人甲，则不明所以，一脸懵逼的模样。
<br/>　　“不知这位小友名讳？”
<br/>　　其中一名灰袍人，目光紧紧盯着慕清霄，嘶哑的声音中，却透露出一抹岁月堆积而成的沧桑，还有一抹难以掩饰的惊喜与好奇。
<br/>　　对于两位灰袍人的问题，慕清霄不卑不亢，微微抱拳，道：“在下慕清霄，见过二老。”
<br/>　　“嗯，非常的不错，后生可畏啊，虽看不出你的来历，但你的天赋是我二人有史以来，见过最为恐怖的！”
<br/>　　苍老的声音中，尽是赞赏与欣慰，在慕清霄耳边徘徊，却也使得他心情放松下来，看来两人只是好奇，并非要归根究底。
<br/>　　“只不过，你来这里，怕是学不到什么东西啊......罢了，罢了。”
<br/>　　旋即，老者视线在小医仙身上停顿了一下，又在古薰儿身上停顿了一下，最后才落到琥乾身上。
<br/>　　“除了你身后的三名，最近的几届都比较一般。”
<br/>　　闻言，琥乾松了口气，心中却非常的震惊，他来这里许多回了，可还是第一次见两位长老说如此话多。
<br/>　　而且，根据他们话语中的信息推测，好似站在身后的青年，来历非常不简单，而且天赋更是有点吓人。
<br/>　　但二十岁的八星斗师，天赋怎么也跟恐怖扯不上边吧？
<br/>　　“既然如此，那便请二老解开一下空间枷锁吧。”
<br/>　　慕清霄目光一凝，果然已经领悟到了空间领域，随后，目光望着琥乾面前半米处的空间。
<br/>　　片刻后，半米处的空间出现一丝丝极浅的皱褶，皱褶隐藏在空间中，极难被发现。
<br/>　　目光顺着若隐若现的皱褶移动，直到最后，整片空间皱褶几乎是形成一扇墙壁模样，将整个藏书阁护在其后。
