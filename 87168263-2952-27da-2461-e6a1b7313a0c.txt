　　虽然李承乾说的是有机会要见见那个彭城，但皇帝陛下要见的人，就算是再忙也必须有时间，白月宁可不管那个彭什么彭有没有时间，眼下再干什么，反正只要没死，就是爬也必须立刻爬到皇帝陛下的面前。
<br/>　　不过话说回来，彭城在见到白月宁之后也是懵的，就算打死他也想不到，自己竟然有面前皇帝的机会。
<br/>　　这已经不是祖坟冒青烟可以形容的了，如果非要把祖坟和烟联系到一起，彭城认为那一定是老祖宗的棺材板被点着了，把祖坟烧的浓烟滚滚才行。
<br/>　　“见到皇帝陛下自然一些，不要谄媚，说话要注意措词，尽量不要说一些让陛下不开心的事情，陛下问什么你就说什么，明白么？”御书房外面，小白最后对彭城叮嘱道。
<br/>　　“是，白，白大人放心，小人都明白了。”彭城兴奋的脸色通红，把头点的飞快，这让小白怀疑他不会兴奋过度，以至于脑袋被自己的血液胀爆掉。
<br/>　　……
<br/>　　“你就是彭城？不错，正是年富力强的时候。”李承乾在长孙冲离开之后，召见了彭城，看着眼前的中年人点头称赞道。
<br/>　　“谢，谢陛下！”彭城手足无措的站在那里，手只手一会儿抬起来一会儿放下，想要抱拳又不知道对不对。
<br/>　　李承乾已经不是第一次遇到彭城这样的例子，笑着让内侍给他搬了把椅子，又让他坐下，然后才说道：“你不用紧张，朕找你来只是想跟你聊聊，并没有为难你的意思。”
<br/>　　“是！”彭城只坐了一个椅子边，看上去就像在蹲马步差不多。
<br/>　　可能是因为太过紧张让他全身的肌肉都在绷紧的状态，否则他现在的造型无论如何也不可能坐得住。
<br/>　　李承乾也知道彭城的紧张不会因为自己的一两句话就消除，所以也没有多说什么，淡淡看了他一眼之后问道：“第二次阶段的招标工作是在明天吧？”
<br/>　　“是，是的！”彭城点点头，牢记小白的叮嘱：问什么答什么，不要多说一个字。
<br/>　　“那么你觉得明天第二阶段的均价能够达到多少？随便说说，错了也没有关系。”李承乾从桌子后面走出来，亲自递了一杯茶给彭城，顺便问道。
<br/>　　“这，草民不知！”
<br/>　　“没关系，你就当满足朕的一个好奇心，比如说如果你明天要竟价的话，你的心理底限会是多少？”
<br/>　　“这……”彭城此时急了已经快要哭了，想到白月宁的叮嘱，他实在不知道自己应该怎么开口，应不应该继续说下去。
<br/>　　李承乾似乎看出了他的纠结，顺着他的目光看了眼小白，摆了摆手说道：“你们都出去吧，朕与老彭单独聊聊。”
<br/>　　“是！”小白答应着，瞪了彭城一眼之后，带着房间中的其他宫女和内侍走了出去，顺带在外面将门关了起来。
<br/>　　“好了，现在没有外人了，你可以说了吧？”李承乾等到小白她们都出去了，这才从一边拖了一张椅子过来，坐到彭城的对面问道。
<br/>　　“陛下，草民，草民真的不知道应该怎么说。”房间里已经没了外人，可是彭城却更紧张了。
<br/>　　刚刚小白在房间里，多少可以算是他的主心骨，什么该说什么不该说他怎么也能得到一些提示，可是现在人都走光了，这到底啥是该说的，啥是不该说的，谁能搞的清楚，万一说错了话，会不会被打出去拖死？
<br/>　　“彭城啊，你可知道什么是欺君之罪么？那可是要杀头的！”李承乾见彭城实在太过紧张，忍不住生出恶搞的心思，故意逗他说道。
<br/>　　“噗通”一声，随着‘杀头’两个字出口，彭城再也坚持不住，一屁股坐到了地上，刚刚还兴奋的发红的老脸瞬间惨白，嘴唇哆嗦着，但‘饶命’二字却怎么也说不出来。
<br/>　　李承乾也没想到彭城这家伙胆子竟然这么小，被自己一吓竟然直接瘫了，一时间也是尴尬不已，最后想了半天才说道：“不过如果你的回答如果能让朕满意，朕可以不追究你的欺君之罪。”
<br/>　　“谢，谢陛下，草民，草民一定知无不言，言无不尽。”彭城应该是真的被吓到了，说话比刚才利索了不少，末了似乎想起什么，连忙说道：“陛下，草民觉得明天的均价应该在三千贯左右，草民打算把余下的三万贯全部投入进去，直接购下七块广告牌。”
<br/>　　“嗯！”李承乾心中好笑，不过脸上却没有表现出来，点了点头之后对彭城说道：“你先起来坐下，朕还有话要问你。”
<br/>　　“是，是是！”彭城把头点的飞快，强撑着从地上爬起来，坐到刚刚李承乾给他安排的椅子上面，两腿并拢就像蒙学中的孩童一般。
<br/>　　李承乾等他坐好了，自己也坐到了他的对面，正色问道：“说说你的判断吧，是什么让你做出三千贯这个判断的。”
<br/>　　“陛下，草民觉得一切都是因为今天的均价。”彭城定了定神，整理了一下思路，把心一横说道：“暗标竟价大家并不是第一次，不过这次很多人都被表面现象迷惑了，将目光全都投在了广告牌的上面，却没有意识到接下来的商业洽谈才是根本。”
<br/>　　“但是经过今天的竟价之后，相信那些被迷惑的人也已经回过神了，他们一定会重新审视自己的标价。”
<br/>　　“今天已经有三百个广告牌被拍出去了，而在所有人都意识到商业洽谈的重要性之后，余下的广告位只会越来越贵。可是大家的财力是不同的，那些大商人或许不会在乎价格的高低，但是像草民这样的小商人却不能不在乎。”
<br/>　　彭城说到这里顿了顿，似乎在考虑接下来应该怎么说，半晌之后才再次开口：“所以草民认为，明天大多数与草民身价差不多的商户会选择孤注一掷，把手里的全部资金都砸在第二阶段的广告位上面。”
<br/>　　“嗯，十鸟在林，不如一鸟在手是吧？”李承乾接过彭城的话头，笑着说道。
<br/>　　“呃……，是，是的！”彭城点点头，心里长长的出了一口气，暗道总算是把这一段给应付过去了。
<br/>　　可是，接下来李承乾却笑着对他说道：“朕记住你的话了，如果明天的均价真的达到三千，朕重重有赏，若是没有达到……”。
<br/>　　没有达到会怎么样？难道又是杀头？冷汗不由自主从彭城的头上流了下来。
