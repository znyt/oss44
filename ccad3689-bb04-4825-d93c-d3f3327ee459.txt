www.biqusa.info]爱好者，二十二年前的时候，我四处旅居偶然间来到滇西抚仙湖，从当地人嘴中听说每有大风大浪的时候，可能会有尸体从湖下被卷上来，而且还不是现代人的尸体，模样都很古怪，于是我就产生了兴趣，做了一番准备后，我租了一条船到了湖中心然后就带着潜水设备下去了”<br/>　　<br/>　　从耿明的嘴里，关于抚仙湖的神秘，渐渐的被揭开了一层面纱。<br/>　　<br/>　　耿明下湖的那天，天气是不错的，七月中旬左右，水温是一年中最高的时候了，当时耿明带了个能足够他支撑一个半小时的氧气瓶还有各种装备，就从抚仙湖中间水域潜了下去，一百多米的深度，要不是资深潜水员的话，光是水压都能让正常人有去无回了。<br/>　　<br/>　　耿明下降到一百一十米左右，水面上透过来的光亮就已经渐渐消失了，这还是因为抚仙湖水质不错的原因，换成是别的湖或者江河可能早就看不见了，这时耿明打开了水下强光手电，一路继续下沉，等到了差不多一百五十米左右，他就隐约发现下方有一大片漆黑的区域，然后就是若隐若现的一些建筑轮廓了，当时耿明就被大大的给惊到了，然后迅速下潜，片刻后在他的眼前展现了震惊世人的一幕，抚仙湖的湖底居然是一座古城，并且绝大多数的建筑都保存的十分完善，良好，只有一些稍高点的出现了破损。<br/>　　<br/>　　湖底很干净，只有砂石和水草，就连鱼都很少游过来，这一座古城就像安安静静的躺在水下睡着了一样，毫无声息，但却也显得异常诡异。<br/>　　<br/>　　耿明说他当时足足愣了差不多有两三分钟都没有任何举动，就那么静悄悄的飘在水中没有动弹，等他回过神来之后就意识到自己可能是发现了一个惊人的隐秘，于是连忙往下彻底来到了湖底。<br/>　　<br/>　　耿明说到这时，就深深的吸了口气，然后从桌子上拿起烟点上，抽了大半根后，才继续说道：“我原以为发现一座保存完好的水下城市就已经跟让人吃惊的了，但没想到我下到湖底后，更为意想不到的状况出现了”<br/>　　<br/>　　王惊蛰点头说道：“水下有尸体！”<br/>　　<br/>　　“对，当时在古城上方，能见度也很低，等我下到湖底进入那座古城中的时候才发现，城市里站着很多很多的人，有的是十几人，甚至几十人站在一起的，有的是三三两两单个站着的，他们的身体全都直立向上，男的朝前倾斜，女的身子后仰，身上穿的衣服还在随着水流缓缓的飘动着，体表的皮肤上面布满了水珠，头发慢慢的摆动着……”<br/>　　<br/>　　耿明掐灭烟头，脸上露出惊悚和紧张的表情，说道：“当时真的是把我吓坏了，当了这么多年的潜水员，水里的尸体我也见过不少，但还从来没有见过这么多，并且还是站立在水中的尸体”
