　　“给我断。”
<br/>　　一声冷喝，白骨手杖，应声断裂。
<br/>　　断裂之处，一缕缕浓郁血色萦绕。
<br/>　　“嗯？”毒袍眼眸一睁，闪过一丝不可置信。
<br/>　　“荒影蛛之骨，被你生生切开？”
<br/>　　“是血煞的力量…”毒袍瞬间反应了过来。
<br/>　　之前他身映万毒下，化作无数毒虫，连龙啸都奈何不得。
<br/>　　最后，是易霄的巨大血龙冲撞而来，吞噬了他的所有毒虫。
<br/>　　血龙之内，血气之力暴走异常，锋锐无比。
<br/>　　无数毒虫置身其中，宛若身处血刃风暴，死伤大片。
<br/>　　这才逼得他不得不化回本体。
<br/>　　如今，白骨手杖上断裂处的血色，如之前明显如出一辙。
<br/>　　看似普通血色，却锋利得惊人，仿佛连这片天地都能切割开，更何况他的荒影蛛之骨？
<br/>　　萧逸冷笑一声，“若你早早将我震开，而非得意洋洋地废话一堆，我也没把握切断你的荒影蛛之骨。”
<br/>　　“不过，以我对你的了解，邪修向来谨慎，你毒袍作为邪修中的凶名佼佼者，理应更谨慎。”
<br/>　　“如果我没有猜错的话，你刚才打算算计我吧。”
<br/>　　萧逸的语气，玩味而戏谑。
<br/>　　毒袍的脸色，则陡然发黑。
<br/>　　不错，他毒袍可不是什么善茬，更不是那些话语连篇的蠢货。
<br/>　　他确实对易霄拥有极大的恨意，确实极讨厌易霄的自信。
<br/>　　但，那只是他的一种发泄，他大可等真正胜了此战之后再发泄。
<br/>　　他之所以说那么多，只是卖个破绽给易霄。
<br/>　　易霄只要觉得有破绽可循，有机会，便绝不会撒手。
<br/>　　那么，这般近距离下，他才有更大的把握让紫炎易霄死在太荒之炎下。
<br/>　　如同他毒袍拥有极其过人的逃命手段。
<br/>　　他也深信，堂堂紫炎易霄，这个与别的妖孽不同的狠人，这个终日爱闯荡于各种危机之境的家伙，同样有极强的逃命手段。
<br/>　　易霄这些天来，一路横扫，剿杀他邪君府分部。
<br/>　　今日大好机会，他怎会放过。
<br/>　　若能一举击杀易霄，那自然是最好的。
<br/>　　所以他故意卖了破绽，故意话语极多，让易霄不在他手杖之上撒手。
<br/>　　只可惜，他还是对自己太自信，也太轻视了易霄。
<br/>　　他的算计，一切的基础，在于他自信易霄无法毁去他的荒影蛛之骨。
<br/>　　荒影蛛，乃是太荒十兽之一，骨头何等坚硬，就算是世间强悍火焰也难以烧伤丝毫，更别说弄断了。
<br/>　　可他万万没想到，易霄也在算计他。
<br/>　　单凭易霄，自然无法奈何他的荒影蛛之骨。
<br/>　　可若加上这条孕育百万年的血煞之力，加上那惊人的血气手段呢？
<br/>　　与其说他败给了易霄，不如说是败给了这条血煞，还有那连他都想不明白的血气手段。
<br/>　　二者的互相算计，他，还是败给了易霄。
<br/>　　嘭…
<br/>　　恰在此时，原本凝聚数百丈的白冷苍炎，瞬间溃散。
<br/>　　毒袍身后的千丈白骨之影，也一并消失。
<br/>　　作为‘主体’的荒影蛛之骨已断，自然的，毒袍的手段也一并消散。
<br/>　　“你以为你赢定了？”毒袍双眼一眯。
<br/>　　萧逸眼中的杀意，在这一瞬间顷刻迸发。
<br/>　　“毒袍，我说过，今日你我必须分出胜负。”
<br/>　　“今日，我也再不会让你逃了去。”
<br/>　　话音落下的瞬间，萧逸双手齐出。
<br/>　　有力的双手，瞬间扣住了毒袍的双肩。
<br/>　　“血界第三式，血刹斩。”萧逸心头暗喝一声。
<br/>　　不见萧逸有什么动作，毒袍的脸色，却陡然大变。
<br/>　　毒袍体内气血，开始急剧翻滚，阴翳的脸庞上，开始潮红异常。
<br/>　　他甚至发现，自己竟无法调动自己体内元力半分，更别说使用丝毫手段。
<br/>　　“你…你…这到底是什么武技…”毒袍脸色大变。
<br/>　　仅仅是一秒时间，毒袍已一口腥血喷出。
<br/>　　第二秒，毒袍嘴中喷出的鲜血，愈发多。
<br/>　　第三秒，毒袍最终的鲜血，已不是通过自己喷出，而是仿佛自动汹涌而出。
<br/>　　不错，事实上，萧逸一直与毒袍战斗，所用的手段，皆是气血一道。
<br/>　　而所用的武技，则是血界斩。
<br/>　　之前的血气，变得锋利无比，便是因为血界斩。
<br/>　　荒影蛛之骨的断裂，亦是因为血界斩。
<br/>　　单单血界斩，当然不够。
<br/>　　但若是配合上这条孕育百万年的龙形血煞呢？
<br/>　　而且，还有血刹魔经所赋予的掌控。
<br/>　　这条龙形血煞蕴含的力量，何等庞大，谁敢说能悉数掌控？
<br/>　　血刹魔经敢。
<br/>　　修炼了血刹魔经的萧逸，同样敢。
<br/>　　他不知道碑灵给他的这门功法，到底是何等层次。
<br/>　　但碑灵的一身本事、功法、手段，尽皆来自于剑帝，那位剑道一帝。
<br/>　　自然的，血刹魔经，必然也曾是剑帝的功法之一。
<br/>　　这门功法有多强，萧逸不敢说。
<br/>　　但他可以确定，刚才自己操控这整条血煞时，显然有着接近绝对的掌控力。
<br/>　　六龙六煞，乃是整个六龙地域六方所生，孕育百万年。
<br/>　　即便是龙家历代家主，还有他龙家的功法，也不敢说做到绝对的掌控。
<br/>　　否则，之前六煞之四的掌控权，也不会被毒袍夺了去。
<br/>　　但刚才的萧逸，却明显将整条血煞绝对掌控，也将其威力悉数发挥。
<br/>　　这才有了断荒影蛛之骨，还有如今毒袍几无反抗之力。
<br/>　　“你想杀我？”毒袍冷笑一声。
<br/>　　萧逸同样冷笑，“不是想，是必然。”
<br/>　　“那便一起死吧。”毒袍猛然大喝，脸上无半分惧色，有的，仅仅是疯狂与狰狞。
<br/>　　“你忘了，我还有邪道封天阵。”
<br/>　　毒袍嘴中鲜血四溢，不断流出。
<br/>　　但他的怒吼，却丝毫未受影响。
<br/>　　“爆。”
<br/>　　一声暴喝。
<br/>　　“引动邪道封天阵直接自爆？”萧逸语气一惊。
<br/>　　这次，他是真的一惊，瞳孔也为之一缩。
<br/>　　邪道封天阵，能波及整个六龙地域，甚至让毒袍绝对地掌控了六煞之四。
<br/>　　此刻轰然自爆，其威力…
<br/>　　吼…吼…吼…
<br/>　　刹那间，天地气息暴走不堪，六声惊天震吼，响彻整个六龙地域。
<br/>　　毒袍脚下的两条巨大之龙，毒煞、邪煞，陡然消失。
<br/>　　萧逸脚下的两条巨龙，风煞、血煞，亦一并消失。
<br/>　　还有龙啸脚下的雷煞、木煞，同样消失。
<br/>　　下一秒。
<br/>　　远方，天际，更加狂猛的气息暴走，更加恐怖的怒吼声，震颤天地。
<br/>　　六条颜色各异的游龙，从六方狂猛而来。
<br/>　　“桀桀桀桀。”毒袍嘴溢鲜血，却狞笑着。
<br/>　　“六龙齐噬，任你紫炎易霄手段滔天，今日也难逃一死。”
<br/>　　“那也是你先死。”萧逸眼眸一狠。
<br/>　　毒袍，顷刻化作一个血人。
<br/>　　他那阴蛰的眼眸，渐渐失去了生气，缓缓闭合。
<br/>　　他的体内，已鲜血流尽，生机尽消。
