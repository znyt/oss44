　　听到慕清霄只有二十岁，紫裙女子轻轻惊叹一声，柳眉却是一皱，道：“先生真是年轻有为，但您的导师信息，似乎并不在我们炼药师公会记录之列。”
<br/>　　“他喜欢隐居生活，并不喜欢有人打搅他，难道没有别的办法吗？”
<br/>　　“先生，确实有别的办法，比如通过炼药师公会的考核，同样能够领取等级徽章，但考核一般都是白天，晚上没有高层在此。”
<br/>　　慕清霄剑眉皱了皱，如果可以，他并不想浪费时间，参加考核，但等级徽章拥有的效果不错，也能给他节省许多时间，等上一番倒是无所谓。
<br/>　　“既然如此，那我明日在过来一趟。”
<br/>　　就在慕清霄准备带着小医仙离开之际，门口忽然走进两名老者，目光在四周扫视后，快步向两人走了过来。
<br/>　　停在慕清霄面前，仔细打量一番后，其中一名老者微微点头，惊叹道：“你好，我是炼药师公会的会长，听说小友已经是一名凝聚实火的四品炼药师，不知可否属实？”
<br/>　　慕清霄瞥了眼两名老者胸口的徽章，都是四品炼药师，微微点头，道：“不错。”
<br/>　　他现在虽只炼制过三品丹药，但真要说他是几品的话，六品也不为过，如果手中有炼制六品丹药的材料，他绝对能炼制成丹。
<br/>　　话语刚落，整个大厅都陷入一片死寂，登记信息的紫裙女子笔掉落在地都有所不知，两位老者面面相觑，皆是倒吸一口凉气，显然都惊呆了。
<br/>　　“不......不知小友来我们炼药师公会，所为何事？”炼药师会长开口问道，语气中不参杂一丝傲气，倒像是在与平辈交谈。
<br/>　　“我来领取等级徽章，但听这位姑娘说考核从早上开始，既然如此，我就只能明天早上在来一趟。”
<br/>　　闻言，老者心中一急，他听说有一名二十岁的四品炼药师来到公会，于是急忙赶了过来，想亲自见一面，发现后者气质不凡，更对他的炼药技术产生了浓浓的兴趣。
<br/>　　不光是炼药师会长，连旁边的老者也是如此。
<br/>　　“小兰说的没错，考核是在早上，但小友也不必多跑一趟，你看我们都四品炼药师，由我们来监考，你看如何？”
<br/>　　慕清霄自然没有拒绝的理由，他本就不是一个喜欢复杂的类型，道：“也好。”
<br/>　　“小友，请随我们来吧。”
<br/>　　于是，慕清霄就拉着小医仙的小手跟在身后，在大厅众人惊骇与不敢置信的目光中，向公会内部走去。
<br/>　　路上，两位老者也是产生了浓浓的求知欲，互相交谈炼丹经验，慕清霄都能一口回答上来，两人更加确定，信息不假，四品炼药师，估计没的跑。
<br/>　　二十岁的四品炼药师啊，先前收到消息的他们，差点没将胡子给拽下来，究竟是何等恐怖的存在，能教导出如此妖孽的徒弟？
<br/>　　在他们看来，二十岁能够成为一品炼药师都算是天才了，面前的青年绝对是妖孽！
<br/>　　“小友，敢问家师名讳......”
<br/>　　“他老人家喜欢清静的生活。”慕清霄也不怕得罪人，一口回绝。
<br/>　　虽然在小小的加玛帝国，估计没人知道药尘的身份，但不怕一万就怕万一，暗中可是有魂殿虎视眈眈，他现在的实力，根本无法与之抗衡。
<br/>　　走了几分钟后，慕清霄就跟随两位老者来到公会内部的炼药房，光是站在门口，就能感觉到，空中弥漫着浓郁的药香味。
<br/>　　“这里是我们专属的炼药房，小友不用客气，药材也已经命员工去准备了。”
<br/>　　进入炼药房中，墙壁四周都镶嵌着荧光石，将偌大的炼药房照的极其明亮，在中央位置，摆放着一尊古铜色的药鼎，是一尊中品药鼎。
<br/>　　片刻后，炼药师公会的员工，恭敬的将药材送了进来，并且放在药鼎旁的石台上。
<br/>　　托盘中共有四株五十年份的墨叶莲，两粒成熟的蛇涎果，一颗二十年份左右的聚灵草，还有一枚水属性的二级魔核，正是炼制四品聚气散所需要的材料。
<br/>　　小医仙坐在两位老者身边，美眸痴痴的望着炼药房中央的慕清霄，两位老者各自捧着茶杯，心中也是极其期待起来。
<br/>　　目光在石台上扫过，慕清霄定下心神，盘坐在炼丹炉前，以他的知识与经验，完美的控火力，炼制区区四品聚气散，手到擒来，他缺的只是材料。
<br/>　　现在，炼药师公会给他提供材料，自然是最好不过，同时也替他省下不少的时间。
<br/>　　盘坐在药鼎前，慕清霄抬起手来，掌心顿时冒出一团黑色火焰，火焰没有一丝温度，但空间却因为它的存在而扭曲，黑色极致莫不过如此。
<br/>　　妖异无比的黑色火焰，在慕清霄的控制下传入药鼎中，顿时内部就燃烧起来。
<br/>　　与此同时，坐在边缘的两位老者，茶杯差点就从手中滑出，盯着药鼎中的黑色火焰，瞳孔剧烈收缩。
<br/>　　会长指着慕清霄，惊骇道：“黑色火焰，如此妖异，绝对是异火，异火啊！”
<br/>　　另外一名老者也是惊呆了，目光紧紧盯着慕清霄，眼珠子都要掉在地上，呼吸也变得急促起来。
<br/>　　小医仙美眸则扑闪扑闪的望着慕清霄，蕴含着极致的温柔，芳心不争气的加速跳动，在她看来，后者炼丹时，认真的模样比平时帅气数倍不止。
<br/>　　“绝对是异火，感受不到温度，如此妖异，丹火与兽火绝不可能，如此年纪就是四品炼药师，还能如此娴熟的控制恐怖的异火，简直是惊世妖孽啊！”
<br/>　　炼药师会长目光凝重，嘴角牵了牵，道：“长江后浪推前浪，真是有点想见见慕小友的师尊啊。”
<br/>　　另一名老者道：“我看不透这异火，不知道是异火榜上排名多少，如此控制力，慕小友的实力，估计已经是大斗师级别。”
<br/>　　大斗师是他判断的极限，在高他连想都不敢想，炼药如此妖孽，实力更恐怖，连他自己都不相信。
<br/>　　坐在一旁，将两名老者的话听入耳中，小医仙心中甜蜜极了，若是他们知道，自己的男人是斗皇，不知会露出何等令人期待的表情。
