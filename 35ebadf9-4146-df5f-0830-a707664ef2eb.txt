        “当然怕！怎么会不怕！”连星儿如实回答道，脸色也稍稍有些偏白。
<br/>
<br/>        郎安瞥了连星儿一眼，拿起另一只烤好的野兔，咬了一口。
<br/>
<br/>        连星儿咽了咽口水，弱弱地问道：“你为什么要躲进这片受保护的森林里？”
<br/>
<br/>        这片森林算得上是原始森林里，这里面的一草一木，以及野生动物都受国家的保护。
<br/>
<br/>        所以，这里算得上是动物的一片净土与乐园。
<br/>
<br/>        郎安不禁嗤之以鼻道：“你外公年轻的时候，带着团队拿人体做实验，研究生化病毒与基因变异，甚至不惜拿自己身边的人做实验。你母亲童筱颖的血是万物解药，却是个半成品，能治百病，后遗症最严重的却是将人变成失去记忆的怪物。而我们这种，被研制出来的‘怪物’，既然无法恢复正常，那么就只能被追杀了。因为，上头的大人物，不允许有我们这种‘怪物’出现。”
<br/>
<br/>        听完郎安的这段话，连星儿冷不丁地打了个寒颤。
<br/>
<br/>        而就在这下一秒，郎安突然凑到了连星儿的脸前，让她的两只瞳仁里只有他的影子。
<br/>
<br/>        “其实，你也算是个‘怪物’。因为，你的血，继承了你母亲的‘解药’体质，对我们这种人来说很有吸引力。就算没有我，你还会被其他异类人盯上。这就是你父亲，为什么派那么多保镖暗中保护你的原因。”郎安勾唇笑了笑。
<br/>
<br/>        连星儿双手撑在身后的地面上，悄悄往后挪动着身子，远离郎安。
<br/>
<br/>        她知道自己打不过他，所以对他时刻戒备着。
<br/>
<br/>        郎安又滔滔不绝道：“明明，我们是一类人。只是，你的出生，比我们任何人都要幸运。有‘解药’作为你的母亲，还有一个领头人是你的外公，除此之外，你还有个能护你万全的父亲。而我，什么都没有。”
<br/>
<br/>        连星儿讷讷地咽了咽口水。
<br/>
<br/>        郎安的眸光暗晦不明。
<br/>
<br/>        “你要我跟你走，就是为了告诉我这些？”连星儿凑着眉头，弱弱地问道。
<br/>
<br/>        郎安只是轻轻地笑了笑，转眼间，他的笑容邪肆了：“我想要你给我生狼崽。”
<br/>
<br/>        “？！”连星儿赫然瞪大了眼睛，刚想转身逃跑，却被郎安给直接扑倒在地。
<br/>
<br/>        郎安倾身而上，大手紧紧地抓住她的两只手腕压在她身体两边的地面上。
<br/>
<br/>        即便这地面上铺满了树叶，但连星儿仍旧觉得背后很硬，有什么东西疙得她生痛。
<br/>
<br/>        “郎安，如果你敢碰我，你的下场是什么，不用我说，你应该也知道！”连星儿眉心紧蹙，冷冷地呵斥着。
<br/>
<br/>        但她心里却极其害怕。
<br/>
<br/>        此刻，郎安的瞳仁变成了金色，咧嘴反问：“你不是说，我要选择当‘人’。既然要选择当‘人’的话，就得跟你结合。”
<br/>
<br/>        “别胡扯了！你以为我会信？”连星儿怒斥道。
<br/>
<br/>        郎安俯身，凑到连星儿的耳边，忽然低沉地喃喃细语：“其实，小时候，我不是想攻击你。只是，因为喜欢你，想成为你的宠物罢了。”
<br/>
<br/>        连星儿身心一怔。
<br/>
<br/>        当郎安从她耳畔启开，再次出现在她眼前的时候，却已经不再是刚刚那副俊美的少年模样，而是一只有着金色瞳仁的灰狼。
<br/>
<br/>        连星儿吓得瞪大了眼睛，想要尖叫出声，而声音似乎是卡在喉咙里了，却怎么也发不出来。
<br/>
<br/>        而此刻，郎安却从她的身上退了下来，往洞口走去。
<br/>
<br/>        连星儿连忙起身，小跑着追到了洞口，但是郎安的身影已经消失在了夜色中。
<br/>
<br/>        这里是原始森林吧……
<br/>
<br/>        郎安那家伙把她丢在这里，是打算让她自生自灭吗？
<br/>
<br/>        而就在此刻，不远处传来了野兽嗷嚎的声音。
<br/>
<br/>        连星儿吓得浑身直嘚瑟，立即跑回洞内，找了一个角落蹲着。
<br/>
<br/>        她现在真的是又饿又怕！
<br/>
<br/>        谁说饿了，吃什么都好吃？
<br/>
<br/>        那个烤兔子肉就难吃死了！
<br/>
<br/>        她好想回家……
<br/>
<br/>        连星儿抱着双膝，蜷缩成一团，豆大的泪珠从眼眶里滚落到下巴处汇集，打湿了她的裙子。
<br/>
<br/>        突然，洞口外传来急促的脚步声。
<br/>
<br/>        连星儿已经分不清是人还是兽的，吓得抱头闭上了眼睛。
<br/>
<br/>        “喂！臭丫头！看你下次还乱不乱跑！”
<br/>
<br/>        头顶上方突然传来明司寒的声音。
<br/>
<br/>        连星儿怔愣地抬起头来。
<br/>
<br/>        泪水模糊的视野里出现了一个高大的人影。
<br/>
<br/>        连星儿眯了眯眼睛，挤掉眼中的泪水后，才看清楚面前的俊容。
<br/>
<br/>        明司寒走到连星儿的跟前，蹲下身去，心疼地擦拭着她脸颊上的泪痕：“臭丫头，以后不许再到处乱跑了！听到没？”
