        “我的孩子，只能是我和筱颖的，如果筱颖不能生，我便不要孩子就是。”傅夜沉不假思索地回答。
<br/>
<br/>        连华生发现傅夜沉的性子还是这么倔强，自知劝不动他，也就不再多说什么了：“你忙，不打扰你了，我走了。”
<br/>
<br/>        “好。”傅夜沉应道。
<br/>
<br/>        这些天，他虽然很忙，却总会有意无意地拿起手机看看。
<br/>
<br/>        筱颖仍旧没有电话进来，也没有任何简讯给他，就好像她真的对他无牵无挂。
<br/>
<br/>        以前，他总是执着于自己想要做的事情，当他彻底失去她后，他才明白，就算是他此刻想给她一个她曾经想要的完整的家，然而，现在于她而言，似乎已经不再那么重要了。
<br/>
<br/>        安防局夏阳枭解除了傅夜沉的通缉令，却又给傅夜沉下了另一个套。
<br/>
<br/>        夏阳枭对外称，傅夜沉抓捕冒险者集团的几个高管为理由，说他将功补过，才得以解除通缉令。
<br/>
<br/>        他把黑锅甩给了傅夜沉，自己坐收渔翁之利，这事让傅炎廷极其不满。
<br/>
<br/>        虽然，傅夜沉现在可以不用再隐匿自己，还可以光明正大地回家陪爷爷和母亲吃饭，但是在饭桌上，傅炎廷忍不住发起火来：“如果你爸还在世，安防局的局长之位，哪能轮到他夏阳枭！”
<br/>
<br/>        爷爷气势汹汹，说这话时有恃无恐。
<br/>
<br/>        上次，夏阳枭的手下绑架了傅夜沉的母亲这事，傅炎廷并不知道，若是知道了，冷夏两家的关系会进入白恶化。
<br/>
<br/>        傅夜沉好不容易回一趟傅家大庄园，面对爷爷傅炎廷大动肝火，于是耐着性子安慰道：“爷爷，您吃饭，别生气。”
<br/>
<br/>        傅炎廷依旧恼火：“若不是他在幕后操手，你又岂会落得像现在这样两边都不是？你一心为他做事，他却拿你当棋子使。”
<br/>
<br/>        傅夜沉无言以对。
<br/>
<br/>        毕竟，爷爷说的是事实。
<br/>
<br/>        “小沉，你不是说，筱颖还活着吗？而且，她就是你干妈的亲生女儿。那么她人呢？我怎么就不见你带她回来？我和你干妈提起这件事情，她说她要见见筱颖。”冷夫人试图转开话题。
<br/>
<br/>        傅夜沉知道，自己的母亲和干妈是闺蜜，关系要好到胜过亲姐妹。所以，母亲替干妈关问筱颖的事情，他也能理解，只是……
<br/>
<br/>        冷夫人见傅夜沉剑眉微蹙，心里一揪，慌忙问道：“我听小连说，筱颖好像是失忆了，她是不是对你不好？”
<br/>
<br/>        “小沉，你和筱颖离婚，我们傅家的长孙不能无后。”提起童筱颖的事情，傅炎廷显得更加冷酷无情。
<br/>
<br/>        童筱颖不孕不育这事，是连华生故意说给傅炎廷听的。
<br/>
<br/>        听到爷爷提及此事，傅夜沉自然知道是谁多的嘴。
<br/>
<br/>        “爷爷，您应该也得知了，筱颖是夏家的少小姐。我和筱颖离婚，您觉得夏家会同意吗？”傅夜沉淡定地反问。
<br/>
<br/>        冷夫人十分认同地点了下头，帮腔道：“爸，现在的医学很发达。只要筱颖积极配合去治疗，她和小沉迟早会有孩子的。”
<br/>
<br/>        “我听说，夏阳枭好像不想认她这个女儿。”傅炎廷不以为然地冷哼。
<br/>
<br/>        “不管他认不认，筱颖确实是夏爷爷的小孙女，夏爷爷认就行了。”傅夜沉意会不明地答。
<br/>
<br/>        傅炎廷便不再吭声。
<br/>
<br/>        虽然他现在对夏阳枭有很大的意见，但是对夏阳枭的父亲夏海东还是有世交之情。
<br/>
<br/>        不看佛面看僧面，希望他俩离婚这事情，傅炎廷也就只是嘴上念叨了这么一次，就没再说第二次了。
<br/>
<br/>        晚饭过后，傅夜沉留宿傅家大庄园，陪爷爷下完棋后，才回的自己房间，此时母亲给他端了夜宵进了屋。
<br/>
<br/>        “我知道你晚饭没吃好，特意去厨房里给你做了夜宵。”冷夫人在桌前坐下，向傅夜沉招了招手，示意他也过来做。
<br/>
<br/>        傅夜沉点了点头，坐在了圆桌旁，看着碗里清淡的煮食，全是一些养胃的食材，忍俊不禁道：“妈，其实我不饿。”
<br/>
<br/>        “那多多少少吃点。”冷夫人和蔼可亲地说。
<br/>
<br/>        傅夜沉拿起勺子，象征性地吃了几口。
<br/>
<br/>        “我听小连说，筱颖不仅不能生育，而且还失忆了，小连的父亲还拿筱颖做了实验。这些都是真的吗？”冷夫人担忧地关问。
<br/>
<br/>        傅夜沉拿着勺子的手一滞，垂下眼眸，微微点了下头。
<br/>
<br/>        之前，他一直在保护着筱颖，不让任何人知道筱颖的身世，他以为可以瞒一辈子的，结果，事与愿违。
<br/>
<br/>        冷夫人若有所思地接着说道：“如果筱颖实在不能生，你不能无后，那就找个干净的女人代孕。”
