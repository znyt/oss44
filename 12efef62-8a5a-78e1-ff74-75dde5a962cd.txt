　　变异狼从狼脑袋到狼屁股，被林峰活生生的一刀两半。
<br/>　　林峰整个人从两半尸体之中犹如穿花蝴蝶一样的穿过。
<br/>　　“什么？”
<br/>　　吴刚还有那三个四阶战士看到这一幕，个个都长大了嘴巴。
<br/>　　“这家伙居然一刀秒杀了？”
<br/>　　“还是从正面砍下去的？”
<br/>　　“他居然这么厉害，我们………………”
<br/>　　每个人都震惊了，犹如石雕一样的矗立在原地。
<br/>　　林峰抹了把脸上的鲜血，看了眼两半的尸体。
<br/>　　四阶罢了，自己一刀下去，不死那还真的是见鬼了。
<br/>　　挖走对方的晶石。
<br/>　　林峰走到了吴刚等人面前，“吴队长，这个东西，应该是归我的吧？”
<br/>　　吴刚这才回过神立即点头：“没错，这是你的。小兄弟，看来我是看走眼了，没想到你的实力居然这么厉害。”
<br/>　　其余人也都堪堪回过神。
<br/>　　一个个看着林峰的目光之中充满了尊敬。
<br/>　　达者为师，林峰年龄虽然是他们最小的，但刚才一刀秒杀四阶变异狼的一幕，他们难以忘记。
<br/>　　三个四阶战士张着嘴巴也不知道该说什么。
<br/>　　最后全部齐刷刷给林峰鞠躬。
<br/>　　“之前是我们有眼无珠！”
<br/>　　“大哥，希望您别和我们一般见识！”
<br/>　　“没错，大哥！我们都是浑人！”
<br/>　　林峰没有和他们计较，他也能够理解这群人，毕竟四阶也不低了，会膨胀甚至目中无人也是很正常的。
<br/>　　“以后都低调点知道吗？你们看我，这么厉害，我膨胀了吗？”
<br/>　　“是是是，我们以后肯定向小兄弟学习！”
<br/>　　“小兄弟当真是厉害！英雄出骚年，我等佩服！！“
<br/>　　几个大汉立即拍马屁。
<br/>　　“吴队长，你们还要清理丧尸吧？”
<br/>　　林峰道。
<br/>　　“没错，这个地区丧尸不多，大部分都转移了，只剩下几千只，之前也被大部队清理过了，就剩下一些零星的，大概几百只，顺便搜救一下看有没有幸存者。”
<br/>　　吴刚道。
<br/>　　“既然如此，那就继续吧。我还急着早点完成任务。”
<br/>　　林峰道。
<br/>　　“好！”
<br/>　　吴刚等人立即开始清理丧尸。
<br/>　　有林峰的存在，接下来三个小时自然没有遇到什么麻烦。
<br/>　　实际上，吴刚等人实力都不错，对付三阶以下的都是游刃有余。
<br/>　　但是林峰依然要出手，倒不是为了显摆自己多能干，而是因为杀了丧尸可以得晶石，就这么简单的原因而已。
<br/>　　三个小时后。
<br/>　　白云镇这个小镇算是清理干净了。
<br/>　　“吴队长，基地好端端的为什么要清理这个白云镇。”林峰问道。
<br/>　　“总部应该会在这个地方设立一个前哨据点，用来侦查丧尸的移动，不过这些事情和我们没有关系了，我们回去报道吧。”
<br/>　　吴刚道。
<br/>　　“丧尸会移动？”林峰道。
<br/>　　“嗯，我说的移动就是迁徙的意思，他们会无意识的聚集在一起，会大规模的移动，这个大规模少说也是几千只甚至上万只的丧尸群也是可能形成的。”吴刚道。
<br/>　　“我听说有研究员发现，这群丧尸会无意识的聚集在一起，就是因为晶石，让他们能够感觉到彼此的存在。”
<br/>　　原来如此。
<br/>　　林峰心中恍然，他之前就感觉这丧尸似乎总是三五成群的。
<br/>　　一行人上了军用吉普车。
<br/>　　回去有车，那自然快多了。
<br/>　　不过四十分钟回到了基地内。
<br/>　　吴刚带着小队人去报道了。
<br/>　　在基地入口，众人再次被检测。毕竟每一次离开基地，代表的就是可能被感染的危险，基地不可能在这种问题上忽视。
<br/>　　毕竟，一粒老鼠屎可以坏了一锅粥。
<br/>　　哪怕是吴刚他们这些军部的特警战士，都没有任何的特权，只是检测的通道和其余那些第一次进入基地的不一样，也算是个小小的特权，不需要排那么长的队伍。
<br/>　　检测完毕后，林峰也拿着盖了章证明林峰完成任务的任务单，去军部综合办事处递交任务领取奖励。
<br/>　　军部综合办事处，说白了就是个综合服务大厅。
<br/>　　这座基地归于军部管辖治理。
<br/>　　军部办事处非常大。
<br/>　　来来往往不少人在这里穿梭。
<br/>　　比如说买房子的。
<br/>　　没错，现在买房子还需要军部审核才可以。
<br/>　　所以军部现在要管辖的事情，非常多，不再只是以前那样扛着枪戍卫边境就行了。
<br/>　　当然了。
<br/>　　处理这些事情的，都是普通人，军部的人可没有时间处理这些事情。
<br/>　　林峰来到负责处理任务的任务悬赏结算中心窗口排队。
<br/>　　这个地方就和银行柜台一样，有一共四个窗口，此时此刻都排起了长队。
<br/>　　“妈的，好险，差点我这个任务就失败了。”
<br/>　　“我也一样。”
<br/>　　“哎，我运气不好，没有完成任务。”
<br/>　　不少人都议论纷纷。
<br/>　　大概十五分钟，轮到了林峰。
<br/>　　有吴刚的盖章，自然一切都没有任何问题。
<br/>　　林峰顺利拿到了任务奖励的几颗晶石。
<br/>　　晶石对林峰来说不多，主要是顺路做一做任务，也算是一笔额外的惊喜。
<br/>　　林峰来到基地入口。
<br/>　　这里之前有许多人摆摊，但现在全部被赶到了一个固定的交易区域，主要是为了好管理。
<br/>　　林峰看天色还早就干脆寻思着去交易区看看逛逛。
<br/>　　来到交易区，一眼看过去简直就像是来到了潘家园一样的。
<br/>　　到处都是人头，其中还有不少持枪的军人来回巡逻，甚至这些军人也在买东西的。
<br/>　　“走过路过不要错过，南孚电池，遥控车用完了，收音机可以再用！只要十五颗晶石！”
<br/>　　“挥泪大甩卖啊！！！防爆警棍，绝对的耐用！三十颗晶石！”
<br/>　　“出手一把仿九六式手枪手枪，五十颗三阶晶石！子弹另外单独出售！”
<br/>　　“出售龙泉打造的宝剑一把！吹毛断发！不还价！八十颗三阶晶石！”
<br/>　　“求一包感冒药！什么牌子的都可以！”
<br/>　　林峰逛了一圈，发现似乎冷兵器比热武器还要贵一些。
<br/>　　“也是，砍刀之类的武器，耐久度高，而手枪虽然好威力大，但是这玩意要子弹，子弹这种东西那就是消耗品了，虽然我们国家之前是世界第一大的弹药生产国家，但是现在，这些工厂早就停工了，毕竟没有电，机床根本无法运作。所以是用一颗少一颗。”
<br/>　　林峰逛着逛着忽然是看到了一个好东西。
<br/>　　这是一张俘虏。
<br/>　　不是那种鬼画符，而是一张蓝颜色的符箓。
<br/>　　“小伙子，天罡符！”
<br/>　　卖符箓的人眯着眼笑呵呵道。
